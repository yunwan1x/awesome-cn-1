    Colorful

    Colorful ¶

    精选的精选资源清单，供您选择下一个配色方案. 受启发 awesome_ _清单. 请阅读 __contribution guidelines__贡献之前.

    Tools¶

    Libraries¶

    • Chroma.js -用于处理颜色的JavaScript库.
    • GradStop.js -生成渐变色停止.
    • Colorable -调色板组合对比度测试仪.
    • Adaptive Backgrounds -从图像中提取主色并应用于其父级（jQuery插件）.
    • Coloor -图像预加载实用程序.
    • Color Thief -从图像中获取调色板.
    • Colorify.js -简单，可自定义的小型JavaScript颜色提取器.
    • Colofilter.css -用CSS制成的双色调过滤器.
    • Polychrome -一个用于解析和处理颜色的小型库.
    • Palettable -Palettable（以前是brewer2mpl）是Python调色板的库
    • picocli -用于创建具有ANSI颜色和样式的Java命令行应用程序的小型库.
    • granim.js -创建流体和交互式渐变动画.
    • tinygradient -快速和小的渐变操作.
    • TinyColor -快速，小巧的颜色处理和JavaScript转换.
    • Distinct Colors -生成视觉上不同颜色的调色板.

    Web App¶

    • Color Picker for Data -保持色度不变，轻松选择调色板.
    • Colorbrewer2 -用于热图和数据可视化的多色和单色调色板.
    • Coolors -为您的设计生成无限的调色板并共享.
    • Flat UI Colors -UI颜色集.
    • Material Design Colors -材料设计颜色，平面UI颜色和材料设计图标.
    • Palettab -每个标签都有新的调色板和字体灵感（Chrome扩展程序）.
    • Paletton -配色方案设计师.
    • Pictaculous -生成带有图像的调色板.
    • Adobe Color Wheel -使用色轮创建配色方案.
    • Drag Gradients -生成和自定义多个css3径向渐变.
    • SassMe -实时可视化SASS颜色功能，无需编译.
    • Shade -数学派生的渐变资源管理器.
    • Colourco -轻松直观地组合颜色.
    • HSL Color Picker -HSL拾色器.
    • SVG Color Filter Playground -了解feColorMatrix的工作原理.
    • Random A11y -生成一个非常友好的随机调色板.
    • Save My Palette -保存，共享和导出调色板.
    • Colordot -人类的颜色选择器.
    • Material Palette -Material Design调色板生成器.
    • 0to255 -查找任何颜色的变化.
    • 147 Colors -了解CSS颜色名称.
    • Shutterstock Spectrum -通过光谱按颜色搜索图像.
    • ColRD -创建并与世界分享色彩灵感.
    • CheckMyColours -检查所有DOM元素的颜色组合以获得足够的对比度.
    • Contrast-A -检查颜色组合以获得可接受的对比度和显示.
    • Palettable -在没有事先设计知识的情况下生成漂亮的调色板.
    • Colorgorical -生成n种颜色的调色板.
    • Color Library -自动分色.
    • Color Supply -来自世界各地的设计师和插画家的鼓舞人心的调色板.
    • ColorFavs -创建和发现美丽的调色板和颜色.
    • Colormind -Colormind是使用深度学习的配色方案生成器.
    • iColorpalette -按颜色或主题创建，浏览或导出（aco，ase，svg，png，pdf）调色板.
    • geenes -在几秒钟内创建一个色标，然后将其导出到草图或代码中.
    • Tints and Shades -以10％的增量显示给定十六进制颜色的色调和阴影.

    Color Palettes¶

    • ColorHunt -具有快速预览功能的调色板.
    • COLOURlovers -用于创建调色板和图案设计的工具.
    • Swiss Style Color Picker -调色板集合.
    • UI Gradients -精选美丽的颜色渐变.
    • Color Hex Palettes -发现新的调色板.
    • Color Palettes -用色彩创意装饰您的房屋，公寓，卧室，厨房，客厅和婚礼.
    • Design Seeds -自然界中发现的色彩和有目的的生活的美学.
    • Gradients -精选的渐变.
    • Colour Names -CSS中的颜色名称列表.
    • LOL Colors -精选的调色板灵感.
    • BrandColors -全球最大品牌的官方颜色代码.
    • Color Lisa -世界上最伟大的艺术家的调色板杰作.
    • Cohesive Colors -创建具有凝聚力的配色方案的工具.
    • Colordrop -外观精美的调色板的集合.
    • Open Color -颜色经过十色和近视测试.
    • ColorClaim -微妙的色彩组合.
    • Color Farm -从流行的Dribbble镜头中精选的颜色.
    • WebGradients -免费收集180个线性渐变.
    • Dave Green's 'cubehelix' colour scheme -cubehelix系列配色方案的资源.
    • Color Name List -精心策划的独特颜色名称列表.
