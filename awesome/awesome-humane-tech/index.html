    Humane Technology




    About this list¶

    科技和社交媒体正在对我们的社会产生重大影响. 尽管许多创新技术发明都在改善我们的生活，但人们对这些趋势带来的负面影响的意识日益增强，例如大规模隐私入侵，监视资本主义和技术垄断. 它们导致社交媒体上瘾，心理健康问题，甚至正在侵蚀我们的社会结构.

    此列表汇总了与人文科技主题直接相关的开源项目. 请阅读 guidelines for contributing 在发送您的PR之前.

    > 重要：以下列出的资源是本列表的贡献者真诚收集的，希望对您有用. 但是根据 License 我们不能保证所引用软件的质量或适用性. 因此，在安装任何东西之前，请先正确告知自己，如果遇到不适当或错误的事情，请在我们的网站上提出问题 Issue Tracker 因此我们可以采取适当的措施. 谢谢！

    如果您的项目在此列表中，则您有权自豪地佩戴我们的人文技术徽章. 看到 instructions.

    Community¶

    我们的使命是： To Help Improve Wellbeing, Freedom and Society!

    我们很高兴邀请您参加我们 Humane Tech Community Forum 阅读有关有趣的人文科技主题的更多信息，参加我们的讨论，并成为一名真正的：heart：Humane Tech Activist！

    您也可以通过以下网址跟随我们 @humanetech@mastodon.social.

    Humane design¶

    人性化设计是人性化技术的一个子集，也是信息技术的一个新兴领域. 此空间中的资源在策展列表中单独跟踪，该策展列表是 delightful project on Codeberg.

    • delightful-humane-design -为UX设计人员和开发人员精心设计的人性化设计资源列表.
    • delightful-small-ai -精选的开放式人性化人工智能资源列表.

    • Ethics Pledge -全球倡议，旨在将我们的道德框架和原则纳入您的数据工作.
    • Hippocratic Oath for Technologists -一种可选的誓言，用于建立符合道德规范的经验.
    • IoT Design Manifesto -参与开发物联网的每个人的行为准则.

    • The Digital Product Ethics Canvas -识别并减轻数字产品对个人和社会的风险.
    • Fairwork Foundation -一个旨在制定和衡量平台经济中的标准和认证的项目.
    • Guidelines for Trustworthy AI -基于基本权利和道德原则的可信赖AI指南.
    • Ethical Web Dev -符合道德规范的网站开发和维护指南.
    • Declaration of Digital Autonomy -我们的技术必须尊重使用它的人的权利和自由.
    • A New Digital Manifesto -我们在数字时代的不可剥夺的权利.
    • Society Centered Design -超越以人为本的设计. 我们必须为集体，为社会而设计.
    • The Hippocratic License -开发人员的开放源代码许可证，可强制其代码以道德方式使用.

    • awesome security -软件，库，文档，书籍，资源以及有关安全性的一些很酷的东西.

    • PrivacyScore -网络隐私测量平台.

    • Decentraleyes -模拟内容交付网络的浏览器扩展，可改善您的在线隐私.
    • User-Agent Switcher -浏览器扩展允许通过更改User-Agent标头伪装成另一个浏览器.

    • Disable WebRTC -浏览器扩展程序，禁用WebRTC，以避免从VPN后面泄漏IP地址.
    • Syncthing -取代专有的云服务以分散多个设备之间的文件同步.
    • Tox -点对点即时通讯工具，旨在使普通用户易于获得安全性和隐私性.
    • Oversec -捕获并加密您键入的文本，然后将其传递给实际应用的Android应用.
    • Cloud Firewall -可以根据其IPv4 / 6范围选择和阻止排名前5位的云计算公司的插件.
    • Librefox -一个旨在加强Firefox的隐私和安全性而又不分叉该项目的项目.
    • amp2html -浏览器扩展程序，可自动将加速移动页面（AMP）重定向到实际页面.
    • Location Privacy -共享位置链接的隐私过滤器.
    • DeepPrivacy -图像和视频的全自动面部匿名化技术.
    • PrivacyFlash Pro -根据其代码为您的iOS应用生成隐私权政策.
    • Reveal Frame Component -注重隐私的iFrame占位符（JS脚本）. 仅在用户同意后加载.
    • FreedomBox -易于管理，面向隐私的家庭服务器.
    • EteSync -为您的联系人，日历和任务提供安全，端到端加密和尊重隐私的同步.
    • Image Scrubber -基于浏览器的工具，用于匿名化照片和清理EXIF元数据.
    • LocalCDN -一种浏览器扩展，可模拟内容交付网络以改善您的在线隐私.

    Tracking¶

    • Google Container -将您的Google活动与其他网络活动隔离开，以防止跟踪.
    • Tracking Token Stripper -浏览器扩展程序，可从URL查询字符串中剥离Google Analytics（分析）（UTM）令牌.
    • SilverDog -阻止超声波跟踪应用程序的音频防火墙.
    • Blokada -适用于Android的紧凑，快速的广告拦截器，适用于所有应用，无需root用户.
    • Better -iPhone，iPad和Mac上的Safari隐私工具，可阻止行为广告和跟踪器.
    • Firefox ProfileMaker -创建具有良好默认设置的Firefox配置文件的工具.
    • Searx -可以自我托管的无追踪器，尊重隐私的元搜索引擎.
    • DNS66 -适用于Android的基于DNS的主机阻止程序（和轻量级广告阻止程序）.
    • Cookie-AutoDelete -浏览器扩展程序，该选项卡关闭后将删除cookie，除非站点为allowListed.
    • ClearURLs -使用130条​​规则从地址栏网址中删除跟踪元素的插件.
    • Pi-hole -DNS漏洞，可在不安装任何客户端软件的情况下阻止本地网络上的广告或跟踪器.
    • Shariff -使网站用户可以共享自己喜欢的内容，而不会损害其隐私.
    • Social Share Privacy -jQuery插件，用于添加不允许社交网站跟踪用户的社交共享按钮.
    • TrackMeNot -浏览器扩展程序，通过添加干扰和混淆来隐藏您的真实搜索查询.
    • Deda -打印机跟踪点提取，解码和匿名化工具包.
    • MailTrackerBlocker -适用于macOS Apple Mail的电子邮件跟踪器，阅读回执和间谍像素阻止程序插件.

    Consent¶

    • re:consent -浏览器插件，用于查看/更改您对网站进行数据处理的同意.
    • Trackless -在您的网站上添加GDPR友好的Google Analytics（分析）选择加入/退出按钮.
    • ConsentCookie -可自定义的脚本，允许用户根据情况选择加入Cookie.
    • GDPR Transparency and Consent Framework -帮助遵守欧盟数据保护和隐私规则.
    • Klaro -一个简单而强大的工具，可以管理网站上的用户同意.

    Analytics¶

    • Matomo -Google Analytics（分析）的领先开放替代方案，可让您完全控制数据.
    • Fair Analytics -类似于Google Analytics（分析）的服务器，不会破坏用户的隐私.
    • GoAccess -在终端或浏览器中运行的实时Web日志分析器和交互式查看器.
    • Countly -实时，移动和Web分析，丰富的推送通知和崩溃报告平台.
    • Ackee -基于Node.js的自托管分析工具，适合那些关心隐私的人.
    • Koko Analytics -不使用外部服务且尊重访问者隐私的WordPress插件.
    • freshlytics -自我托管的隐私友好型，可靠且易于部署的分析服务.
    • Plausible Analytics -简单，轻量级（<1 KB）和易于使用的Web分析. 托管在欧盟.
    • Goatcounter -简单的网络统计. 不追踪个人数据.
    • Open Web Analytics -掌握如何衡量和分析网站和应用程序使用情况的控制权.
    • Shynet -无需使用Cookie或JS即可进行隐私保护的详细网络分析.
    • PostHog - Developer-friendly, open-source product analytics.
    • umami -一种简单，快速的网站分析替代Google Analytics（分析）.
    • AWStats -免费的实时日志文件分析器，以获取高级统计信息.
