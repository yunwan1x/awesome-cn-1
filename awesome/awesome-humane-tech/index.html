    Humane Technology




    About this list¶

    科技和社交媒体正在对我们的社会产生重大影响. 尽管许多创新技术发明都在改善我们的生活，但人们对这些趋势带来的负面影响的意识日益增强，例如社交媒体成瘾，心理健康问题和社会侵蚀. 讲话 these problems，由特里斯坦·哈里斯（Tristan Harris）领导的世界一流团队组成，该团队深为关注 The Center of Humane Technology.

    该列表重点关注与人文科技主题直接相关的（大多数是开源）项目. 请阅读 guidelines for contributing 在发送您的PR之前.

    重要：以下列出的资源是本列表的贡献者真诚收集的，希望对您有用. 但是根据 License 我们不能保证所引用软件的质量或适用性. 因此，在安装任何东西之前，请先正确告知自己，如果遇到不适当或错误的事情，请在我们的网站上提出问题. Issue Tracker 因此我们可以采取适当的措施. 谢谢！

    Community¶

    我们的愿景是：无所不在的人性化技术，刺激人类蓬勃发展，人类蓬勃发展.

    我们邀请您参加我们 Humane Tech Community Forum 了解有关这些有趣的人文技术主题的更多信息. 如果您想参与我们的讨论，为真正的解决方案做出贡献，那么我们很高兴欢迎您成为我们的新成员之一. 我们正在建造“人道技术金字塔” ，您可以与我们一起建造.

    Be a Builder Help Improve Wellbeing, Freedom and Society! 并成为一名真正的_Humane Tech Activist！_

    • Programmer's Oath -类似于希波克拉底誓言的程序员道德誓言.
    • Ethical Design Manifesto -为实践道德设计的网站和项目重新绘制了Maslov需求金字塔.
    • Ethical OS Toolkit -预测当今技术未来影响的指南.
    • ACM Code of Ethics -计算机协会的道德和专业行为守则.
    • Ethics Pledge -全球倡议，旨在将我们的道德框架和原则纳入您的数据工作.
    • Hippocratic Oath for Technologists -一种可选的誓言，用于建立符合道德规范的经验.
    • IoT Design Manifesto -参与开发物联网的每个人的行为准则
    • switching.social -流行社交应用程序的道德，易于使用和注重隐私的替代方案.
    • The Critical Engineering Manifesto -适用于工程态度的批判性思维概念宣言.
    • The Digital Product Ethics Canvas -识别并减轻数字产品对个人和社会的风险.
    • Fairwork Foundation -一个旨在制定和衡量平台经济中的标准和认证的项目.
    • Guidelines for Trustworthy AI -基于基本权利和道德原则的可信赖AI系统指南.
    • Ethical Web Dev -符合道德规范的网站开发和维护指南.

    • Location Privacy -共享位置链接的隐私过滤器.

    • Firefox ProfileMaker -创建具有良好默认设置的Firefox配置文件的工具.
    • Searx -可以自我托管的无追踪器，尊重隐私的元搜索引擎.
    • DNS66 -适用于Android的基于DNS的主机阻止程序（和轻量级广告阻止程序）.
    • Cookie-AutoDelete -浏览器扩展程序，该选项卡关闭后将删除cookie，除非站点为allowListed.
    • ClearURLs -使用130条​​规则从地址栏网址中删除跟踪元素的插件.

    • Shariff -使网站用户可以共享自己喜欢的内容，而不会损害其隐私.

    • EasyList -一组规则（最初是为Adblock设计的），可自动从互联网中删除不需要的内容.
    • TrackMeNot -浏览器扩展程序，通过添加干扰和混淆来隐藏您的真实搜索查询.

    • Fathom -Google Analytics（分析）的一种更简单，更注重隐私的替代方案.

    • Google Analytics Proxy -代理服务，用于Google测量协议和客户端替换分析脚本.
    • GoAccess -在终端或浏览器中运行的实时Web日志分析器和交互式查看器.
    • Countly -可自我托管的实时，移动和Web分析，丰富的推送通知和崩溃报告平台.
    • Ackee -适用于那些关心隐私的人的基于Node.js的自托管分析工具.
    • Koko Analytics -不使用外部服务且尊重访问者隐私的WordPress插件.
    • freshlytics -自我托管的隐私友好型，可靠且易于部署的分析服务.
    • Plausible Insights -简单，轻巧的分析服务，尊重访问者的隐私.
    • Goatcounter -简单的网络统计. 不追踪个人数据.

    • Referrer Spam Blacklist -社区推荐的垃圾邮件发送者列表.
    • FilterLists -广告，跟踪器，恶意软件和烦恼的过滤器和主机列表的目录.
    • hosts-blocklists -专为dnsmasq设计的基于主机和域名的阻止列表.
    • All In One Customized Adblock List -广告屏蔽列表，可彻底屏蔽跟踪器，广告，不需要的Cookie等.
    • Web Annoyances Ultralist -修复并从许多网站中删除烦人和分散注意力的Web元素.
