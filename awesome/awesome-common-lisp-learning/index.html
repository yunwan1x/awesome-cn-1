    Awesome common lisp learning

    Awesome Common Lisp Learning ¶

    该列表重点介绍了学习Common Lisp的资源，特别是我发现有用的资源.

    还有其他一些令人敬畏的Common Lisp列表，它们专注于其他主题： - Libraries - Curated Libraries -从以上列表中进行了分叉和更新. - Software

    欢迎捐款. 请阅读 contribution guidelines 有关详细信息.

    How To Use¶

    1.获得一个Lisp环境. 2.将 The Common Lisp Hyperspec. 3.下载并阅读适当级别的Lisp书籍. 输入示例并使用代码. 随意切换书籍并尝试其他书籍. 4.尝试一下 Exercism. 5.如果遇到困难，请找到一个在线社区，然后 ask smart questions. 6.了解 Quicklisp. 7.在某些时候，请阅读Lisp实施手册.

    Lisp Environments¶

    您可以直接运行Lisp实现，但是编辑环境使体验更轻松. -预包装环境 - Portacle -便携式多平台Common Lisp环境. 它带有SLIME，SBCL（流行的Common Lisp实现），Quicklisp和Git，提供了稍微定制的Emacs. 无需安装，因此这是一种非常快速简便的方法. - Lispbox -一个IDE（Emacs + SLIME），Common Lisp实现（Clozure Common Lisp）和库管理器（Quicklisp），它们预先打包为Windows，macOS和Linux的归档文件. “盒中的Lisp”的后代 mentioned in Practical Common Lisp. - Lispworks Personal Edition -用于LispWorks Lisp的非基于Emacs的IDE，但有一些限制. - Allegro Common Lisp -有免费 Express Edition IDE，以及 training videos on YouTube. -对于高级用户 - Articulate Common Lisp -用于整合Lisp环境的HOWTO，并提供有关以下内容的信息： useful libraries 以及如何 structure projects. -如果您是经验丰富的人 Emacs 用户，您只需安装 SLIME 和一个 supported Common Lisp implementation . 见 SLIME manual 更多细节. -《通用Lisp食谱》包含有关以下内容的更多信息： installing an implementation 使用像这样的实现管理器 Roswell，或在Docker上.

    Online References¶

    • The Common Lisp Hyperspec (CLHS) - Common Lisp的语言参考文档. 立即为其添加书签.
    • Chapter 7 -涵盖了通用Lisp对象系统（CLOS）.
    • The ANSI Common Lisp Standard Draft -ANSI INCITS 226-1994（以前为ANSI X3.226-1994）标准的最新草案版本. -草稿是免费的，标准不是. -与官方标准几乎相同，有些人更喜欢CLHS.
    • The Common Lisp Cookbook -有用的Lisp食谱列表. 还包含其他在线CL信息来源的列表.
    • Common Lisp the Language (2nd Edition) by Guy L. Steele -在ANSI标准化之前对Common Lisp语言的描述. 不要将其用作参考.
    • Cliki on CLtL2 -声称它具有以下更有用的描述 LOOP 和 FORMAT 比CLHS好.
    • Stack Overflow on CLtL2 -有些人建议使用CLtL2以获得见识和启发，但在编程时要使用CLHS.
    • A list of differences between CLtL2 and the ANSI Common Lisp specification.
    • Chapter 28 -涵盖Common Lisp对象系统.

    Online Books¶

    这些可在线免费访问的书籍，其排列顺序从基本到高级： - Common Lisp: A Gentle Introduction to Symbolic Computation by David S. Touretzky -对编程新手的不错的介绍. 包含一些实用工具的代码，例如 DTRACE 和 SDRAW. - Common Lisp: An interactive approach by Stuart C. Shapiro -一本主要通过练习进行教学的教科书. - Practical Common Lisp by Peter Seibel -对经验丰富的程序员的精彩介绍，并从一开始就着重强调使CL与其他语言不同的原因. - Common Lisp Koans -并非完全是一本书，而是一系列的koans可以帮助您学习该语言. - On Lisp by Paul Graham -一本适合有经验的Lispers的好书. - Let Over Lambda by Doug Hoyte -有关Lisp宏的高级书籍. -前六章可在​​线获得. - Comments on Let Over Lambda -有些人建议您在阅读本书之前先熟悉On Lisp，并且开始缓慢. - The chapter on closures -有几个不平凡的示例，并演示了此功能的强大功能. - Paradigms of Artificial Intelligence Programming by Peter Norvig -具有许多Lisp有趣的应用程序，但不再是AI的良好参考.

    Offline Books¶

    这些是您必须购买或从图书馆借来的书. - Land of Lisp by Conrad Barski, MD -一个有趣的Lisp简介，它使用漫画并让您编写游戏. -可作为 ebook. -编写了两个LOOP宏，分别是“ for x for y ...”，您可以将其从“ 0 for y ...”更改为“ for x for y ...”，以使其在SBCL中工作. -有一个 errata page. -本书的大部分内容都很有趣并且很容易理解，但是从第18章开始却显得有些毛茸茸.以后的章节并没有太多介绍，因此我建议此时改用另一本书. -第13、19和20章中的Web服务器示例仅适用于CLISP，并且需要添加HTTP响应标头才能正确呈现HTML. - ANSI Common Lisp by Paul Graham -很好地作为参考，涵盖了CLOS，并提供了一些示例程序实现. -的纯文本版本 Chapter 1 和 Chapter 2 可以在线获得. - Common Lisp Recipes by Edmund Weitz -大量的Common Lisp模式. - Object-Oriented Programming in Common Lisp by Sonya E. Keene -CLOS的深入描述，并说明如何在示例应用程序中使用它. - The Art of the Metaobject Protocol by Gregor Kiczales, Jim des Rivieres, and Daniel G. Bobrow -描述CLOS元对象协议（MOP）. - Chapters 5 and 6 (available online) -对CLOS的元对象协议扩展的规范. - Adam Tornhill on AMOP -建议先阅读实用的Lisp，然后再阅读Lisp中的面向对象编程，然后再阅读AMOP.

    Online Community¶

    • 这 Cliki - 这 Common Lisp Wiki. A great resource for all things CL. Has a great Getting Started 页面和广泛的清单 Lisp books.
    • The #clschool, #lisp, #ccl, #sbcl and other rooms 在 Freenode IRC network 是学习Common Lisp的好地方.
    • Lisp Subreddit -一个活跃的社区，并且在侧边栏中拥有大量有用的链接和参考文档.
    • Common Lisp Subreddit -Common Lisp的Subreddit.
    • Exercism's Common Lisp track -一种通过实践学习语言，审查代码并与其他人讨论解决方案的绝妙方法.
    • Lisp Discord server -是Lisp的不和. 有一个专门针对Common Lisp的频道，将很乐意回答问题.

    Library Management¶

    这些不是库，但是可以帮助您查找和安装其他库. - Quicklisp -Lisp库的程序包管理平台. - Quickdocs -Quicklisp中的项目文档. - State of the Common Lisp Ecosystem, 2015 -有关应使用哪些库以及原因的文章. - Articulate Common Lisp -列出了一些有用的库.

    Common Lisp Implementations¶

    本节按字母顺序列出了一些常见的CL实现及其手册. 除非另有说明，否则这些都是自由软件实现. 另请参阅Cliki的清单 free software Common Lisp Implementations. - Allegro Common Lisp (ACL) -商业，但有免费 Express Edition. - ACL manual - Training videos on YouTube - CLISP - CLISP Manual - Clozure Common Lisp (CCL) - CCL manual - Carnegie Mellon University Common Lisp (CMUCL) - CMUCL manual and other useful information - Embeddable Common Lisp (ECL) - ECL manual - LispWorks -商业广告，但如前所述，具有 Personal Edition 有轻微的限制. - LispWorks manual - Steel Bank Common Lisp (SBCL) -我个人的最爱. - SBCL manual - Scieneer Common Lisp (SCL) -商业Linux和Unix实现，但具有不受限制的免费评估和非商业使用版本. - SCL manual
