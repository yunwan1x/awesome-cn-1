    Self Hosted

    Awesome-Selfhosted¶

    自我托管是在本地托管和管理应用程序的一种做法，而不是从 SaaSS 提供者.

    这是一个清单 Free 软件 network services 和 web applications 可以在本地托管. 非自由软件列在 Non-Free 页.

    See Contributing.
