    JVM

    Awesome JVM ¶

    精选的JVM低级，性能和非框架相关内容的精选列表.

    - [Communities](#communities)

    Bytecode¶

    用于字节码操作和分析的工具.

    • asmtools - 用于开发用于生成Java .class文件的工具.
    • Byte Buddy - 代码生成库在运行时无需编译器的帮助即可创建Java类.
    • Jitescript - 类似于BiteScript的字节码生成库.

    Garbage collectors¶

    • JVM的垃圾收集器.*

    • Azul Pauseless Garbage Collection - 为Java应用程序提供连续，无间歇的操作.

    • Balanced GC - 用于IBM WebSphere Application Server V8的Java虚拟机中提供的GC策略.
    • Epsilon GC - 完全被动的GC实现，具有有限的分配限制，并且可能具有最低的运行时性能开销.
    • G1 - Garbage-First垃圾收集器.
    • Shenandoah - 超低暂停时间垃圾收集器.
    • The Garbage Collection Handbook - 预定了最近硬件和软件方面的垃圾收集新挑战.
    • ZGC - 垃圾收集器针对低延迟和非常大的堆进行了优化.

    Load tools¶

    生成负载并精确测量系统的工具，无需协调遗漏

    • Gatling - 用于测试HTTP服务器的异步非阻塞方案驱动的负载测试工具.
    • wrk2 - wrk的恒定吞吐量，正确延迟记录变体.

    Languages¶

    在JVM上运行的语言. * Ceylon - 由Red Hat创建的面向对象的强大而静态的编程语言，强调不变性. * Clojure - Rich Hickey创作的Lisp方言. 动态类型，强调函数式编程. * Erjang - 基于JVM的Erlang VM. * Eta - JVM上纯粹，懒惰，强类型的函数式编程语言. * Frege - Pure functional programming language in the spirit of Haskell. * gojava - Go包的Java绑定. * Golo - 一种简单的动态语言，广泛使用invokedynamic. * Groovy - 可选的类型和动态语言，具有静态类型和静态编译功能. * Java - 通用，并发，强类型，基于类的面向对象语言. * JRuby - 在JVM上实现Ruby语言. * JPHP - Java VM上的PHP. * Jython - 用于Java平台的Python. * Kawa - Scheme语言的扩展，它是Lisp系列编程语言. * Kotlin - 用于JVM，Android和浏览器的静态类型编程语言. * LuaJ - 以Java为中心的lua vm实现，旨在利用标准Java功能. * Nashorn - 使用本机JVM的Java轻量级高性能JavaScript运行时. * OCaml-Java - 支持OCaml语言v4. 生成普通的Java字节码并与Java无缝集成. * Rembulan - Rembulan是JVM的Lua 5.3实现，用纯Java编写，具有最小的依赖性. * Renjin - 用于R语言的基于JVM的解释器，用于统计分析 * Scala - 强大而静态的编程语言，结合了面向对象和函数式编程思想. * Xtend - Flexible and expressive dialect of Java, which compiles into Java 5 source code.

    Machine Learning¶

    • Deeplearning4j - 用于JVM的开源，分布式深度学习库.
    • H2O - 快速统计，机器学习和数学运行时.
    • Smile - 统计机器智能和学习引擎.

    Memory and concurrency¶

    用于高效内存布局和并发访问的工具和数据结构.

    • Agera - Google的Android反应式编程.
    • Agrona - 构建高性能应用程序时常见的数据结构和实用程序方法库.
    • Apache Arrow - 用于柱状内存分析的高性能跨系统数据层.
    • bloofi - 多维Bloom过滤器的Java实现
    • Cap’n Proto - 疯狂快速的数据交换格式和基于功能的RPC系统.
    • caffeine - Java 8的高性能缓存库.
    • Chronicle-Bytes - 低级内存访问包装器.
    • Chronicle-Queue - 微秒存储，将所有内容存储到磁盘.
    • Chronicle-Map - 为低延迟和/或多进程应用程序设计的内存中键值存储.
    • clj-ds - Clojure的数据结构被修改为在Clojure之外使用.
    • colfer - 二进制序列化格式和类生成器.
    • commons-math - 轻量级，独立的数学和统计组件库.
    • CuckooFilter4J - Bloom过滤器替换近似集合成员资格查询.
    • cyclops - 用于RxJava，Reactor，FunctionalJava，Guava和Javaslang的集成模块.
    • Eclipse Collections - Java的集合框架.
    • externalsortinginjava - 使用多核和外部存储器算法对非常大的文件进行排序.
    • failsafe - 用于处理故障的轻量级零依赖库.
    • fasttuple - 在堆内和堆外内存中相邻布局的集合.
    • fast-uuid - 用于快速有效地解析和编写UUID的Java库.
    • FlatBuffers - Efficient cross platform serialization library for C++, C#, Go, Java, JavaScript, PHP, and Python.
    • geohash - 用于geohashing的Java实用程序方法.
    • gs-collections - 高盛集合框架.
    • hollow - 用于利用小到中等大小的内存数据集的Java库和综合工具集.
    • high-scale-lib - Cliff Click的高级图书馆.
    • hppc - 高性能原始集合.
    • injector - 一个新的Java Executor.
    • java-concurrent-hash-trie-map - 来自Scala集合的并发trie哈希映射实现的Java端口.
    • java-hll - 用于HyperLogLog算法的Java库.
    • JavaFastPFOR - 库可以非常快速地压缩和解压缩整数数组.
    • java-string-similarity - 字符串相似度和距离度量，包括Levenshtein编辑距离和sibbling，Jaro-Winkler，最长公共子序列，余弦相似度等.
    • JCTools - JDK目前​​缺少并发数据结构.
    • DSL-JSON - 具有高级编译时数据绑定的高性能JSON库.
    • jsoniter - 声称是有史以来最快的JSON解析器（DSL-JSON的副本）.
    • jOOL - Java 8 lambdas的有用扩展.
    • Koloboke - Java Collections直到内存和性能的最后一个痕迹.
    • LevelDB - 用Java重写LevelDB（端口）.
    • lightweight_trie - 一种非常节省内存的trie（基数树）实现.
    • lmdbjni - LMDB的Java API（HawtJNI），这是一个用C语言编写的超快速，超紧凑的键值嵌入式数据存储.
    • lmdbjava - 用于LMDB的Java API（JNR），它是用C编写的超快速，超紧凑的键值嵌入式数据存储.
    • low-gc-membuffers - 使用直接ByteBuffers的内存循环缓冲区，以最大限度地减少GC开销.
    • lwjgl3 - Java库，支持跨平台访问流行的本机API，可用于开发图形（OpenGL），音频（OpenAL）和并行计算（OpenCL）应用程序.
    • MapDB - 由堆外或磁盘存储支持的集合.
    • mph-table - Minimal Perfect Hash Tables是一个不可变的键/值存储，具有高效的空间利用率和快速读取.
    • mug - 源自Google的小型，零功能的util库.
    • netty-buffers - 内存缓冲池实现类似于jemalloc.
    • ObjectLayout - 布局优化的Java数据结构包.
    • ohc - 为Apache Cassandra 3.0开发的Java大型堆缓存.
    • okio - 现代Java IO库，可以做出巧妙的事情来节省CPU和内存.
    • onyx-java - 通过为Onyx工作流的每个组件提供Java等效项来镜像Onyx Platform核心API.
    • parquet - 使用Dremel文件中描述的记录粉碎和汇编算法的列式存储格式.
    • PauselessHashMap - 一个java.util.HashMap兼容的映射，在调整大小时不会停止放置或获取.
    • pcollections - 持久Java集合库.
    • protobuf - 谷歌的数据交换格式.
    • Quasar - JVM的轻量级线程和actor.
    • rtree - Java中具有反应式api的不可变内存中的R树和R * -tree实现.
    • RTree2D - RTree2D是一个2D不可变R树，具有STR（Sort-Tile-Recursive）打包，用于平面和球面上的超快速最近和交叉查询.
    • Reactive Streams - 具有无阻塞背压的异步流处理标准.
    • Reactive Streams Utilities - Reactive Streams的未来标准实用程序库.
    • RoaringBitmap - Java中更好的压缩位集.
    • rollinghashjava - 在Java中滚动哈希函数.
    • Reactor - JVM for Java，Groovy，Clojure等上的反应性数据应用程序.
    • RxJava - 用于使用可观察序列组合异步和基于事件的程序的库.
    • SmoothieMap - java.util.Map impl，最差put延迟比java.util.HashMap小100多倍.
    • Simple Binary Encoding - 高性能消息编解码器.
    • splitmap - 并行位图实现.
    • DataSketches - 随机流算法的Java软件库.
    • stormpot - JVM的快速对象池.
    • stream-lib - 用于汇总流中数据的Java库，用于存储所有事件是不可行的.
    • streamvbyte - 使用StreamVByte编解码器在C中进行快速整数压缩.
    • TraneIO - Future抽象的高性能实现.
    • transducers-java - 可组合的算法转换，独立于其输入和输出源的上下文.
    • VarInt - 没有deps的No-deps变量int实现（由Bazel提供）.
    • vavr - Java 8+功能库.
    • wire - 适用于Android和Java的简洁轻量级协议缓冲区.
    • Zero-Allocation-Hashing - 在Java中散列任何字节序列，包括各种原始数组，缓冲区，CharSequences等.

    Metaprogramming¶

    针对JVM的解析器，解释器，编译器和源代码生成.

    • Antlr - 用于读取，处理，执行或翻译结构化文本或二进制文件的分析器生成器.
    • auto - Java的源代码生成器集合.
    • Apache Calcite - 动态数据管理框架和SQL解析器插件.
    • Checker Framework - 编译器插件，可以发现错误或验证其缺失.
    • compile-testing - javac和注释处理器的测试工具.
    • derive4j - 代数数据类型构造函数，模式匹配，态射，光学和类型类.
    • error-prone - 将常见的Java错误作为编译时错误捕获.
    • GHCVM - 支持GHC Haskell的Haskell到JVM编译器.
    • Graal - 与HotSpot虚拟机集成的新的Java实时即时编译器.
    • grappa - Parboiled的Java分支. 编写没有预处理阶段的语法.
    • immutables - 生成简单，安全且一致的值对象.
    • javacc - 用于Java的分析器生成器.
    • javaparser - 用于Java的Java 1.8分析器和抽象语法树.
    • JavaPoet - 用于生成.java源文件的Java API.
    • jparsec - 使用纯Java和Haskell Parsec构建迷你解析器.
    • JSweet - 从Java到TypeScript / JavaScript的转换器.
    • MPS - 设计和构建可扩展的DSL和编辑器.
    • lombok - 减少通常为Java类编写的样板代码量.
    • parboiled - 基于解析表达式语法解析任意输入文本.
    • Sulong - 使用Truffle和Graal用Java编写的LLVM IR解释器.
    • TeaVM - 从Java字节码到JavaScript的提前翻译编译器（转换器）.
    • Truffle - 将语言实现为简单解释器的框架.
    • Xtext - 编程语言和DSL开发框架.

    Native¶

    互连JVM和本机代码

    • hawtjni - 基于Eclipse SWT中使用的JNI生成器的JNI代码生成器.
    • Java Grinder - 将Java字节码编译为微控制器组件.
    • j2v8 - 用于Google V8 JavaScript引擎的Java API.

    Network¶

    用于网络编程，数据包捕获，监控，测试和弹性的工具.

    • Aeron - 高效可靠的UDP单播，UDP多播和IPC消息传输.
    • armeria - 基于Java 8，Netty 4.1，HTTP / 2和Thrift构建的异步RPC / API客户端/服务器库.
    • Chronicle-Network - 高性能网络库.
    • comcast - 模拟糟糕的网络连接.
    • gor - 实时HTTP流量重放.
    • gRPC - A high performance, open source, general RPC framework that puts mobile and HTTP/2 first.
    • jRT - 测量java应用程序对基于套接字的请求的响应时间.
    • JXIO - AccelIO上的Java API（C库），一种针对硬件加速优化的高性能异步可靠消息传递和RPC库.
    • K3PO - 创建任意网络流量和行为，以验证网络端点是否正常运行.
    • muxy - 模拟真实的分布式系统故障.
    • Netty - 用于高性能协议服务器和客户端的异步事件驱动网络库.
    • okhttp - 适用于Android和Java应用程序的HTTP + HTTP / 2客户端.
    • one-nio - 用于构建高性能Java服务器的库.
    • proteus-java - 基于RSocket的Proteus Java Client.
    • reactive-grpc - 反应性gRPC是一套用于将gRPC与Reactive Streams编程库一起使用的库.
    • RSocket - RSocket是一种二进制协议，用于字节流传输，如TCP，WebSockets和Aeron.
    • SimianArmy - Resiliency tool that helps ensure that your applications can tolerate random instance failures.
    • pcap4j - 使用libpcap捕获，制作和发送数据包的Java库.
    • pig - Linux数据包制作工具.
    • tcpdump - 用于网络流量捕获的数据包分析器
    • tcpflow - 以便于协议分析和调试的方式捕获TCP连接流.
    • tcpreplay - Pcap编辑和重播工具.

    Nix tools¶

    分析JVM和与主机环境交互时有用的 nix工具* * atoptool - 记录系统和过程活动以进行长期分析，突出显示过载系统. * bcc - 基于BPF的Linux IO分析，网络，监控等工具. * Flame Graphs - 可视化的配置软件，允许快速准确地识别最频繁的代码路径. * ioping - 简单的磁盘I / 0延迟测量工具. * javap - 将类文件反汇编为反映java字节码的代码. * jhat - Java堆分析工具 * jinfo - 打印给定进程的配置信息. * jstack - 为给定的Java进程打印线程的堆栈跟踪. * jstat - 监视JVM中的GC和编译器统计信息. * hwloc - 报告处理器的结构，核心数，超线程和高速缓存大小. * likwid - 读取Intel和AMD处理器上的硬件性能计数器. * numactl - 控制进程或共享内存的NUMA策略. * oprofile - 系统范围的硬件性能监控，易于使用的接口，开销低. * perf - 使用性能计数器进行Linux分析. * perf-tools - 基于Linux perf_events（aka perf）和ftrace的性能分析工具. * sysdig - 从正在运行的Linux实例捕获系统状态和活动，然后保存，过滤和分析. * sysstat - Linux的性能监视工具. * taskset/process-affinity - 检索或设置进程的CPU亲和性. * tiptop - 像顶部一样，但也显示每个周期的指令（IPC）.
