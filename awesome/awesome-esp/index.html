    Awesome ESP ¶

    精选的ESP8266 / 32专案和程式码清单.

    >两者 ESP8266 和 ESP32 是上海的制造商乐鑫系统有限公司生产的具有完整TCP / IP堆栈和微控制器功能的低成本Wi-Fi微芯片.


    >请参阅 Contributing 有关如何为该列表做出贡献的信息.

    Firmware¶

    • Espressif AT -ESP8266的默认原始固件.
    • NodeMCU -用于ESP8266的基于eLua的固件.
    • ESPBasic -用于8266的BASIC固件，可轻松进行无线编程.
    • MicroPython -为ESP8266和32实现Python3的实现.
    • ESP32 -用于ESP32和8266的3D打印机的实验固件.
    • Frankenstein -具有ESP8266强大功能的快速脏固件.
    • MongooseOS -具有C和JS的特定于IoT的固件. 适用于ESP32 / 8266.
    • DeviceHive -作为DeviceHive IoT数据平台客户端的固件，仅适用于8266.
    • RT-Thread -ESP32可用的中文开源固件.

    Tools¶

    • ESP Flash Tool -两个ESP的香草固件更新程序.
    • Arduino Core/8266 -ESP8266的Arduino核心.
    • Arduino Core/32 -ESP32的另一个Arduino核心.
    • ESPTool -用于两个ESP的Bootloader通讯的Espressif命令行工具.
    • ESP-Open-SDK -ESP8266的开放SDK.
    • ESPTool-ck -用于在ESP8266中刷新的CLI工具.
    • ESPTool-gui -基于ESPTool-ck的Flash GUI工具.
    • NodeMCU Flasher -NodeMCU为其操作系统提供的官方刷新工具.
    • LuaNode -适用于32/8266的仅lua的SDK.
    • Arduino FS Plugin - An Arduino plugin for filesystem uploads in the 8266.
    • PlatformIO -同时支持ESP32和ESP8266的跨平台IDE和调试器.

    Projects¶

    Smart Home and IoT¶

    • OpenMQTTGateway -针对其他设备中的两个ESP的多协议MQTT网关的实现.
    • ESPHome -通过简单但功能强大的配置文件和家庭自动化系统控制ESP的功能齐全的系统.
    • Sonoff-Homekit -Sonoff设备（和其他8266设备）的备用固件，允许通过Apple的Homekit进行控制.
    • DoorsignEPD -使用ESP32的带电子纸显示屏的智能门牌.
    • EPaperWeatherDisplay -使用ESP32的非常可爱的电子墨水天气显示.
    • SuperGreenOS -用于ESP32的功能齐全的家庭农业自动化软件.
    • CanAirIO -公民科学项目，该项目使用移动站和固定站通过ESP32和智能手机测量空气质量.

    InfoSec¶

    • ESP32-BLECollector -可以显示BLE设备并从中收集数据的管理设备，所有操作都在一个漂亮的屏幕界面中进行.
    • ESP32Marauder -用于WiFi和蓝牙的综合攻防工具套件.
    • ArduinoPcap -一个库，可以为两个ESP生成具有网络流量的.pcap文件.
    • WiFi Satellite -巨大的Wifi“卫星”，可以使用14个ESP32监视所有14个2.4Ghz通道.
    • ESP8266 Deauther -使用ESP8266的Wifi网络非常酷的伪干扰（解除身份验证）.
    • PacketMonitor -漂亮的OLED监视器，用于WiFi通道中的数据包活动. 每个ESP有两个版本.
    • WiFiDuck -具有无线功能的按键喷射器，与Rubber Ducky相似，但功能更强大.
    • ESP8266 Beacon Spam -想混淆别人吗？ 此设备会创建数百个伪造的WiFi网络.
    • DeauthDetector -如果检测到WiFi取消身份验证攻击，就会发光的小型设备. 与最后六个项目由同一个人制作.

    Biomedical¶

    • HeartyPatch -使用ESP32的可穿戴BLE和WiFi连接的ECG-HR补丁.
    • HealthyPi v4 -一款出色的开源生命体征监护仪，可以监控EG32的ECG，呼吸，脉搏血氧饱和度和体温.

    LoRa¶

    • Meshtastic -ESP32 LoRA板作为安全的，长电池寿命的网状GPS通信器.
    • ESP32-Paxcounter Wifi和蓝牙驱动，启用LoRaWAN，电池供电的迷你Paxcounter，基于便宜的ESP32 LoRa IoT板构建
    • Disaster Radio -由太阳驱动的抗灾通讯网络

    Others¶

    • Open SmartWatch -带有GPS，惯性单元和超酷3D打印表壳的FOSS智能手表.
    • SoftRF -可以在无人机项目中使用的DIY航空近距离感知系统.
    • Retro ESP32 -适用于Odroid Go（带有ESP32）的超酷启动器，它可以模拟多个复古控制台.
    • PedalinoMini -ESP32内置的无线MIDI踏板控制器，适用于吉他手.
    • StickWatch -使用ESP32的基于M5Stick的智能手表模块.
    • DroneBridge -DroneBridge的实现，ESP32上用于无人机和无人机的信号链路.

    Libraries¶

    • Wasm3 -专为嵌入式设备而设计的，闪电般的WebAssembly解释器，与两种ESP兼容.
    • Homie8266 -8266的Homie协议的框架实现.
    • ESP-Dash -用于在8266/32中创建远程仪表盘的美观，快速的框架. 无需互联网.
    • ESP_mqtt -用于ESP8266的MQTT帮助程序库.
    • GUIslice -用于多个设备和屏幕控制器的拖放GUI框架. 兼容8266和32.
    • MicroWebSrv2 -一个非常强大的MicroPython Web服务器，可以在ESP32中使用.
    • IRremoteESP8266 -在ESP8266中发射和接收IR信号.
    • esphomelib -在8266中与HomeAssistant集成的框架.
    • TTS -用于多个Arduino设备（包括ESP）的某种程度上不错的文本到语音库.
    • Free802.11 -通过ESP32发出任意802.11信号的库.
    • Koyn -用于ESP32和ESP8266的分散式比特币库.
    • TFTLibrary -ESP32的TFT兼容性.
    • UTFT-ESP -对ESP32 / 8266的UTFT支持.
    • ESPAudio -用于在ESP8266 / ESP32中播放各种音频格式的库.
    • AsyncTCP -8266和32的异步TCP库.
    • ESP-HomeKit -在RTOS上实现8266的Homekit实现.
    • ESPHelper -8266的面向MQTT和Wi-fi自动化的库.
    • ESPHelper/32 -32的ESPHelper库的端口.
    • ESP8266Wifi -8266的简单Arduino Wifi库.
    • WiFiESP -用于Wifi管理的Arduino库，用于8266开发板的客户端/服务器.
    • TinyGSM -一个快速简单的Arduino库，用于与GSM模块交互，该模块还可以通过AT命令控制8266.
    • mJS -MongooseOS使用的轻量级受限JS引擎，与32和8266兼容.
    • ESPUI -一个简单的库，用于为两个ESP制作交互式Web界面.
    • ESP32 ePaper -功能齐全的库，可将ePaper模块与ESP32一起使用.
    • TinyUPnP -轻量级的UPnP IGD库，用于在8266和32上进行自动端口转发.
    • Esp32SSHClient -在ESP32中实现SSH客户端的库.
    • painlessMesh -一个负责使用ESP8266和ESP32硬件创建简单网状网络的细节的库.
    • WifiEspNow -的Arduino库 ESP-NOW，由定义的无连接WiFi通信协议 Espressif.
    • go-mcu -Golang软件包，用于与基于NodeMCU的板进行交互.
    • CanAirIO SensorLib -具有自动配置多个PM2.5，CO2和环境传感器的ESP32 / 8266库.
    • Dhyara -使用ESP Now制作移动自组织网络（MANET）的AC / C ++库.
