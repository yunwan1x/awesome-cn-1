    Xamarin

    Awesome-Xamarin ¶

    精选的书签集合，包含针对Xamarin Android / iOS / Windows / OSX的主观现代/有趣且理想的开源库/工具

    General¶

    • AutoMapper ★5,724 -.NET中基于约定的对象-对象映射器.
    • Fluent Validation ★3,570 -用于建立验证规则的流畅接口和lambda表达式.
    • Humanizer ★3,426 -处理和显示字符串，枚举，日期，时间，时间跨度，数字和数量.
    • NodaTime ★970 -.NET的替代日期和时间API.
    • Polly ★4,666 -异常处理策略，例如重试，永远重试，等待和重试或断路器.

    Architecture¶

    • Behaviors Toolkit ★34 -一种以最少的代码向Xamarin应用程序添加常见和可重用的交互性的方法.
    • Conditions ★48 -帮助开发人员以流畅的方式编写前置条件和后置条件验证的库.
    • Stateless ★2,294 -状态机.

    Async¶

    • AsyncEx ★1,253 -异步/等待的帮助程序库.
    • LinqToAwait ★99 -基于任务的LINQ，旨在与异步/等待一起使用.

    Charts¶

    • MicroCharts ★1052 -创建跨平台（Xamarin，Windows等）的简单图表.
    • MPAndroidChart/iOSCharts ★6 -Xamarin Forms中使用了MPAndroidChart / iOSCharts绑定支持.
    • OxyPlot ★1,311 -.NET的跨平台绘图库.

    Cloud¶

    • Azure -Microsoft Azure.
    • Bugfender ★10 -将您的应用程序日志存储在云端（iOS和Android）.
    • FireSharp ★479 -Firebase REST API包装器.

    Database¶

    • Akavache ★1,692 -用于本机应用程序的异步键值存储.
    • Breeze ★63 -富客户端应用程序开发人员的数据管理库.
    • Couchbase.Lite ★299 -轻巧的嵌入式NoSQL数据库.
    • Lager ★2 -使用Akavache作为存储后端的跨平台设置存储.
    • Massive ★1,708 -用于数据库表的“包装器”，并广泛使用System.Dynamic.
    • Realm ★690 -直接在手机，平板电脑或可穿戴设备内部运行的移动数据库.
    • Settings ★58 -Xamarin和Windows的跨平台设置插件.
    • SQLite.Net-PCL ★340 -SQLite 3数据库支持，PCL，异步.
    • LiteDB ★4245 -在单个数据文件中的.NET NoSQL文档存储.
    • DB4O-GPL ★8 -面向对象的数据库，支持嵌入式和远程连接.

    Fody¶

    • Fody ★2,150 -编织.net程序集的可扩展工具.
      • AutoDependencyProperty.Fody - Automatically generates DependencyProperty boilerplate from simple C# properties.
      • PropertyChanged.Fody ★820 -在编译时将INotifyPropertyChanged代码注入属性.
      • ReactiveUI.Fody ★105 -为属性和ObservableAsPropertyHelper属性生成RaisePropertyChange通知.

    Framework¶

    • Ammy -用于XAML平台的现代UI语言. 免费用于非商业开发.
    • Appercode.UIFramework ★13 -允许使用单个XAML布局构建跨平台移动应用程序的用户界面.
    • Invention - Develop native apps for iOS, Android and Windows with 100% code sharing using Visual Studio and C#.NET.
    • SimplyMobile ★103 -抽象的移动功能集合.
    • Xamu-Infrastructure ★104 -扩展，MVVM类，行为和其他杂项. Xamarin大学的有用代码位.

    Game Engine¶

    • CocosSharp ★463 -Cocos2D和Cocos3D API的CSharp实现.
    • MonoGame ★5,276 -Microsoft XNA 4.x框架的开源实现.
    • Paradox ★1,706 -Paradox3D + Silicon Studio .NET.
    • UrhoSharp ★275 -跨平台的高级3D和2D引擎.

    IO/Storage¶

    • IO ★6 -使用熟悉的api访问系统文件夹和文件.
    • PCL Storage ★270 -用于.NET的一致，可移植的一组本地文件IO API.
    • FilePicker-Plugin-for-Xamarin ★37 -简单的跨平台插件，使您可以选择文件并使用它们.

    IOC/DI¶

    • Autofac ★2,169 -令人上瘾的.NET IoC容器.
    • DryIoc -用于.NET的快速，小型，功能齐全的IoC容器.
    • Funq ★2 -高性能DI框架，通过使用lambda和泛型函数作为工厂来消除所有运行时反射.
    • LightInject ★320 -用于.NET的轻便，简单且快速的IoC容器.
    • Ninject ★2,034 -.net依赖注入程序的忍者.
    • Stiletto ★36 -Dagger的.NET端口，Square的轻量级Android依赖注入程序.
    • TinyIoC ★585 -单类简易IoC容器.

    Internationalization¶

    • I18NPortable ★49 -Xamarin和.NET的简单跨平台国际化/翻译.
    • Resxible ★9 -从单个RESX文件自动生成多个平台相关资源文件的工具.
    • Vernacular ★167 -跨平台本地化，用于转换标准字符串格式的工具.

    Layout¶

    • Flex ★128 -灵活的盒子布局系统.

    Media¶

    • EZ-Compress ★11 -Xamarin的简单图像流压缩插件.
    • Fast & Furious Image Loading ★820 -Xamarin库可快速轻松地加载图像.
    • Lottie ★643 -在Xamarin的Android和iOS上原生渲染After Effects动画.

    • MediaManager ★269 -跨平台Xamarin插件可播放PCL中的媒体.
    • NGraphics ★482 -用于呈现矢量图形的跨平台库.

    • SkiaSharp ★920 - Powerful C# API for doing 2D graphics. It is powered by Google’s Skia library.
    • Splat ★590 -跨平台图像加载，颜色和内容.
    • SimpleAudioPlayer ★14 -用于将本地文件和音频数据作为流播放的简单插件.
