    Awesome tensorflow lite

    Awesome TensorFlow Lite ¶

    TensorFlow Lite 是一组工具，可帮助转换和优化TensorFlow模型以在移动设备和边缘设备上运行. 它目前在超过40亿个设备上运行！ 使用TensorFlow 2.x，您可以使用tf.Keras训练模型，轻松地将模型转换为.tflite并进行部署; 或者您可以从模型动物园下载经过预训练的TensorFlow Lite模型.

    这是TensorFlow Lite模型的绝佳列表，其中包含示例应用程序，有用的工具和学习资源- 展示社区使用TensorFlow Lite构建的内容 将所有样品并排放置以便于参考 *分享知识和学习资源

    如果您想贡献并遵循指南，请提交PR here.

    What is new¶

    以下是TensorFlow Lite的新功能和工具： * Announcement of the new converter - MLIR基于-并支持转换新模型类别（例如Mask R-CNN和Mobile BERT等），支持功能控制流程并在转换过程中更好地处理错误. 默认情况下在每夜构建版本中启用. * Android Support Library -使移动开发更加容易（Android 示例代码）. * Model Maker -创建您的自定义 image & text 几行代码即可轻松完成分类模型. 请参阅下面的图标分类器以获取社区的教程. * On-device training -终于到了！ 目前仅限于将学习用于图像分类，但这是一个很好的开始. 见官方 Android 示例代码，另外一个来自社区（Blog | Android). * Hexagon delegate -如何使用Hexagon委托在移动和边缘设备上加快模型推断. 另请参阅博客文章 Accelerating TensorFlow Lite on Qualcomm Hexagon DSPs. * Model Metadata -提供模型描述的标准，这也使 Code Gen and Android Studio ML Model Binding.
