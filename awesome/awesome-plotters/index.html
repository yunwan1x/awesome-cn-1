    Awesome plotters

    Awesome Plotters ¶

    用于计算机控制的绘图机和其他视觉艺术机器人的精选代码和资源清单.

    Hardware¶

    Plotters¶

    • AxiDraw -来自的笔式绘图仪 Evil Mad Scientist，在 #plottertwitter.
    • Line-us -可爱的小启动机器人绘图臂.
    • Makeblock XY Plotter -可破解的XY绘图仪套件（已停产？）.
    • Drawing Robot -带3d打印的AxiDraw克隆 Arduino CNC Shield 控制器运行grbl固件.
    • 4xiDraw -另一个3d可打印的AxiDraw克隆，带有 Arduino CNC Shield 控制器运行grbl固件.
    • WaterColorBot -XY艺术机器人和软件以水彩颜料作图.
    • EggBot -用于蛋形和球形物体的笔式绘图仪.
    • HP Pen Plotters -HPGL标准创建者提供的老式台式和落地式笔式绘图仪. 7475A型非常常见，通常可以在eBay上找到.
    • Roland Pen Plotters -老式平板HPGL笔式绘图仪. 在eBay上搜索“ roland dxy”.
    • Polargraph -原始的Polargraph硬件和软件项目.
    • Makelangelo -开源Polargraph Artbot.
    • BrachioGraph -一种便宜，简单的绘图仪，由棍棒，伺服器和运行Python的Raspberry Pi制成.
    • Arduino CNC Drawing Machine -相当简单的3d打印AxiDraw样式绘图仪，带有良好的视频文档.

    Motor Controllers¶

    • grblShield -所有需要的步进电机控制硬件 Arduino 使用grbl固件导入基于G代码的运动控制器. （adafruit)
    • TinyG -功能更强大的基于6轴G代码的运动控制硬件. （adafruit)
    • Arduino CNC Shield -与Arduino兼容的Grbl步进电机控制板，类似于 grblShield.
    • Raspberry Pi CNC Hat -带步进控制器的Raspberry Pi附加板和运行grbl的微控制器. 与Pi的串行引脚接口.

    Accessories¶

    • WiFi232 -通过DB25插头Wifi到RS-232串行. 无线控制串行绘图仪.
    • Plotter Cable Pinout -适用于大多数HP和Roland绘图仪的绘图仪电缆的示意图. 在eBay或Amazon上搜索“ DB9到DB25串行零调制解调器电缆”或类似产品以查找出售.

    Pens¶

    • Sharpie Fine Point Plotter Adapter -3d打印适配器，适合HP-GL绘图仪中的标准Sharpie.
    • Parametric 3d-Printable Plotter Pen Adapter -可调型号，可打印各种笔的适配器.
    • Plotter Pen STL Models -短和长标准绘图笔的精确STL模型.
    • Pens for AxiDraw -适用于一般绘图仪滥用的笔清单.
    • Pens for EggBot -着重于鸡蛋和玻璃杯的钢笔建议，但仍普遍适用.
    • JetPens – The Best White Ink Pens -对许多白色墨水笔进行全面审查，并附上其覆盖范围的图片.

    Software¶

    HPGL¶

    HPGL是大多数旧笔式绘图仪，甚至许多新的乙烯基切纸机都使用的基于串行/文本的协议.

    • Chiplotle -用于生成HPGL并与串行绘图仪接口的Python库.
    • Chiplotle on Parallel Port -Chiplotle的调整可与并行端口连接一起使用.
    • HPGL Reference Guide -基于HTML的HPGL参考.
    • HP 7475A Interfacing and Programming Manual -已扫描的PDF手册，其中包含完整的HPGL参考.
    • djipco/hpgl -一个Node.js库，可与HPGL兼容的绘图仪和打印机进行通信.
    • hp2xx -GNU工具，可将HPGL转换为其他矢量和栅格格式. 也可以在X11中用作预览.
    • vec -用于生成HPGL的C示例​​代码，带有龟图形界面.
    • d3-hpgl -HTML Canvas API的适配器，因此您可以使用常用的输出HPGL D3 图书馆.
    • HPGL Viewer -使用JavaScript和HTML5画布的HPGL Viewer.
    • HPGL Sender -用于预览HPGL并将其发送到绘图仪的Web界面.
    • HPGLGraphics -用于编写HPGL文件的处理库.

    G-code¶

    G代码是用于控制CNC机器的基于文本的标准. 尽管它是为工业机器设计的，但它在许多爱好者3D打印机固件中的使用也使其在小型DIY项目中无处不在.

    • grbl -用于Atmega 328微控制器和Arduino的高性能G代码解释固件.
    • cncjs -基于Web的界面，用于控制运行grbl，TinyG或其他基于G代码的固件的CNC机床.
    • node-gcode -基于Node.js的G代码解释器和模拟器.
    • svg2gcode -基于Node.js的命令行实用程序，用于将SVG转换为G代码.
    • svg2gcode -基于Python的实用程序，可将SVG快速转换为G代码.
    • jscut -用于将SVG转换为G代码的基于Web的实用程序.
    • Universal-G-Code-Sender -基于Java的grbl兼容跨平台G代码发送者.
    • ChiliPeppr Hardware Fiddle -基于Web的模块化工作区，以可视化G代码并控制硬件.
    • gcode-generative-for-processing -处理库，旨在从简单形状创建gcode. （设计用于Creality CR10）

    Plotter-Specific¶

    特定于绘图仪或控制器的软件.

    • axidraw -Inkscape的官方AxiDraw扩展.
    • axi -AxiDraw v3的非官方Python库.
    • xy -Makeblock XY绘图仪机器人套件的​​实用程序.
    • LaserGRBL -针对grbl控制器的激光优化Windows GUI. 可以重新用于使用螺线管进行笔上/下移动的DIY笔式绘图仪.
    • Line-us Inkscape Plugin -将图纸发送到 Line-us plotter 直接来自Inkscape.
    • Line-us API Examples -的示例代码 Line-us 绘图仪基于G代码的API.
    • @beardicus/line-us -JavaScript库，用于控制 Line-us 节点或浏览器访问计算机.
    • PenPlotter -使用高级固件的Polargraph控制器.
    • Makelangelo-firmware -Makelangelo极谱图机器人的固件.
    • RoboPaint -WaterColorBot的软件.
    • AxiTurtle -处理中AxiDraw的Turtle图形.
    • GRBL-Plotter -适用于grbl控制器的绘图仪优化的Windows GUI，具有SVG和DXF导入功能以及灵活的笔上/下控制功能.
    • saxi -AxiDraw的驱动程序和库. 使用恒定加速运动计划，并自动调整为纸张大小.
    • MP2300-Tools -用于将HPGL转换为Graphtec的GPGL格式的软件，以及用于Graphtec绘图仪笔适配器的CAD文件.

    Vector Creation¶

    从零开始或通过其他格式转换创建矢量插图的工具.

    • Inkscape -流行的跨平台开源矢量图形编辑器.
    • p5.js -“ JavaScript库，使艺术家，设计师，教育者和初学者都可以访问编码”.
    • Paper.js -《矢量图形脚本的瑞士军刀》.
    • ln -用Go语言编写的基于矢量的3D渲染器.
    • autotrace -将位图图像转换为矢量图形.
    • stipplegen -从位图图像创建有趣的点画. （blog post)
    • SquiggleDraw -“ SquiggleDraw将使用亮度更改正弦波的幅度，从图像创建SVG文件”.
    • svgurt -基于Web的PNG到SVG创意面条.
    • maptrace -通过跟踪栅格图像来生成水密多边形矢量地图.
    • Drawbot_image_to_gcode_v2 -创建可在绘图机器人上使用的G代码.
    • blackstripes -将PNG图像转换为SVG线图.
    • Ribbon -用Go语言编写的蛋白质带状图.
    • penplot -JavaScript中绘图仪艺术的开发环境.
    • penkit -用于创建基于行的SVG图形的Python库.
    • generativeExamples -生成可打印PDF的示例处理代码.
    • Let's make map -基于网络的工具，可从Mapzen磁贴中导出SVG地图.
    • SuperformulaSVG for web -生成线条艺术的网络应用.
    • scribbleplot -处理中的图像image草变换.
    • Maker.js -用于为CNC和激光切割机创建2D矢量绘图的库.
    • Turtletoy -具有SVG导出功能的基于浏览器的JavaScript turtle图形API.
    • cozyvec -Web / Standalone终端环境，用于绘图员艺术作品和推文图.
    • makio135/plotter -一个 Observable 笔记本收集器中充满了针对绘图仪的工作.
    • PlotterFun -基于浏览器的图像到SVG转换器，类似于SquiggleDraw.

    Vector Utilities¶

    用于操纵和优化基于矢量的文件格式的工具.

    • svgsort -用于绘制SVG文件的路径规划，减少了用笔向上移动所花费的时间.
    • svgo -基于Node.js的工具，用于优化SVG文件.
    • Polargraph Optimizer -优化极谱图的绘制计划.
    • penkit-optimize -使用车辆路径求解器的SVG优化器，以最大程度地缩短绘图时间.
    • svg-crowbar -仅限Chromebook的书签，用于从HTML文档中提取SVG.
    • vpype -基于绘图仪的基于Python的CLI实用程序，用于生成和处理SVG，包括缩放和优化路径.

    Fonts¶

    单行矢量字体或“雕刻字体”.

    • Summary of single line fonts -良好的信息以及其他资源和字体的链接.
    • Hershey Vector Font -60年代矢量字体的.fnt格式. 很好地概述了字体的原始数据格式.
    • hershey-fonts -C库和好时字体的原始字体数据.
    • OneLineFonts.com -具有一些单行字体的商业站点可供购买.
    • svg-fonts -SVG格式的单行字体，主要用于 Hershey Text Inkscape插件.
    • CNC Text Tool -基于浏览器的Hershey Text工具，可导出到SVG.

    Inspiration and Instruction¶

    • Hektor -2002年的原始基于电缆的牵引机器人.
    • Pen Plotter Art & Algorithms -分为两部分的介绍，介绍了如何生成绘图图形.
    • Surface Projection -使用Python和penplot深入研究表面投影和隐藏线去除.
    • Fractal Generation with L-Systems -用于创建基于线的分形图形的技术.
    • Pen Plotter Art & Algorithms -绘图仪简介，AxiDraw，penplot库的演练.
    • Introduction to TSP art -用于旅行业务员问题（单路径）艺术的资源.
    • Hidden wireframe removal -讨论和指向线框的STL文件删除代码的链接.

    • The Best XY Plotters in 2020 -很好地概述了AxiDraw及其克隆，以及一些DIY选项.
