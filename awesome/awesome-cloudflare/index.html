    Awesome cloudflare

    >精选的精选列表 Cloudflare 工作人员食谱，开源项目，指南，博客和其他资源.

    Cloudflare提供内容交付网络（CDN）服务，DDoS缓解，互联网安全和分布式域名服务器（DNS）服务，位于访问者和Cloudflare用户的托管服务提供商之间，充当网站的反向代理.

    DNS¶

    • DNS Zone Files - 准备导入通用配置区域文件，以便轻松配置各种服务.
    • Lexicon - CLI工具，以标准化方式处理各种DNS提供商的DNS记录.
    • Th3inspector - 用于信息收集的一体化CLI工具.
    • Partner Panel - 托管合作伙伴为客户提供DNS管理面板的工具.
    • CFPMP - 合作伙伴管理小组.
    • Flares - DNS备份工具.
    • Block Bad Bot Ruleset - 收集规则集以阻止具有防火墙规则的恶意爬虫.
    • Docker DDNS - Docker镜像将免费DNS服务用作动态DNS提供程序.
    • DDNS script for Synology - Synology NAS的DDNS脚本.
    • Dynamic DNS Bash - bash脚本中的动态DNS更新程序.
    • Dynamic DNS PHP - PHP中的动态DNS更新程序.
    • Dynamic DNS Python - python中的动态DNS更新程序.
    • Multi-Provider DDNS Script - 多个提供程序ddns脚本没有依赖项.
    • Argo Tunnel Client - Argo隧道的CLI客户端，一个隧道守护程序，通过Cloudflare网络代理任何本地Web服务器.
    • Upper - 使用网络服务器前面的缓存代理大大加快工艺速度.
    • Cloud Buster - 一个全面的测试工具，可检查原始IP泄漏的站点.
    • CLI Tool - 用于与Cloudflare交互的CLI工具.
    • Detector - 检测网站是否在Cloudflare后面运行.
    • Scrape - Python模块绕过反bot页面.
    • CloudFlair - 查找受公开公开的CloudFlare保护的网站的原始服务器的工具.

    Apps¶

    Cloudflare Apps 允许您将工具或服务发送到数百万个站点. 任何Cloudflare客户都可以在几秒钟内在他们的网站上预览和安装您的代码.

    Workers¶

    Cloudflare Workers 提供无服务器执行环境，允许您创建全新的应用程序或扩充现有应用程序，而无需配置或维护基础架构.

    Tools¶

    • Wrangler - wrangler是一个CLI工具，专为有兴趣使用Cloudflare worker的人设计.
    • Playground - 直接在浏览器中针对任何站点预览和测试代码的简单，即时方式.
    • Serverless Plugin - 插件 serverless framework 使用Workers开发和部署无服务器应用程序.
    • Worker App Kit - 用于创建，开发，测试和部署工作者应用程序的便捷工具集.
    • GitHub Action - 将工作人员部署到主分支.
    • Workers KV Viewer - 用于工人KV存储的基于CLI的交互式查看器.
    • Redirect Management - 生成重定向工作者.
