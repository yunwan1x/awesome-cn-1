    Awesome Tailwind CSS ¶

    >与Tailwind CSS相关的精选精彩清单

    Tailwind CSS 是实用程序优先的CSS框架，用于快速构建自定义用户界面.

    Tailwind UI-设计精美的组件和模板，使用您已经了解和喜欢的CSS框架手工制作.

    Plugins¶

    • Custom Forms (Official) -为表单元素添加更好的默认样式.
    • benface's transitions -添加带有或不带有CSS变量的可配置过渡实用程序.
    • Theming (Dark/Light) -通过CSS变量和prefers-color-scheme添加了强大的主题功能.
    • Text Indent -添加text-indent实用程序.
    • Transforms -添加transform实用程序.
    • Image Rendering -添加了“图像渲染”实用程序.

    • Filters -添加filter实用程序.
    • Spinner -添加了一个微调器实用程序.
    • Elevation -添加 Material UI elevation 实用程序.
    • Caret Color -添加caret颜色实用程序.
    • Direction -添加了“ RTL”和“ LTR”变体.
    • Touch -添加touch变体.
    • Blend Mode -添加blend-mode工具.
    • Colorize -添加filter实用程序.
    • Writing Mode -添加了“写入模式”实用程序.
    • Responsive Embed -添加了“响应嵌入”组件.
    • Aspect Ratio -添加aspect-ratio实用程序.
    • Hyphens -添加了“连字符”实用程序.
    • Border Gradients -添加了“边界图像”渐变实用程序.

    • Font Variant Numeric -添加font-variant-numeric实用程序.
    • Dark Mode -根据“ prefers-color-scheme”媒体查询添加“ dark”变体.
    • Vue.js -添加与Vue.js的v-cloak指令配合使用的可见性类.
    • List Reset -重新添加在Tailwind CSS 1.0之前删除的list-reset类.
    • Fluid -添加流体上浆工具.
    • Flexbox Order -扩展order实用程序.
    • benface's gradients -添加渐变实用程序.
    • lorisleiva's gradients -添加背景渐变实用程序.
    • Alpha -添加alpha颜色变体.
    • Bootstrap Tables -根据Bootstrap的表添加表实用程序.
    • Typography -添加排版工具.
    • Card -添加卡组件.
    • webdna's transitions -添加了可配置的过渡实用程序.
    • glhd's transitions -添加了基本的转换实用程序.
    • Triangle After -添加了CSS三角形实用程序.
    • Scrims -添加稀松布实用程序.
    • Colors to CSS Variables -将颜色配置导出到CSS自定义属性.
    • CSS Variables -将配置导出到CSS自定义属性.
    • Spaced Items -添加间隔组件，为除最后一个容器之外的所有容器项添加固定边距.
    • Heropatterns -广告 Hero Patterns 组件.
    • Localized -添加基于HTMLlang属性的变体，以仅对某些语言使用实用程序.
    • Skip link -向主要content_可访问组件添加_Skip.
    • Pseudo -将自定义变体添加到Tailwind CSS的配置中.
    • Custom Native -利用Tailwind CSS的配置来创建实用程序.
    • Truncate Multiline -添加了用于截断多行文本元素的实用程序.
    • Debug Screens -添加显示当前活动屏幕（响应断点）的组件.
    • Dark Mode with Class -添加基于CSS类的“深色”变体.

    >-_以下这些插件提供的功能已在最新的Tailwind CSS版本中完全或部分实现.
