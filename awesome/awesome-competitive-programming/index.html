    Awesome Competitive Programming ¶

    精选的“竞争性编程”、“算法”和“数据结构”资源的精选列表.

    为了将人们与信息联系起来而创建，下面的这个列表包含了我在 11 年的竞争性编程生涯中收集到的所有精彩资源的完整集合. 我希望你会发现它有用:)

    Tutorial Websites¶

    > 很棒的网站，有很棒的教程.

    ☆ 姓名 描述
    ★★★ Topcoder Data Science Tutorials 由受人尊敬的 Topcoder 成员编写的教程列表. 许多顶级程序员从这里开始学习数据科学.
    ★★★ E-Maxx (Russian), (English) 在俄语竞争性编程社区中广泛使用和引用的教程网站. 原网站的大部分文章已被翻译成英文，谷歌翻译对其余文章还可以.
    ★★☆ Algorithms - GeeksforGeeks 一个拥有大量关于不同主题的精美文章档案的网站. 它是算法课程的绝佳免费资源.
    ★★☆ PEGWiki 一个网站，在许多主题上都有令人惊叹的深度 wiki 类文章. 在我看来，它比维基百科上的要好得多.
    ★★☆ Notes - HackerEarth 一个很棒的教程众包平台. 同时访问 Code Monk.
    ★★☆ USA Computing Olympiad (USACO) 在其网站上包含几个培训页面，旨在培养一个人按照自己的节奏解决困难和各种算法问题的编程技能.
    ★★☆ basecs 一个包含有关基本算法和数据结构的深入图解教程的博客.
    ★★☆ Competitive Programming - Commonlounge 针对初学者和中级概念的简短视频教程. 高级教程选自各种 CP 博客上的最佳教程.
    ★☆☆ OLYMPIADS IN INFORMATICS 一本国际期刊，专注于在教学和学习信息学领域工作的专业人士的研究和实践，以帮助有才华的学生.
    ★☆☆ algolist (Russian) 一个致力于各种算法的俄罗斯网站. 这个网站上列出的一些主题似乎很有趣.
    ★★☆ 演算法筆記 (Algorithm Notes) (Chinese) 台湾竞技编程社区中最受欢迎的教程网站之一. 该网站的维护者花费了大量精力研究算法.
    ★★☆ 国家集训队论文 1999-2015 (Papers from Chinese IOI training camps) (Chinese) 来自中国IOI训练营的论文. 有趣的是，人们可以告诉不同的地区强调不同的事物.
    ★★★ Mini-Editorials， 经过 Mostafa Saad Ibrahim 学员 许多问题的迷你编辑解决方案，包括UVA/SPOJ/IOI/Olympiad问题
    ★★☆ OI Wiki (Competitive Programming) (Chinese) OI Wiki致力于成为一个免费、开放、持续更新的编程竞赛（competitive Programming）知识集成站点. 这个 wiki 是那些试图学习一些东西的有竞争力的程序员的一种伴侣或指南

    Books¶

    > 竞争性编程推荐书籍列表.

    ☆ 姓名 描述
    ★★☆ Competitive Programming ，史蒂文和费利克斯哈利姆 本书包含相关数据结构、算法和编程技巧的集合. 这是一本广受欢迎的书. ... 第一个版本是免费 download (pdf).
    ★★☆ Programming Challenges: The Programming Contest Training Manual ，史蒂文斯基纳和米格尔雷维拉 本书包括 100 多个编程挑战，以及解决这些挑战所需的理论和关键概念. 问题按主题组织，并辅以完整的教程材料.
    ★★☆ Competitive Programmer's Handbook , by Antti Laaksonen 为有抱负的 IOI 和 ICPC 参赛者介绍竞争性节目. 免费到 download (pdf).
    ★★☆ Computational Geometry: Algorithms and Applications , 作者: Mark de Berg, Otfried Cheong, Marc van Kreveld, Mark Overmars 这是一本写得很好的书，涵盖了广泛的计算几何问题.
    ★☆☆ The Hitchhiker’s Guide to the Programming Contests , 通过 Nite Nimajneb 这本书是免费 download (pdf) . 本书涵盖了与竞争性编程相关的各种主题.
    ★★★ プログラミングコンテストチャレンジブック (Japanese), by 秋叶拓哉, 岩田阳一, 北川宜稔 An absolutely phenomenal book. The contents, organized in a very coherent manner, are nothing short of amazing. ... 培养与锻炼程式设计的逻辑脑：世界级程式设计大赛的知识、心得与解题分享(Chinese Traditional)
    ★★☆ 算法竞赛入门经典 (Chinese), by 刘汝佳 The Art of Algorithms and Programming Contests (English), 打下好基础：程式设计与演算法竞赛入门经典(Chinese Traditional)
    ★★☆ 算法竞赛入门经典——训练指南 (Chinese), by 刘汝佳, 陈锋 提升程式设计的解题思考力─国际演算法程式设计竞赛训练指南(Chinese Traditional)
    ★★★ 算法艺术与信息学竞赛 (Chinese) , by 刘汝佳, 黄亮 一个古老的经典. 这本书很旧，但以今天的标准来看，这本书的内容仍然被认为是非常困难的.

    Books for Algorithms¶

    ☆ 姓名 描述
    ★★★ Introduction to Algorithms ，作者：Thomas H. Cormen、Charles E. Leiserson、Ronald L. Rivest 和 Clifford Stein 也称为 CLRS（取自名字首字母），这本书通常被称为算法和数据结构的“圣经”. 它是大学算法课程最受欢迎的教科书之一. 本书详细介绍了各种算法和数据结构. 写作更严谨，对某些人来说可能很难.
    ★★☆ Algorithm Design ，作者：Jon Kleinberg 和 Éva Tardos 本书围绕设计算法的技术展开. 它组织良好，以清晰易懂的语言编写. 每章都附有实际示例和有用的练习. 网络流量章节受到了很多人的高度评价. ... 教科书附带的讲座幻灯片可在其上找到 official website.
    ★★☆ The Algorithm Design Manual , 作者：Steven S. Skiena 这本书是用更易读的文字写成的. 有些人认为它比其他书籍更全面. 您还可以在其上找到一些不错的资源（包括作者自己的视频讲座） official website.
    ★★★ Algorithms ，罗伯特·塞奇维克和凯文·韦恩 这本书分类整齐，配以详尽的解释和精彩的插图. 它在一些IOI训练营中被用作教科书.
    Algorithms and Data Structures in Action ，由 Marcello La Rocca 本书提供了一种不同的算法方法，平衡了理论与更实用的角度，每章都有一节重点介绍如何将算法应用于您的日常工作或竞争性编程中可以找到的实际用例； 它还融合了经典、高级和新算法.
    ★★★ Algorithms ，杰夫·埃里克森 自行出版教科书的免费电子版，在 CC by 4.0 下获得许可. 这是一本写得很好的书，来自伊利诺伊大学理论计算机科学课程的讲义. 涵盖回溯、动态规划、贪婪的主要范式，尤其是深度图.

    Books for Mathematics¶

    ☆ 姓名 描述
    ★★☆ Discrete Mathematics and Its Applications , 肯尼斯 H. 罗森 离散数学与竞争性编程密切相关. 这本书提供了关于广泛主题的综合材料，包括：逻辑和证明、集合、函数、序列、矩阵、数论、递归、计数、概率、图、树和布尔 Alegra 仅举几例.

    ★★☆ Linear Algebra and Its Applications ，David C. Lay、Steven R. Lay、Judi J. McDonald 这本书在弥合物理系统（对于科学家和工程师）和抽象系统（对于数学家）之间的差距方面做得非常出色.
    ★★☆ Introduction to Probability , 查尔斯 M. Grinstead, J. Laurie Snell 这是一本写得很好的介绍概率的书. ...它是免费 download (pdf) （在 GNU 自由文档许可证下发布）.
    ★★☆ How to Solve It: A New Aspect of Mathematical Method ，由 G. 波利亚 一个古老的经典. 在本书中，作者提供了一种创造性地解决问题的系统方法.
    ★★☆ Intermediate Counting & Probability , 大卫帕特里克 前美国数学奥林匹克冠军大卫帕特里克的计数和概率主题，主题包括包含-排除、1-1 对应、鸽巢原理、建设性期望、斐波那契数和加泰罗尼亚数、递归、条件概率、生成函数、图论等等..

    Sites for Practice¶

    > 良好的在线评委系统/竞赛平台可供练习.

    ☆ 姓名 描述
    ★★★ Codeforces Codeforces 是最受欢迎的竞赛平台之一，如果不是的话. 目前由萨拉托夫州立大学维护，它以定期比赛和无数令人敬畏的原创问题为特色. 此外，每场比赛都会提供由作者自己编写的即时有用的教程（通常）. Codeforces 还拥有一个强大且引人入胜的社区. 总而言之，在这里确实会得到极大的学习和提高.
    ★★★ Topcoder Topcoder 成立于 2001 年.历史悠久，在技术竞赛方面被认为是最负盛名的组织之一. 数百个 SRM 产生了丰富的问题​​集. 这里的问题通常比其他问题更具挑战性，因此 Topcoder 吸引了许多精英程序员. 一年一度的 Topcoder Open (TCO) 也是一个广泛讨论的活动.
    ★★★ Google Code Jam Google Code Jam 无疑是最受推崇的编程竞赛之一. 比赛包括必须在固定时间内解决的独特编程挑战. 参赛者可以使用任何编程语言和开发环境来获得他们的解决方案.
    ★★★ AtCoder AtCoder 是一个全新但非凡的竞赛平台，由一群高评价的日本竞技程序员创建.
    ★★☆ CodeChef CodeChef 是 Directi 的一项非盈利教育计划. 它是一个具有全球竞争力的编程平台，拥有庞大的程序员社区，可帮助学生和专业人士测试和提高他们的编码技能. 其目标是为学生和专业软件开发人员提供一个练习、竞争和改进的平台. 除此之外，它的目的是在学生年轻时接触他们，并在印度灌输编程文化.
    ★★★ SPOJ SPOJ 平台以在线裁判系统为中心. 它拥有数量惊人的问题，由其问题制定者社区准备或从以前的编程竞赛中提取，其中一些是实践中的重要问题（请参阅 Problem classifiers 部分）. SPOJ 还允许高级用户根据自己的规则组织比赛.
    ★★☆ Timus Timus Online Judge 是俄罗斯最大的带有自动评判系统的编程问题档案. 问题主要是从在乌拉尔联邦大学、乌拉尔锦标赛、乌拉尔 ACM ICPC 次区域比赛和彼得罗扎沃茨克训练营举办的比赛中收集的.
    ★☆☆ HDU HDU是杭州电子科技大学维护的在线评委. 它是中国 IOI 场景中许多经典问题的发源地.
    ★★☆ Aizu Online Judge 会津在线评委是会津大学主办的竞赛平台和题库. 它有很多来自日本编程比赛的大问题.
    ★★☆ UVa 历史悠久的老式问题档案/在线法官. 数以千计的问题，包括许多经典问题，都在这里精选. 但是，强烈建议您练习 uHunt 遵循其“竞争性编程练习”部分.
    ★★☆ HackerRank HackerRank 是一家专注于为消费者和企业解决竞争性编程挑战的公司. HackerRank 的编程挑战可以用多种编程语言解决，并跨越多个计算机科学领域.
    ★★☆ POJ POJ是北大维护的一个有很多大问题的在线评委. 大多数中国竞技程序员的旅程都是从这里开始的. 该平台确实过时了，因此可能会出现神秘的编译和运行时问题.
    ★★☆ Project Euler Project Euler 提供了一组令人惊叹的数学问题. 它还举办了一个论坛，人们可以在那里进行讨论.
    ★☆☆ Hackerearth HackerEarth 是一家位于印度班加罗尔的初创科技公司，提供招聘解决方案.
    ★☆☆ Caribbean Online Judge COJ 由位于古巴的信息科学大学（UCI，西班牙语的首字母缩写词）主办. Feature ACM ICPC 和 Progressive 比赛风格，主要来自加勒比海和拉丁美洲的问题制定者，也有问题分类器和比赛日历.
    ★★☆ CS Academy CS Academy 是竞争性编程领域的新成员，它是一个成长中的在线评委，每两周举办一次比赛. 它支持实时聊天、交互式课程和集成的在线编辑器（实际有效）.
    ★★☆ Russian Code Cup 由 Mail.Ru Group 提供支持的编程比赛. 比赛包括3个资格赛、1场淘汰赛和1场决赛. 每轮参赛者有 4-8 个必须在固定时间内解决的问题.
    ★★☆ CodeFights CodeFights 是一个用于竞争性编程练习和面试准备的网站. 它具有不同难度的每日挑战、问题档案和定期（每 15 分钟）迷你锦标赛. 适合初学者.
    ★★☆ OmegaUp (Spanish) OmegaUp is a community-based website for competitive programing training. It's also where Mexican Olympiad in Informatics (OMI : Olimpiada Mexicana de Informática) is hosted.

    IDEs¶

    ☆ 姓名 平台 描述
    ★★★ Vim CLI / 跨平台 Vim 是高级程序员中最受欢迎的文本编辑器之一. 它允许仅通过击键就可以非常有效地完成文本编辑. Vim 还具有高度的可配置性、可扩展性，并且与 shell（命令行）集成得非常好. Vim 唯一的挫折是它对初学者有很高的学习曲线.
    ★★★ Emacs CLI / 跨平台 Emacs 是另一种流行的文本编辑器（或者更准确地说是开发环境）. 由于“Vim 与 Emacs”的流行，关于“Vim 与 Emacs”的争论不断被提起. 基本上 Emacs 不仅仅是一个文本编辑器. 它具有文件管理器、Web 浏览器、邮件客户端和新闻客户端等插件，允许用户直接在 Emacs 中执行这些任务. Emacs 因此“更重”，但可以说它对于初学者来说具有相对容易的学习曲线.
    ★★★ Far Manager 混合/Windows Far Manager 是 RU/CIS 竞争性编程社区中使用最广泛的编辑器. 它实际上是一个简单的文件管理器，但您可以安装 FarColorer - 一个语法高亮插件来编程. 正确配置后，Far Manager 允许您在编写代码时非常有效地在文件之间导航.
    ★★★ Code::Blocks GUI / 跨平台 Code::Blocks 是 C/C++ 的首选 IDE. 它是一个功能齐全的多功能 IDE，具有许多强大的功能. Code::Blocks 通常在编程竞赛中与 Vim 一起提供.
    ★★★ IntelliJ IDEA GUI / Cross-Platform IntelliJ IDEA is certainly one of the best IDEs for Java. It's used by most competitive programmers who use Java as their main language. Be sure to check out CHelper ，一个为编程比赛编写的非常方便的插件.
    ★★☆ Sublime Text GUI / 跨平台 Sublime Text 是一个非凡的文本编辑器. 包含多个插入符号、小地图和命令面板等强大且创新的功能，它吸引了强大且引人入胜的社区. Sublime Text 是高度可扩展的，所以一定要 Package Control 安装并探索可能是最大的插件目录之一！
    ★★☆ Eclipse GUI / 跨平台 Eclipse 是另一个很好的 Java IDE. 它是 Intellij IDEA 的一个不错的替代品（按照今天的标准，它比 IDEA 差一点）. 有时比赛出于某种原因只提供 Eclipse，所以这可能是尝试和使用 Eclipse 的一个很好的激励.
    ★★☆ CLion GUI / 跨平台 由 JetBrains 生产的 CLion - 与制造 Intellij IDEA 的公司是同一家公司，是一个强大的 C++ IDE. 免费的教育许可证可用，或者您可以尝试他们 EAP (Early Access Program) 截至 2018 年 4 月，它仍然是免费的.您可能需要关闭其代码检查功能，因为它会导致相当多的延迟.
    ★☆☆ 其他 IDE 混合 Visual Studio is the IDE to use in case you want to code in C#. ... Both Atom 和 Visual Studio Code are built with Electron (written in JavaScript) 和 therefore somewhat resource-hogging. ... CodeLite is a newly rising IDE. Beware that the load-up 和 project-creation times can be extraordinary.
