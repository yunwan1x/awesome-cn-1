    Awesome Arch

    Awesome Arch ¶

    >很棒的项目的很棒的清单.

    Arch Linux 是独立开发的x86-64通用GNU / Linux发行版，具有足够的通用性以适合任何角色. 开发侧重于简单性，简约性和代码优美性. Arch是作为最小的基础系统安装的，由用户配置，通过仅安装其独特目的所需或期望的组件，在其上组装他们自己的理想环境.

    Arch-based projects¶

    Linux¶

    > Linux发行版受到Arch Linux的强烈启发.

    • Arch Linux 32 -社区维护了对Arch Linux的32位支持.
    • Arch Linux ARM -用于ARM计算机的Linux发行版.

    Not Linux¶

    >基于Arch Linux的非Linux项目.

    • PacBSD -轻巧灵活的BSD发行版，为使用pacman软件包管理器提供了基础系统.
    • Arch Hurd -Arch Linux的衍生作品，通过针对i686架构进行了优化的软件包将其移植到GNU Hurd系统.
    • ArchWSL -ArchLinux作为WSL实例.

    Arch-based distros¶

    Desktop¶

    >通用的基于Arch Linux的Linux发行版.

    • Manjaro -专业制作的基于Linux的操作系统，可以替代Windows或macOS.
    • ArcoLinux -功能齐全的发行版.
    • ArchBang Linux - A lightweight distribution based on Arch Linux.
    • Archcraft OS -简约的Linux发行版，专注于美学并且基于Arch Linux.
    • Archman -基于Arch Linux的发行版，具有Calamares系统安装程序和预配置的桌面环境.
    • Bluestar Linux -基于Arch Linux的GNU / Linux发行版.
    • EndeavourOS -基于Arch Linux的滚动发行版Linux发行版.
    • ArchLabs Linux -基于Arch Linux的发行版，以Openbox窗口管理器为主要桌面界面.
    • Namib GNU/Linux -基于Arch Linux发行版（并与之兼容）的桌面操作系统.
    • ArchEX -基于Arch Linux的新的（190612）Linux live DVD. 座右铭是KISS（保持简单愚蠢.
    • Arch XFerience -基于Arch Linux的现代OS，具有大量软件包和功能，可用于日常操作.
    • Ctlos Linux -基于Arch Linux的发行版.
    • MagpieOS -获得Arch的全新体验.
    • Netrunner Rolling -基于Manjaro / Arch Linux的发行版.
    • PuppyRus-A -基于Arch，Puppy，Porteus的轻便，便携式和灵活的Linux.
    • TeLiSc OS -基于Arch Linux的简单GNU / Linux发行版，可为您提供带有i3窗口管理器的稳定Arch Linux.
    • Qubes OS (Archlinux Template) -专注于安全性的桌面操作系统，旨在通过隔离来提供安全性（非官方的Archlinux模板）.

    Specific purpose¶

    >具有特定用途的基于Arch Linux的Linux发行版.

    • BlackArch Linux -（安全/取证）-基于Arch的Linux发行版，专门用于渗透测试人员和安全研究人员.
    • ArchStrike -（安全性）-基于Arch Linux的面向安全专业人员和研究人员的发行版.
    • SystemRescueCd -（Data Rescue）-可引导CD-ROM或USB驱动器上的基于Arch的Linux系统，用于在崩溃后修复系统和数据.
    • MorpheusArch Linux -（Data Rescue）-基于Arch Linux的发行版.
    • LinHES -（MythTV）-尝试使GNU / Linux和MythTV的安装尽可能简单.
    • UBOS -（服务器）-Linux发行版，旨在极大地减少设置和维护通用网络服务所需的时间.
    • TalkingArch -（辅助）-Arch Linux实时ISO映像的重制版，已修改为包括针对盲人和视障用户的语音和盲文输出.
    • Archphile -（Raspberry Pi / ARM / Audio）-嵌入式板的另一种Linux音频发行版.
    • BBQLinux -（Android开发人员）-专为Android开发人员设计的用户友好型Linux发行版. 它具有构建AOSP或基于AOSP的发行版（如LineageOS或OmniROM）所需的一切.
    • RaspArch -（Raspberry Pi / ARM）-Arch Linux ARM的“重新制作”.

    Pacman-based Linux distros¶

    >使用pacman作为软件包管理器的Linux发行版.

    • Chakra GNU/Linux -最初是从Arch Linux派生而来的用户友好且功能强大的发行和现场CD.
    • KaOS -KaOS是专注于Qt和KDE的独立发行版.

    SystemD-free Arch-based distros¶

    >基于Arch Linux的Linux发行版，实现了SystemD的替代init.

    • Obarun -基于Arch Linux的发行版，具有S6初始化软件来代替systemd.
    • Artix Linux -Arch-OpenRC和Manjaro-OpenRC项目的分支（或作为自治项目的延续）.
    • Hyperbola GNU/Linux-libre -由社区推动的努力，以提供稳定，安全，简单，轻巧的完全免费（如自由）操作系统，并尝试通过长期支持（LTS）保持简单愚蠢（KISS）.
    • Parabola GNU/Linux-libre -Arch Linux的非正式“自由”变体.
    • Alfheim Linux -基于系统的免费Arch基于Linux.

    Arch Linux Installers¶

    >专门为促进Linux Arch的安装而创建的安装程序.

    • Anarchy -简单直观的Arch Linux安装程序.

    AUR Helpers¶

    Search and download¶

    • Auracle （C ++）-用于Arch Linux用户存储库的灵活命令行客户端.
    • pbget （Python）-从Git，ABS和AUR中检索PKGBUILD和本地源文件，以获取makepkg.
    • repoctl （开始）-轻松管理本地Arch Linux存储库.
    • yaah （重击）-Archlinux用户存储库（AUR）的帮助程序.

    Search and build¶

    • Aurutils （重击）-AUR的帮助器工具.
    • Bauerbill （Python）-Powerpill的扩展，具有AUR和ABS支持.
    • PKGBUILDer （Python）-Python 3中的AUR帮助器（和库）.
    • repofish （重击）-用于管理本地archlinux存储库的简单shell脚本.
    • RUA （Rust）-用于Arch Linux的构建工具，提供控制，查看和监禁的构建选项.

    Pacman wrappers¶

    • Aura （Haskell）-用于Arch Linux和AUR的安全，多语言包管理器.
    • Pacaur （重击）-AUR帮助程序，可最大程度地减少用户交互.
    • Pakku （Nim）-具有AUR支持的pacman包装器.
    • pikaur （Python）-具有最小依赖性的AUR帮助器. 一次查看所有PKGBUILD，然后无需用户交互即可全部构建它们.
    • Trizen （Perl）-轻量级AUR软件包管理器.
    • Yay （Go）-另一个酸奶，Go编写的AUR帮助程序.

    Graphical¶

    • Argon （Python | GTK3）-用于Arch Linux的轻量级GUI软件包管理器.
    • Cylon （Bash | TUI）-CLI TUI菜单驱动的bash shell脚本，支持基于Arch的Linux发行版的更新，维护，备份和系统检查.
    • Pamac （Vala | GTK3）-用于Manjaro Linux的图形软件包管理器，具有Alpm，AUR，Appstream，Flatpak和Snap支持.
    • Pakku GUI （Python | GTK3）-使用pakku到（install | remove | upgrade）软件包的基本GTK接口.
    • PkgBrowser （Python | Qt5）-用于浏览“ pacman”数据库和AUR的实用程序.
    • Octopi （C ++ | Qt5）-Arch Linux软件包管理工具“ pacman”的图形用户界面.
    • Yup （转到| TUI）-具有ncurses功能以及更好的搜索和排序功能的Arch Linux AUR Helper.

    Maintenance¶

    • aur-out-of-date -确定上游版本的过期AUR软件包.
    • aurpublish -用于Arch用户存储库的PKGBUILD管理框架.
    • devtools -用于构建和维护官方存储库软件包的Arch Linux发行版的工具.
    • pkgbuild-watch -提醒您上游活动.
    • pkgoutofdate -检查Arch软件包是否已过期.

    Other¶

    • aur-talk -提取和显示AUR软件包注释的脚本.
    • aurvote-utils -一组用于管理AUR投票的实用程序.
    • haskell-aur -Haskell库，用于访问Aurweb RPC接口.
    • package-query -查询libalpm和AUR的工具.
    • python3-aur -与AUR相关的模块和帮助程序实用程序（aurploader，aurquery，aurtomatic）.
    • raur -用于访问Aurweb RPC接口的Rust库.

    Other projects¶

    >与Arch Linux链接或受其启发的项目不属于任何先前的类别.

    • Termux Arch -使用Termux在手机上运行Arch Linux ARM.
    • archlinux/base -基于Arch Linux的docker映像.
    • Arch Linux on EC2 -基于Arch Linux的Amazon EC2 AMI
