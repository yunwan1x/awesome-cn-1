    Awesome AWS ¶

    精选的AWS库，开放源代码存储库，指南，博客和其他资源的精选列表.

    Appendix of Core Services¶

    官方服务的附录，按服务类别分组.

    Services in Plain English¶

    • Amazon Web Services in Plain English -娱乐和教育，是社区的贡献.

    Compute Services¶

    • Auto Scaling -根据策略，时间表和运行状况检查启动或终止EC2实例.
    • Batch -大规模运行批处理作业.
    • Blox -用于在ECS上构建自定义调度程序的开源项目.
    • EC2 Container Service (ECS) -在EC2实例上支持Docker容器.
    • EC2 Systems Manager -轻松配置和管理EC2和本地系统.
    • Elastic Beanstalk -提供云中应用程序的快速部署和管理.
    • Elastic Compute Cloud (EC2) -使用Xen提供可扩展的虚拟专用服务器.
    • Elastic GPUs -将低成本GPU附加到EC2实例以进行图形加速.
    • Elastic Load Balancing (ELB) -在多个EC2实例之间自动分配传入流量.
    • Lambda -运行代码以响应事件并自动管理EC2实例.
    • Lightsail -启动和管理简单的虚拟专用服务器.
    • Virtual Private Cloud (VPC) -创建逻辑隔离的EC2实例集，可以使用VPN连接将其连接到现有网络.

    Networking Services¶

    • Direct Connect -提供与AWS的专用连接，以实现更快，更便宜的数据吞吐量.
    • Elastic Load Balancing (ELB) -在多个EC2实例之间自动分配传入流量.
    • Route 53 -提供高可用性和可伸缩的域名系统（DNS）Web服务.
    • Virtual Private Cloud (VPC) -创建逻辑隔离的EC2实例集，可以使用VPN连接将其连接到现有网络.

    Enterprise Applications¶

    • WorkDocs -提供全面管理，安全的企业存储和共享服务.
    • WorkMail -提供托管的电子邮件和日历服务.
    • WorkSpaces -为最终用户提供基于云的桌面体验.
    • Workspaces Application Manager (WAM) -简化了WorkSpaces的部署和管理.

    Analytics Services¶

    • Athena -立即在S3上查询数据.
    • Data Pipeline -通过在服务之间处理和移动数据来提供工作负载管理.
    • Elastic MapReduce (EMR) -托管在EC2和S3上运行的Hadoop和Spark框架.
    • Elasticsearch Service (ES) -Managed Elasticsearch，一个流行的开源搜索和分析引擎.
    • Glue -准备数据并将其加载到数据存储中.
    • Kinesis -在大型分布式数据流上提供实时数据处理.
    • Kinesis Analytics -在流数据上编写标准的SQL查询，而无需学习任何新的编程技能.
    • Kinesis Firehose -捕获流数据并将其自动加载到S3和Redshift中.
    • Quicksight -提供云驱动的商业智能，成本仅为传统BI解决方案的1/10.
    • Redshift -通过列存储和多节点计算提供PB级的数据仓库.

    Artificial Intelligence¶

    • Lex -通过语音或文字建立对话界面.
    • Machine Learning -提供托管的机器学习技术.
    • Polly -将文字变成逼真的语音.
    • Rekognition -基于深度学习的图像分析.

    Management Tools¶

    • CloudFormation -提供用于提供其他资源的基于文件的界面.
    • CloudTrail -提供所有活动的日志.
    • CloudWatch -从EC2开始，提供对AWS云资源和应用程序的监视.
    • Command Line Interface (CLI) -提供CLI来管理所有服务.
    • Config -提供所有资源的详细视图.
    • Management Console (AWS Console) -基于Web的界面，用于管理所有服务.
    • OpsWorks -使用Chef提供EC2服务的配置.
    • Personal Health Dashboard -您对服务健康状况的个性化看法.
    • Service Catalog -服务目录使IT管理员可以创建，管理批准的产品组合并将其分发给最终用户，然后他们可以在个性化门户中访问所需的产品.

    Security and Identity Services¶

    • Certificate Manager -使您轻松配置，管理和部署SSL / TLS证书以与AWS服务一起使用.
    • CloudHSM -通过使用AWS云中的专用硬件安全模块（HSM）设备，帮助满足公司，合同和法规对数据安全的合规性要求.
    • Directory Service -一项托管服务，可让您将资源与现有的本地Microsoft Active Directory连接或在AWS Cloud中设置新的独立目录.
    • Identity and Access Management (IAM) -隐式服务，用于对各种服务的访问进行身份验证的身份验证基础结构.
    • Inspector -自动化安全评估服务，可帮助改善AWS上部署的应用程序的安全性和合规性.
    • Key Management Service (KMS) -一项托管服务，可让您轻松创建和控制用于加密数据的加密密钥.
    • Shield -托管DDoS保护.
    • WAF -监视和管理CloudFront分布的Web应用程序防火墙服务.

    Internet of Things Service¶

    • IoT -支持通过MQTT和HTTP在Internet连接的事物（例如传感器，执行器，嵌入式设备或智能设备）与AWS云之间进行安全的双向通信.

    Mobile Services¶

    • API Gateway -发布，维护和保护Web服务API的服务.
    • Cognito -提供用户身份和数据同步.
    • Device Farm -适用于物理设备上的iOS，Android和Fire OS应用程序的应用程序测试服务.
    • Mobile Analytics -收集，可视化和了解应用使用数据的服务.
    • Mobile Hub -提供集成控制台，可帮助您构建，测试和监视您的移动应用程序.
    • Pinpoint -针对移动应用的定向推送通知.

    • Simple Notification Service (SNS) -为应用程序提供托管的多协议“推送”消息.

    • Elastic Block Store (EBS) -为EC2提供持久的块级存储卷.

    • Elastic File System (EFS) -用于EC2实例的文件存储服务.
    • Glacier -提供低成本，长期存储选项，用于归档数据.
    • Import/Export -使用便携式存储设备进行传输，加快将大量数据移入和移出AWS的速度.
    • Simple Storage Service (S3) -提供基于Web服务的存储.
    • Storage Gateway -具有基于云的备份的iSCSI块存储虚拟设备.

    Databases¶

    • Aurora -与MySQL和PostgreSQL兼容的关系数据库，具有改进的性能.
    • DynamoDB -提供由SSD支持的可扩展，低延迟的NoSQL在线数据库服务.
    • ElastiCache -为Web应用程序（Memcached，Redis）提供内存中缓存.
    • Redshift -通过列存储和多节点计算提供PB级的数据仓库.
    • Relational Database Service (RDS) -提供具有MySQL，Oracle，SQL Server，PostgreSQL和MariaDB支持的可伸缩数据库服务器.
    • Schema Conversion Tool -该应用程序可帮助您将数据库架构从Oracle或Microsoft SQL Server数据库转换为RDS MySQL数据库实例或Aurora数据库集群.
    • SimpleDB -允许开发人员对结构化数据运行查询.

    Application Services¶

    • API Gateway -发布，维护和保护Web服务API的服务.
    • AppStream -针对应用和游戏的灵活，低延迟的流媒体服务.
    • DevPay -提供帐单和帐户管理.
    • Elastic Transcoder (ETS) -提供S3托管视频的视频转码.
    • Flexible Payments Service (FPS) -提供小额付款的界面.
    • Simple Email Service (SES) -提供批量和交易电子邮件发送.
    • Simple Notification Service (SNS) -为应用程序提供托管的多协议“推送”消息.
    • Simple Queue Service (SQS) -提供Web应用程序的托管消息队列.
    • Simple Workflow (SWF) -用于构建可伸缩，弹性应用程序的工作流服务.
    • Step Functions -协调分布式应用程序的组件.

    Developer Tools¶

    • CodeBuild -构建和测试代码.
    • CodeCommit -托管的Git版本控制服务.
    • CodeDeploy -提供自动代码部署到EC2实例.
    • CodePipeline -持续交付服务.
    • Command Line Interface (CLI) -提供CLI来管理所有服务.
    • X-Ray -分析和调试您的应用程序.

    Miscellaneous Services¶

    • Fulfillment Web Service -为卖家提供程序化Web服务，以使用Amazon Fulfillment将商品运入和运出Amazon.
    • Mechanical Turk -管理分布在许多人中的小型工作单元.
    • Partner Network (APN) -为合作伙伴提供技术信息以及销售和市场支持，以增加商机.
    • Product Advertising API -提供对产品数据和电子商务功能的访问.
