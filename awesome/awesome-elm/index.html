    真棒榆木


    由社区驱动的有用Elm教程，库和软件的列表. 受启发 awesome 列出的东西. 随时改进此列表.

    -贡献准则

    Learn¶

    了解这真棒的东西是什么

    • Official tutorial —常规信息和带有示例的深入指南.
    • Elm in Action —来自曼宁出版社（Manning Publications）的针对榆木初学者的深入书籍.
    • Architecture Tutorial -如何创建可与您的应用很好地缩放的模块化Elm代码.
    • Exercism Elm Track -榆木练习集.
    • Learn you an Elm -Elm教程，提供详尽的示例和说明.
    • Beginning Elm -对Elm编程语言的简要介绍.
    • Elm patterns -榆树的常见模式的集合.
    • Elm Koans -学习榆木的练习.
    • Learn Elm -探索美丽的编程语言，使前端Web应用程序能够愉快地构建和维护！
    • Learn Elm in Y Minutes -语法和功能概述. 榆木页面 learnxinyminutes.com
    • Elm Maybe - Dealing with null/Nothing -使用Maybe类型，并带有注释良好的代码示例.
    • Programming Elm -实用程序员的详尽书籍，涵盖了基础知识和高级概念.
    • Elm cheat sheet -语法和功能概述.
    • Elm Tutorials on Codementor -关于使用Elm构建Web应用程序的两个教程.
    • Elm programming language -Elm作为一种编程语言的简要概述.
    • Elm: A Beginners' Guide to Elm and Data -榆木和数据初学者课程
    • Practical Elm for a Busy Developer -一本关于开发Elm应用程序的实际方面的入门书籍.
    • Haskell to Elm -针对Elm与Haskell不同的地方收集了一些示例，针对的是来自Haskell背景的Elm初学者.
    • A nice app on Elm street -榆树简介
    • The Elmish Book 在中构建Web应用程序 F# 遵循Elm Architecture的首要原则.
    • Elm patterns -编码模式的集合.
    • Codings hints -Evan的自述文件列表.
    • Ellies catalog - A Collection of small example in Ellie.
    • Elm Cookbook -有关榆木的电子书.

    Outdated Tutorials and books (Elm 0.18 or earlier)¶

    • Elm: Building Reactive Web Apps -了解如何使用Elm构建响应式Web应用程序.

    • Elm Tutorial by Auth0 -有关在Elm中构建应用程序的教程，从身份验证到调用API.

    Tools¶

    与榆木相关的有用工具.

    • Elm Catalog -Elm工具的目录.
    • Dependabot -为您的elm.json自动更新PR.
    • Elm Analyse -用于Elm编程语言的Linter.
    • Elm Doc Preview -Elm离线文档预览器.
    • Elm Format -自动Elm代码格式化程序遵循 Elm Style Guide.
    • Elm JSON -安装，升级和卸载Elm依赖项
    • Elm Live -Elm的灵活开发服务器. 包括实时重新加载！
    • Elm Oracle -查询有关榆木源文件中值的信息. 大多数编辑器插件使用.
    • Elm Pages -静态网站生成器，可预渲染HTML并合并到Elm客户端应用程序中（here is a brief comparison with elmstatic).
    • Elm Review -Elm编程语言的代码审阅者.
    • Elmstatic -基于Elm的静态网站生成器.
    • type-o-rama -JS类型系统的可移植性.
    • Html to Elm -将HTML转换为Elm.

    Code Generators¶

    • HTML to Elm -将HTML转换为Elm Html. 将应用程序移植到Elm时很有用.
    • OpenApi Generator -为Elm生成OpenApi类型.
    • Elm TS Interop -在Elm和TypeScript之间建立编码器/解码器.
    • Elm TypeScript Interop -从Elm生成TypeScript定义（“ Elm TS Interop”是对此的改进版本）.
    • Quicktype -从JSON生成JSON解码器和编码器
    • JSON Schema to Elm -根据JSON模式规范生成Elm类型，JSON解码器，JSON编码器和模糊测试
    • Elm Bridge -从Haskell生成榆木类型
    • haskell-to-elm -从Haskell类型生成Elm类型，编码器和解码器
    • PostCSS Elm Tailwind -在榆木中放些尾风

    Run Elm¶

    • Ellie -Elm Live编辑器
    • Elm Editor -高级Elm Live编辑器
    • Online REPL -ELM在线REPL
    • run-elm —从命令行运行Elm代码
    • elm-instant -atom软件包，可从编辑器尝试您的elm代码. 提供可视的REPL和预览窗格.
    • Glitch -在浏览器中构建快速，全栈的Web应用程序.

    Compile and bundle¶
