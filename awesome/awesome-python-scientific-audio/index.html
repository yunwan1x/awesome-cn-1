    Scientific Audio

    Python for Scientific Audio ¶

    该存储库的目的是创建与Python相关的，用于音频/音乐应用科学研究的Python软件/工具的全面，精选列表.

    Audio Related Packages¶

    -包裹总数：66

    Read-Write¶

    • audiolazy :octocat: -用于Python的富有表现力的数字信号处理（DSP）软件包.
    • audioread :octocat: -跨库（GStreamer +核心音频+ MAD + FFmpeg）音频解码.
    • mutagen :octocat: -读取和写入各种格式的所有音频元数据.
    • pyAV :octocat: -PyAV是FFmpeg或Libav的Python绑定.
    • (Py)Soundfile :octocat: -基于libsndfile，CFFI和NumPy的库.
    • pySox :octocat: -袜子包装.
    • stempeg :octocat: -读/写STEMS多流音频.
    • tinytag :octocat: -读取MP3，OGG，FLAC和Wave文件的音乐元数据.
    • audiomate :octocat: -加载不同类型的音频数据集.

    Transformations - General DSP¶

    • acoustics :octocat: -声学师的有用工具.
    • AudioTK :octocat: -DSP过滤器工具箱（许多过滤器）.
    • AudioTSM :octocat: -实时音频时标修改程序.
    • Gammatone :octocat: -Gammatone滤波器组的实现.
    • pyFFTW :octocat: -FFTW（3）的包装器.
    • NSGT :octocat: -非平稳gabor变换，常数q.
    • MDCT :octocat: -MDCT转换.
    • pydub :octocat: -通过简单易用的高级界面处理音频.
    • pytftb :octocat: -MATLAB时频工具箱的实现.
    • pyroomacoustics :octocat: -室内声学模拟（RIR发生器）
    • PyRubberband :octocat: -包装 rubberband 进行音调转换和时间拉伸.
    • PyWavelets :octocat: -Python中的离散小波变换.
    • Resampy :octocat: -采样率转换.
    • SFS-Python :octocat: - Sound Field Synthesis Toolbox.
    • STFT :octocat: -用于短时傅立叶变换的独立程序包.

    Feature extraction¶

    • aubio :octocat: -特征提取器，用C，Python接口编写.
    • audiolazy :octocat: -实时音频处理库，通用.
    • essentia :octocat: -与音乐相关的低级和高级功能提取器，基于C ++，包括Python绑定.
    • muda :octocat: -音乐数据增强.
    • pydiogment :octocat: -音频数据增强.
    • python_speech_features :octocat: -ASR的通用语音功能.
    • pyYAAFE :octocat: -YAAFE功能提取器的Python绑定.
    • speechpy :octocat: -语音处理和识别库，目前主要是特征提取.

    Speech Processing¶

    • aeneas :octocat: -基于MFCC + DTW，35种以上语言的强制对齐器.
    • deepspeech :octocat: -预训练的自动语音识别.
    • gentle :octocat: -基于Kaldi的强制对准器.
    • Parselmouth :octocat: -Python接口 Praat 语音和语音分析，合成和操纵软件.
    • persephone :octocat: -自动音素转录工具.
    • pyannote.audio :octocat: -用于说话人区分的神经构建块.
    • pyAudioAnalysis² :octocat: -特征提取，分类，数字化.
    • py-webrtcvad :octocat: -与WebRTC语音活动检测器的接口.
    • pypesq :octocat: -用于PESQ分数计算的包装器.
    • pystoi :octocat: -短期目标可理解性度量（STOI）.
    • PyWorldVocoder :octocat: -Morise的World Vocoder包装.
    • Montreal Forced Aligner :octocat: -基于Kaldi（HMM），英语的强制对准器（可以训练其他对准器）.
    • SIDEKIT -说话者和语言识别.
    • SpeechRecognition :octocat: -用于多个在线和离线ASR引擎和API的包装器.

    Environmental Sounds¶

    • sed_eval :octocat: -声音事件检测的评估工具箱

    Perceptial Models - Auditory Models¶

    • cochlea :octocat: -内耳模型.
    • Brian2 :octocat: -刺神经网络模拟器，包括耳蜗模型.
    • Loudness :octocat: -感知的响度，包括Zwicker，摩尔/格拉斯伯格模型.
    • pyloudnorm :octocat: -音频响度计和归一化，实现ITU-R BS.1770-4.
    • Sound Field Synthesis Toolbox :octocat: - Sound Field Synthesis Toolbox.

    Source Separation¶

    • commonfate :octocat: -共同的命运模型和转变.
    • NTFLib :octocat: -稀疏的Beta散度张量分解.
    • NUSSL :octocat: -整体源分离框架，包括DSP方法和深度学习方法.
    • NIMFA :octocat: -非负矩阵分解的几种形式.

    Music Information Retrieval¶

    • Catchy :octocat: -用于计算挂钩的语料库分析工具.
    • Madmom :octocat: -MIR软件包非常注重节拍检测，起音检测和和弦识别.
    • mir_eval :octocat: -各种MIR任务的共同分数. 还包括bss_eval实现.
    • msaf :octocat: -音乐结构分析框架.
    • librosa :octocat: -常规音频和音乐分析.

    Deep Learning¶

    • Kapre :octocat: -大声音频预处理器
    • TorchAudio :octocat: -PyTorch音频加载器

    Symbolic Music - MIDI - Musicology¶

    • Music21 :octocat: -计算机辅助音乐学工具包.
    • Mido :octocat: -实时MIDI包装器.
    • mingus :octocat: -带有MIDI文件和播放支持的高级音乐理论和注释包.
    • Pretty-MIDI :octocat: -以良好/直观的方式处理MIDI数据的实用程序功能.

    Realtime applications¶

    • PYO :octocat: -实时音频dsp引擎.
    • python-sounddevice :octocat: -PortAudio包装器通过NumPy提供实时音频I / O.

    Web Audio¶

    • TimeSide (Beta) :octocat: -高级音频分析，成像，代码转换，流传输和标记.

    Audio related APIs and Datasets¶

    • beets :octocat: -音乐图书馆经理和 MusicBrainz 标签.
    • dsdtools :octocat: -解析并处理 demixing secrets dataset.
    • medleydb :octocat: -解析 medleydb 音频和注释.
    • Soundcloud API :octocat: -包装 Soundcloud API.
    • Youtube-Downloader :octocat: -下载youtube视频（和音频）.

    Wrappers for Audio Plugins¶

    • VamPy Host -接口编译的鞋面插件.

    Tutorials¶

    • Whirlwind Tour Of Python [：octocat：]（https://github.com/jakevdp/WhirlwindTourOfPython ）-针对研究人员和开发人员的Python基本要素的快速介绍.
    • Introduction to Numpy and Scipy :octocat: -高度推荐的教程，涵盖了科学的Python生态系统的大部分内容.
    • Numpy for MATLAB® Users -等效的python功能的简短概述.
    • MIR Notebooks :octocat: -收集用于音乐信息检索（MIR）的教学iPython笔记本.
    • Selected Topics in Audio Signal Processing -作为iPython笔记本进行练习.

    Books¶

    • Python Data Science Handbook -杰克·范德普拉斯（Jake Vanderplas），《名著》和随附的教程笔记本.
    • Fundamentals of Music Processing -MeinardMüller，随附Python练习.
