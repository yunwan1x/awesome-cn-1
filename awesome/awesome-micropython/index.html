    Awesome micropython

    精选的很棒的MicroPython库，框架，软件和资源的列表.

    MicroPython 是Python 3编程语言的一种精简而高效的实现，其中包括Python标准库的一小部分，并且经过优化可在微控制器和受限环境中运行.

    Libraries¶

    您可以在其他地方找到MicroPython库：

    • PyPi -此过滤器仅显示PyPi上的MicroPython库. 注意：您不能通过pip安装micropython库. 看 MicroPython docs 有关upip的更多信息.
    • GitHub Search -在GitHub上搜索包含MicroPython的存储库.
    • GitHub Topic - MicroPython -浏览GitHub主题以查找标记有MicroPython的项目.
    • Libraries.io -Libraries.io查询MicroPython.
    • GitLab Explore -在GitLab上浏览存储库.

    AI¶

    • MicroMLP -用于MicroPython的微神经网络多层感知器（用于ESP32和Pycom模块）.

    Analytics¶

    • uMath -微控制器的计算机代数.
    • micropython-ulab -适用于MicroPython的类似numpy的快速向量模块.
    • micropython-fourier -MicroPython的嵌入式ARM汇编器中的快速傅立叶变换.
    • ulinalg -小型矩阵处理模块，具有一些专门用于MicroPython（Python3）的线性代数运算.
    • micropython-mtx -MicroPython上的快速矩阵乘法和线性求解器.
    • micropython-vec -MicroPython上的向量运算.

    Audio¶

    • JQ6500 -JQ6500 UART MP3模块的驱动程序.
    • KT403A-MP3 -DFPlayer Mini和Grove MP3 v2.0使用的KT403A驱动程序.
    • micropython-buzzer -在蜂鸣器上播放诺基亚合成和中音文件.
    • micropython-dfplayer -使用UART的DFPlayer Mini驱动程序.
    • micropython-longwave -MicroPython开发板的WAV播放器.
    • micropython-vs1053 -VS1053b MP3播放器的异步驱动程序.
    • micropython-midi -MicroPython的Midi实现示例.
    • upy-rtttl -铃声文本传输语言（RTTTL）的Python分析器.

    Communications¶

    Bluetooth¶

    • PyBoard-HC05-Android -Pyboard HC05蓝牙适配器示例应用程序.

    CAN¶

    • micropython-spacecan -Spacecan是用于嵌入式系统的SpaceCAN协议的MicroPython实现.

    Cryptography¶

    • mpyaes -用于AES加密的MicroPython模块.
    • micropython-aes -带有纯python实现的AES算法.

    DNS¶

    • ICantBelieveItsNotDNS -“我不敢相信这不是DNS！” （ICBIND）是用MicroPython编写的ESP8266的权威DNS服务器.
    • MicroDNSSrv -用于MicroPython的微型DNS服务器，可以简单地响应带有或不带有通配符的多域上的A查询（用于Pycom模块和ESP32）.

    Ethernet¶

    • Official wiznet5k -WIZnet5x00系列以太网控制器的官方驱动程序.

    FTP¶

    • micropython-ftplib -用于MicroPython的FTP客户端库.
    • FTP-Server-for-ESP8266-ESP32-and-PYBD -适用于MicroPython平台的ESP8266 / ESP32 / PYBD的小型FTP服务器.
    • MicroFTPServer -可以在带有MicroPython的ESP8266上运行的最小FTP服务器.

    GPS¶

    • micropyGPS -功能齐全的GPS NMEA句子解析器.
    • micropython-gnssl76l -用于Quectel GNSS L76-L（GPS）的MicroPython I2C驱动程序.
    • mpy-agps -辅助位置服务（AGPS）的MicroPython实现.

    GSM¶

    • micropython-upyphone -使用pyboard和sim800l的gsm手机.

    IoT¶

    • microhomie -用于物联网的Homie MQTT约定的MicroPython实现.
    • uPyEcho -在ESP32上使用MicroPython仿真的Belkin WeMo设备，可与Amazon Echo（Alexa）一起使用.
    • SonosRemote -用于在ESP8266上运行并使用Sonos HTTP API的Sonos安装的遥控器.
    • micropython-home-assistant -基于MicroPython的脚本，可扩展您的家庭助理驱动的家庭自动化项目.

    IR¶

    • micropython-necir -用于TL1838红外接收器LED的NEC红外捕获.
    • Micropython-IR -Pyboard红外遥控嗅探和重放.
    • micropython_ir -非阻塞设备驱动程序，可从IR远程接收和用于IR“ blaster”应用程序.
    • micropython-amg88xx -Grid-EYE热红外阵列传感器（Adafruit 3538）的驱动程序.
    • micropython-ys-irtm -用于YS-IRTM 5V NEC红外UART收发器的MicroPython示例.
    • esp8266_ir -通过websocket控制IR信号.

    LoRaWAN¶

    • uPyLoRaWAN -使用MicroPython的ESP32符合LoRa和LoRaWAN.
    • DeLoReAN -DeLoReAN是（专用）（Lo）RaWAN节点（重新）编码（A）ny（N）FC标签.

    MDNS¶

    • micropython-mdns -MDNS的纯python实现，支持服务发现.

    MQTT¶

    • MQBoard -在MicroPython板上（主要在ESP32上）使用MQTT和asyncio的微框架.
    • micropython-mqtt -适用于MicroPython的MQTT库和工具.
    • pysmartnode -MicroPython Smarthome框架.
    • umqtt_aws_iot -使用MicroPython将UMQTT消息发布到AWS IoT.
    • sonoff-mqtt by davea -使用MQTT来控制Sonoff / ESP8266的MicroPython脚本.
    • micropython-sonoff-switch -使用MicroPython为iTead Sonoff开关实现了MQTT可控开关.
    • micropython-thingspeak-mqtt-esp8266 -使用带有在ESP8266 / ESP32平台上运行的MicroPython的MQTT发布和订阅Thingspeak.
    • uMQTT -MQTT在WiPy板上为MicroPython发布.

    NTP¶

    • micropython-ntpd -MicroPython中ntpd的实现.
    • micropython_ntpserver -为MicroPython编写的NTP服务器.
    • micropython-ntpclient -使用uasyncio的MicroPython的NTP客户端.

    OneWire¶

    WiFi¶

    • tinyweb -适用于MicroPython的简单轻便的HTTP异步服务器.
    • upy-websocket-server -MicroPython（ESP8266）Websocket服务器实现.
    • micropython-captive-portal -用于MicroPython的强制门户演示.
    • uPyPortal -使用ESP32（WeMos）的MicroPython专属门户.
    • ESP8266WebServer -适用于MicroPython的ESP8266 Web服务器.
    • microCoAPy -MicroPython中CoAP（受限应用协议）的微型客户端/服务器实现.
    • micropyserver -MicroPyServer是用于MicroPython项目的简单HTTP服务器.

    • micropython-noggin -适用于MicroPython的非常简单的Web服务器.
