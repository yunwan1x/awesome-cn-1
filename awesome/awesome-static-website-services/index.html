    静态网站服务

    Awesome Static Website Services ¶

    >精选的静态网站服务列表.

    受启发 awesome 列出的东西.

    Audio¶

    • SoundCloud -带有嵌入式播放器的音频托管. 最多3个小时的内容是免费的.
    • Mixcloud -具有无限上传和嵌入式播放器的音频托管.
    • Spotify -您可以将任何歌曲，专辑或播放列表嵌入其中 Spotify Play Button.

    Books¶

    • Google Books -允许您在Google图书库中显示图书.
    • Goodreads API and widgets -允许您访问任何Goodreads数据. 在您的“窗口小部件”标签上可以找到窗口小部件 settings page.

    Calendar and Scheduling¶

    • Google Calendar -可嵌入的日历，您可以与其他人协作.
    • Booking.js -美丽的嵌入式预订小部件.
    • zenplanner -付费-在线安排健身时间.

    Images¶

    • Flickr -Yahoo在线照片托管.
    • Cloudinary - Image hosting, manipulation and delivery.

    Maps¶

    • Google Maps -易于嵌入Google地图.
    • Mapbox -真正漂亮的可嵌入地图.

    Presentations¶

    • Prezi -具有可缩放和旋转的真正过渡的在线演示.
    • Reveal.js -HTML表示框架.
    • Slides.com -一个用于创建，演示和共享幻灯片的地方.
    • SpeakerDeck -将幻灯片以PDF格式上传，并获得在线共享演示文稿.

    Video¶

    • Mux -付费-直接向客户端播放视频的API. 还可为实时流供电.
    • YouTube -可无限上传的可嵌入视频.
    • Vimeo -付费-无广告的可嵌入视频.
    • Vevo -可嵌入的音乐视频.
    • Wistia -免费方案最多只能有25个视频.

    Code¶

    • Codepen -一个可嵌入的前端代码示例的游乐场.
    • JS Bin -可嵌入的前端代码示例.
    • JSFiddle -可嵌入的前端代码示例.
    • highlight.js -网络语法突出显示.

    Functions as a Service¶

    • 1Backend -在几秒钟内部署您的后端. 包括免费套餐. 开源.
    • AWS Lambda -AWS Lambda使您无需配置或管理服务器即可运行代码. 您只需为您消耗的计算时间付费
    • Google Cloud Functions -创建用于响应Cloud事件的单一用途的独立功能，而无需管理服务器或运行时环境
    • Webtask by Auth0 -使用简单的HTTP在服务器上调用代码，到目前为止比Lambda或Google的设置更容易
    • Azure Functions -由Microsoft-与Azure云上的Lambda相同的前提
    • IronWorkers -by Iron.io-在多语言容器化环境中运行代码，具有无限制的规模和简单的定价
    • IronFunctions -Iron.io撰写-IronFunctions是适用于任何云（私有云，公共云或混合云）的开源无服务器计算平台.
    • OpenWhisk by IBM -他们BlueMix托管平台和开放源代码的一部分，与它们的Watson AI生态系统很好地联系在一起
    • StackPath EdgeEngine -使用您选择的语言编写功能即服务，并将其部署到全球数据中心网络. 所有网络，包括智能路由和负载平衡，都由StackPath通过专用骨干网进行管理.
    • Vercel -Vercel使人们可以使用他们选择的语言编写功能即服务，并作为monorepo的一部分进行部署.
    • Azure Static Web Apps -全栈静态应用托管，包括无服务器功能，身份验证，CDN等

    GraphQL¶

    • FaunaDB -无服务器GraphQL数据库. 免费套餐，没有时间限制. 轻松包含在Netlify应用程序中.

    Community¶

    Comments¶

    • Staticman -Staticman是一个Node.js应用程序，它接收用户生成的内容并将其作为数据文件上传到GitHub存储库. 实际上，这允许您将动态内容（例如，博客文章评论）作为完全静态的网站的一部分，只要您的网站在每次推送到GitHub时自动部署，如GitHub Pages，Netlify和其他网站所示.
    • Disqus -带有嵌套回复，多种登录方法和电子邮件通知的易于嵌入的注释.
    • Facebook Comments -Facebook为您的网站嵌入的评论.
    • IntenseDebate Comments -具有嵌套回复，多种登录方法和电子邮件通知的可嵌入评论.
    • LiveFyre -实时评论，经过SEO优化，具有社交功能，并且在台式机和移动设备上均美观.
    • Redditjs Embed Widget -在您的网站上嵌入Reddit评论. 如果尚未发布，它将显示一个链接以鼓励用户提交.
    • Muut.com -可嵌入的评论，论坛和私人消息. 功能很多，但网站上的空间却很少.
    • Gitment -基于GitHub Issues的评论系统，无需任何服务器端实现即可在前端使用.

    Forms¶

    Really Simple Forms¶

    • Formspree -从您的静态网站上的表单接收电子邮件.
    • elFormo -简单的表格处理和通过电子邮件的回复检索.
    • Flipmail -简单的表格处理和通过电子邮件的回复检索.
    • MailThis -通过电子邮件以可选附件的形式提交简单表格.
    • Simple Form -带有可选文件附件，电子邮件通知和在线提交查看的简单表单.
    • Brisk Forms -免费表单提交服务通过电子邮件发送您的回复，同时保持电子邮件地址的私密性，并且是开源的.
    • 99 Inbound -通过电子邮件/闲置通知和第三方应用程序集成（例如MailChimp）形成端点服务
    • Getform -通过电子邮件和集成为设计人员和开发人员提供表单后端平台.
    • Form.taxi -后端通过电子邮件通知，文件上传和符合GDPR的数据处理功能，轻松，可靠地处理表单提交.

    Normal Forms¶

    • Formcarry -适用于表单的无障碍HTML表单端点，功能强大的仪表板，可靠的垃圾邮件阻止功能，附件上传功能和Zapier集成.
    • Formcake -为开发人员构建的表单后端：Zapier集成，简单的端点API，无限制的表单.
    • Google Forms -将结果保存到Google表格中，可以在提交时通过电子邮件发送给您.
    • FormKeep -付费-在漂亮的Web界面中查看表单提交. 它具有垃圾邮件过滤功能，并且与Gmail，Trello和Basecamp等Webhooks集成.
    • 123 Contact Form -连接到其他在线服务，例如MailChimp，Salesforce和Google Drive. 它还与付款处理程序集成，并包括安全性和分析功能.
    • FormAssembly -允许您构建任何形式的表单，其中可以包括复杂的分支逻辑和多个页面.
    • FormSite -带有付款和表单管理的表单生成器.
    • FormStack -具有A / B测试，部分提交，分析和集成的表单.
    • Sheetsu -发布并获取您的数据到Google Spreadsheet.
    • Typeform -可以嵌入的令人敬畏的表格.
    • Wufoo -免费或付费-您可以使用表单设计器构建的表单，包括通知，报告和付款.
    • Zoho -带有文件上传和验证码的表格.
    • GitHub Issues -这是开发人员获得评论/问题的一种有趣方式. 看 github.com/sindresorhus/ama 举个例子.
    • Utterences -一个基于GitHub问题的轻量级注释小部件.
    • FormBackend -创建表单后端，并将您的HTML表单提交到我们的后端. 在线查看条目并连接到其他服务. 每次提交新条目时都会收到一封电子邮件.
    • Pageclip -用于HTML表单的灵活服务器/后端. 在实时Web界面中查看数据，或使用API​​获取CSV和JSON输出.
    • Formester -具有集成的表格和电子邮件营销（线索收集，电子邮件活动和新闻通讯）.
    • StaticKit -静态网站的现代形式，具有对React的原生支持.
    • SheetDB -将Google Spreadsheet转换为JSON API.

    Provided by the Host¶

    • Netlify -Netlify带有内置的表单处理功能.

    Live Chat¶

    • jivochat -JivoSite是专门针对网站的专业实时聊天，旨在增加您的在线销售.
    • LiveChat -在您的网站上进行实时聊天.
    • Olark -在您的网站上进行实时聊天. 您还可以查看谁在您的网站上以及他们在做什么.
    • SnapEngage -集成和自定义样式的实时聊天.
    • tawk.co -可让您监视网站上的访问者并与之聊天.
    • WebsiteAlive -为您的网站和社交网络进行实时聊天.
    • Zopim -免费试用实时聊天.

    Newsletters¶

    • MailChimp -免费电子邮件营销. 您可以付费添加更多功能.
    • Constant Contact -通过广告系列，自动回复和分析进行电子邮件营销.
    • AWeber -通过广告系列，自动回复和分析进行电子邮件营销.
    • Campaign Monitor -通过广告系列，自动回复和分析进行电子邮件营销.
    • MailerLite -免费电子邮件营销. 您可以为更多的订户付费.

    Social Media¶

    • Pinterest -固定按钮.
    • Twitter -嵌入式推文.
    • Facebook -Facebook嵌入式插件.
    • ShareThis -共享多个社交网络的按钮.
    • Kontaktify -联系人小部件，为访问者提供了一种轻松的联系方式.

    Surveys¶

    • Google Forms -您可以将Google表格用于调查或网站上的表格.
    • SurveyMonkey -易于使用和免费调查.
    • Typeform -真的很漂亮.
    • Qualaroo -在屏幕右下角出现的网站上嵌入调查.
    • Insight Stash -快速，简单的调查表.

    E-Commerce¶

    • Ecwid -嵌入式购物车.
    • FoxyCart -添加带有基本html代码的购物车.
    • Snipcart -包括几行代码，以组成一个完整的在线商店.
    • Gumroad -一站式销售您的作品的解决方案.
    • Payhip -销售数字下载和会员资格的可嵌入方式
    • Moltin -将电子商务功能添加到任何东西.
    • Trolley -将弹出式购物车添加到任何网站-专为静态和JAMstack网站设计.
    • Commerce Layer -将企业电子商务添加到您的JAMstack.

    Payments¶

    • MoneyButton -使用比特币的网站付款和捐赠（Satoshi的愿景）.
    • ShapeShift Shifty Button -接受使用各种加密货币的付款.

    Search¶

    Self-hosted:

    • lunr.js -在浏览器中进行简单的全文本搜索.
    • itemsjs -javascript中的全文，多面，几乎无依赖的搜索引擎
    • minisearch -用于浏览器和Node的小而强大的JavaScript全文搜索引擎
    • flexsearch -用于浏览器和Node.js的下一代全文搜索库
    • fuse.js -功能强大，轻量级的模糊搜索库，零依赖
    • static-search -一个Go程序来生成HTML文件的JSON索引，以及一个带有可选UI的JavaScript组件来搜索该索引
    • elasticlunr -使用JavaScript开发的轻量级全文本搜索引擎，用于基于Lunr.js的浏览器搜索和脱机搜索
    • tinysearch -用于使用Rust和Wasm构建的静态网站的微型全文搜索引擎
    • js-search -客户端搜索JavaScript和JSON对象，与ES5兼容，不需要jQuery或任何其他第三方库
    • search-index -用于浏览器和Node.js的持久性，网络弹性，全文搜索库
    • fuzzysearch -使用JavaScript进行微小而快速的模糊搜索
    • fuzzy -浏览器和节点的模糊搜索/过滤器
    • fullproof -Javascript库，可在浏览器中提供高质量的全文本搜索
    • Jets.js -原生CSS搜索引擎
