    Awesome irc

    Awesome IRC ¶

    >精选的超赞清单 IRC 资源.

    与Internet中继聊天（IRC）协议相关的工具，软件和其他资源的列表.

    IRC（Internet中继聊天）是一种开放源代码协议，可用于通过通道进行基于多用户文本的通信.

    Clients¶

    您使用这些连接到IRC.

    • Komanda -专为编写代码的人（测试版）而设计. （source）Linux
    • Textual -高度可定制，ZNC集成，iCloud同步（4.99美元）. （source）macOS
    • LimeChat -多个服务器的一个窗口，键盘快捷键，快速且稳定. （source）macOS`` iOS
    • HexChat -基于XChat，易于使用，拼写检查和多种语言. （source）Windows macOS Linux
    • Kiwi IRC -强大的现代IRC信使网络. （source, demo）Web
    • CIRC -使用chrome.sockets API直接连接到IRC服务器，而无需代理. （source）Chrome
    • Quassel -分布式（客户端可以连接到永久保持在线状态的中央核心，也可以从中分离.）source）Linux macOS Windows
    • Yaaic -多服务器/通道支持，SASL支持，平滑的通道滚动/滑动. （source）Android
    • Koko -简约的设计，类似VIM的快捷键并基于电子设备. WindowsmacOS
    • relay.js -专注于减少IRC的威胁性并使其更易于使用. （demo）Web
    • Circe -在Emacs中使用时，默认设置为理智. Emacs
    • Smuxi -基于GNOME / GTK +的用户友好型. （source）Linux Windows MacOS
    • KvIRC -基于Qt GUI工具包的免费，可移植的工具. （source）Linux macOS Windows
    • Konversation -建立在KDE平台上的用户友好客户端. （source）Linux
    • sic -S imple I RC lient-少于250行C语言的终端客户端.
    • Colloquy -启用AppleScript的IRC客户端. macOS
    • irssi -终端客户端，对模块作者GPLv2具有多协议友好性. LinuxmacOSCygwinBSD`
    • Revolution IRC -功能齐全，积极维护的Android IRC客户端. （source）Android
    • AdiIRC -从来没有客户为IRC体验的各个方面提供如此精细的设置. （features, screenshots）WindowsWINE
    • IRC for Android™ 适用于高级用户的Android / Chrome OS客户端，具有ZNC内置，通知逻辑，可靠的DCC，用于硬件键盘的键绑定等.

    更多？ 找到包含保镖的客户端 below.

    Bouncers¶

    可用于断开和重新连接而不会丢失聊天会话.

    Hosted¶

    • IRCCloud -与团队，朋友和社区进行群聊. 保持联系，随时随地聊天，并且永远不会错过任何消息（+客户）（每月£0-£3.50）.
    • iOS App -官方. Objective-C
    • Android App -官方. Java
    • Nimbus -独立客户端. macOS``Objective-C
    • Grove -托管IRC，以及更多（+客户端）（每月10-125美元）.

    Self-hosted¶

    • Convos -始终在线的Web IRC客户端. （source, demo）PerlJavaScriptWeb
    • ZNC - 最受欢迎. 许多不同的插件. （source）C ++
    • BIP IRC Proxy -始终在线，轻量级和安全的带积压的开源IRC代理. （source）C
    • TheLounge -响应式，自托管和支持多个用户. （source, demo）JavaScriptNode.jsWeb
    • WeeChat -快速，轻便和可扩展的聊天客户端. （source）Linux macOS Windows
    • soju -用户友好的IRC保镖. 去吧

    Daemons¶

    用于运行自己的IRC服务器或网络.

    • ircd.js -服务器将允许客户端连接，加入频道，更改主题； 基本的东西.
    • InspIRCd -模块化，稳定，从头开始编写. （source)
    • miniircd -非常简单和有限.
    • ngIRCd -轻巧便携，适用于小型或专用网络. （source)
    • Hulk -用于私人用途或业余爱好. 哈斯克尔
    • Oragono -便携式的现代实验服务器，并根据规格进行设计. （source)
    • charybdis -可扩展的IRCv3服务器.
    • RobustIRC -IRC服务器没有网络拆分.

    Services¶

    用于向您的网络提供用户帐户和机器人，例如NickServ / ChanServ.

    • Atheme -专为具有高可扩展性要求的大型网络而设计. （source)
    • anope -专为灵活性和易用性而设计. （source)

    Bots¶

    *为人类提供服务（例如集成或信息）的IRC用户.

    • Sopel -大量现成的功能，教程，已全面记录. （source）Python
    • IRCcat -简化了来自Shell脚本的消息传递. Java
    • Limnoria -健壮，用户友好，对开发人员友好. Python
    • Twitch Plays -从聊天中获取输入，然后按相应的键. Python
    • Skybot -主要目标是简单和强大. Python
    • talkbackbot -以引人注目的女性引语来响应配置的触发短语. （source）Python
    • lazybot -用户友好且功能强大. Clojure
    • IRC-BF -Brainfuck
    • geordi -编译并运行C ++代码段. C ++
    • CloudBot -简单，快速，可扩展. Python
    • yossarian-bot -基于Cinch的大型默认插件集. 红宝石
    • helga -支持多种协议的可插拔聊天机器人. Python
    • EveIRC -可扩展的聊天/频道/服务器管理员服务提供机器人. 使用 Cinch Framework. Ruby
    • BitBot -模块化的，事件驱动的机器人，具有REST API，单独的用户设置等. bitbot.dev Python
