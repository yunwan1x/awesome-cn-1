    Elixir

    Awesome Elixir ¶

    精选的令人惊叹的 Elixir 库、资源和闪亮事物的列表，灵感来自 awesome-php.

    如果您认为应该添加一个包，请在相应的问题上添加一个 (:+1:) 或创建一个新的.

    有 other sites with curated lists of elixir packages 你可以看看.

    Actors¶

    用于与演员等合作的库和工具.

    • bpe - Erlang 中的业务流程引擎. (Doc).
    • dflow - 流水线式流处理引擎.
    • exactor - 帮助在 Elixir 中更轻松地实现 actor.
    • exos - 一个端口包装器，它将转换和调用转发到一个链接的端口.
    • flowex - 使用 Elixir GenStage 的基于铁路流的编程.
    • mon_handler - 监视给定 GenEvent 处理程序的最小 GenServer.
    • pool_ring - 基于哈希环创建一个池.
    • poolboy - 一个笨拙的 Erlang 工人池工厂.
    • pooler - OTP 进程池应用程序.
    • sbroker - 基于逗留时间的活动队列管理库.
    • workex - EVM 过程中的背压和流量控制.

    Algorithms and Data structures¶

    算法和数据结构的库和实现.

    • array - Erlang 数组的 Elixir 包装库.
    • aruspex - Aruspex 是一个可配置的约束求解器，完全用 Elixir 编写.
    • bimap - Pure Elixir 实现 bidirectional maps 和多图.
    • bitmap - Pure Elixir 实现 bitmaps.
    • blocking_queue - BlockingQueue 是一个实现为 GenServer 的简单队列. 它在创建时有一个固定的最大长度.
    • bloomex - 可扩展布隆过滤器的纯 Elixir 实现.
    • clope - Elixir 的实现 CLOPE：一种快速有效的交易数据聚类算法.
    • combination - Elixir 库，用于从 Enumerable 集合生成组合和排列.
    • count_buffer - 缓冲大量计数器并定期刷新.
    • cuckoo - 一个纯粹的 Elixir 实现 Cuckoo Filters.
    • cuid - 用 Elixir 编写的针对水平缩放和顺序查找性能进行了优化的抗碰撞 ID.
    • data_morph - 从数据创建 Elixir 结构.
    • dataframe - 提供类似于 Python 的 Pandas 或 R 的 data.frame() 功能的包.
    • datastructures - 一组用于处理数据结构的协议、实现和包装器.
    • def_memo - 使用 genserver 后备存储的 elixir 记忆宏 (defmemo).
    • dlist - Elixir 中的所有实现.
    • eastar - 纯 Elixir 中的 A* 图形寻路.
    • ecto_materialized_path - ecto 模型的树结构、层次结构和祖先.
    • ecto_state_machine - 在 Elixir 上实现并为 Ecto 采用的有限状态机模式.
    • elistrix - 一个延迟/容错库，有助于将您的应用程序与缓慢或失败服务的不确定世界隔离开来.
    • emel - 一个用 elixir 编写的简单实用的机器学习库.
    • erlang-algorithms - 流行数据结构和算法的实现.
    • exconstructor - 一个 Elixir 库，用于生成轻松处理外部数据的结构构造函数.
    • exfsm - 定义静态 FSM 的简单 elixir 库.
    • exkad - 一种 kademlia 在 Elixir 中实现.
    • exmatrix - ExMatrix 是一个用于处理矩阵的小型库，最初开发用于并行测试矩阵乘法.
    • exor_filter - xor_filters 的 Nif. “比布隆和布谷鸟过滤器更快、更小”.
    • ezcryptex - Cryptex 顶部的薄层.
    • flow - GenStage 之上的计算并行流程.
    • fnv - Fowler–Noll–Vo 哈希函数的纯 Elixir 实现.
    • fsm - 作为功能数据结构的有限状态机.
    • fuse - 此应用程序为 Erlang 实现了所谓的断路器.
    • gen_fsm - 通用有限状态机 - 围绕 OTP 的 gen_fsm 的 Elixir 包装器.
    • graphmath - 用于执行 2D 和 3D 数学的 Elixir 库.
    • hash_ring_ex - Elixir 的一致哈希环实现.
    • hypex - HyperLogLog 的快速 Elixir 实现.
    • indifferent - 使用自定义键转换对 Elixir 映射/列表/元组的访问无关紧要.
    • isaac - Isaac 是 ISAAC 的灵丹妙药模块：快速加密随机数生成器.
    • jumper - 在 Elixir 中跳转一致的哈希实现（没有 NIF）.
    • key2value - Erlang 2 路集关联映射.
    • lfsr - 二进制伽罗瓦线性反馈移位寄存器的 Elixir 实现.
    • loom - 支持 δ-CRDT 的 CRDT 库.
    • luhn - Elixir 中的 Luhn 算法.
    • lz4 - 用于快速数据压缩的 Erlang LZ4 绑定.
    • machinery - 一般用于结构的状态机库，它开箱即用地与 Phoenix 集成.
    • mason - 强制映射到结构中. 这很有用，例如，当您连接 REST API 并希望从响应创建结构时.
    • matrex - 使用 CBLAS 的具有 C 实现的 Elixir/Erlang 的超快矩阵库.
    • merkle_tree - Elixir 中的 Merkle 哈希树实现.
    • minmaxlist - Elixir 库扩展了 Enum.min_by/2、Enum.max_by/2 和 Enum.min_max_by/2 以返回结果列表而不是一个.
    • mmath - 用于对二进制文件中的数字“数组”执行数学运算的库.
    • monad - Haskell 启发了 Elixir 时尚语法中的 monad.
    • monadex - 使用 monad 升级您的 Elixir 管道.
    • murmur - 非加密哈希 Murmur3 的纯 Elixir 实现.
    • nary_tree - 通用 n 元树数据结构的 Elixir 实现.
    • natural_sort - 字符串列表的 Elixir 自然排序实现.
    • navigation_tree - 带有帮助程序的导航树表示，可从中生成 HTML.
    • parallel_stream - Elixir 的并行流实现.
    • paratize - Elixir 库提供了一些方便的并行处理（执行）工具，支持配置工作线程数和超时.
    • parex - Parallel Execute (Parex) 是一个 Elixir 模块，用于并行执行多个（慢）进程.
    • qex - 包装 :queue，具有改进的 API 和 Inspect、Collectable 和 Enumerable 协议实现.
    • ratio - 添加有理数并允许它们用于常见的算术运算. 还支持浮点数和有理数之间的转换.
    • red_black_tree - Elixir 中的红黑树实现.
    • remodel - 用于转换地图结构的 Elixir 演示程序包.
    • rendezvous - 在 Elixir 中实现 Rendezvous 或最高随机权重 (HRW) 哈希算法.
    • rock - ROCK 的 Elixir 实现：分类属性的稳健聚类算法.
    • sfmt - Erlang 的面向 SIMD 的 Fast Mersenne Twister (SFMT).
    • simhash - 使用 Siphash 和 N-gram 实现 Simhash.
    • sleeplocks - 用于 Elixir/Erlang 的 BEAM 友好自旋锁.
    • sorted_set - Elixir 的排序集.
    • spacesaving - 使用“节省空间”算法的流计数不同元素估计.
    • structurez - Elixir 中数据结构的游乐场.
    • supermemo - 一个 Elixir 实现 Supermemo 2 algorithm.
    • tfidf - 词频-逆文档频率的 Elixir 实现.
    • the_fuzz - 模糊字符串匹配算法实现.
    • tinymt - 用于 Erlang 的 Tiny Mersenne Twister (TinyMT).
    • trie - Erlang Trie 实现.
    • witchcraft - Elixir 的常见代数结构和函数.
    • zipper_tree - 带有用于 Elixir 的拉链的可变参数树.
