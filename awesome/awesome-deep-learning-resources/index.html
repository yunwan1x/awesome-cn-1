    Awesome Deep Learning Resources ¶

    这是我最喜欢的深度学习资源的粗略清单. 对于学习如何进行深度学习，用于重新访问主题或供参考，它对我很有用. 一世 （Guillaume Chevalier）已构建了此列表，并仔细浏览了此处列出的所有内容.

    Trends¶

    这是空前 Google Trends，从2004年至今，即2017年9月：

    您可能还想看看Andrej Karpathy new post 关于机器学习研究的趋势.

    我相信深度学习是使计算机更像人类思考的关键，并且具有很大的潜力. 某些困难的自动化任务可以轻松解决，而传统算法则无法实现.

    摩尔定律关于计算机科学硬件中的指数级进展速度，现在对GPU的影响比对CPU的影响更大，这是因为原子晶体管的微小物理限制. 我们正在转向并行架构 [read more ]. 深度学习通过使用GPU在后台开发并行架构. 最重要的是，深度学习算法可能会使用量子计算，并在将来应用于机器脑接口.

    我发现，智力和认知的关键是一个非常有趣的主题，并且尚未得到很好的理解. 这些技术是有前途的.

    Online Classes¶

    • DL&RNN Course -我在深度学习和递归神经网络上开设了这个内容丰富的课程.
    • Machine Learning by Andrew Ng on Coursera -具有知名度的入门级在线课程 certificate . 授课人：斯坦福大学副教授吴安德（Andrew Ng）； 百度首席科学家； Coursera董事长兼联合创始人.
    • Deep Learning Specialization by Andrew Ng on Coursera -Andrew Ng的新系列5深度学习课程，现在使用Python而不是Matlab / Octave，这导致了 specialization certificate.
    • Deep Learning by Google -涵盖高级深度学习概念的高级中级到高级课程，我发现一旦掌握了基础知识，它就会有助于创新.
    • Machine Learning for Trading by Georgia Tech -有趣的课程，用于学习适用于交易的机器学习基础知识以及一些AI和金融概念. 我特别喜欢有关Q学习的部分.
    • Neural networks class by Hugo Larochelle, Université de Sherbrooke -有趣的有关神经网络的课程，雨果·拉奥歇尔（Hugo Larochelle）免费提供在线课程，但我已经观看了其中一些视频.
    • GLO-4030/7030 Apprentissage par réseaux de neurones profonds -这是拉瓦尔大学教授PhilippeGiguère开设的课程. 我特别发现了它罕见的多头注意力机制的可视化效果，可以在 slide 28 of week 13's class.
    • Deep Learning & Recurrent Neural Networks (DL&RNN) -关于“深度学习和递归神经网络”的主题中最丰富，最加速的课程（最后滚动）.

    Books¶

    • Clean Code -回到您愚弄的基础知识！ 了解如何为您的职业做好清洁代码. 这是迄今为止我读过的最好的书，即使该列表与深度学习有关.
    • Clean Coder -了解如何成为一名专业的编码员，以及如何与您的经理互动. 这对任何编码职业都很重要.
    • How to Create a Mind -通勤时听音频版本很不错. 本书旨在激发思想的逆向工程，并思考如何对AI进行编码.
    • Neural Networks and Deep Learning -本书涵盖了神经网络和深度学习背后的许多核心概念.
    • Deep Learning - An MIT Press book -在本书的中途，它包含了令人满意的数学内容，内容涉及如何思考实际的深度学习.
    • Some other books I have read -此处列出的一些书籍与深度学习的关联性较弱，但仍与该列表具有某种关联性.

    Posts and Articles¶

    • Predictions made by Ray Kurzweil -雷·库兹韦尔（Ray Kurzweil）提出的中长期的未来主义预测列表.
    • The Unreasonable Effectiveness of Recurrent Neural Networks -必须阅读Andrej Karpathy的帖子-这是促使我学习RNN的动力，它演示了以NLP的最基本形式可以实现的目标.
    • Neural Networks, Manifolds, and Topology -重新了解神经元如何映射信息.
    • Understanding LSTM Networks -说明LSTM单元的内部工作原理，此外，它还具有有趣的结论.
    • Attention and Augmented Recurrent Neural Networks -对于视觉动画很有趣，它是注意力机制的一个很好的介绍.
    • Recommending music on Spotify with deep learning -非常适合对音频进行聚类-由Spotify的实习生发帖.
    • Announcing SyntaxNet: The World’s Most Accurate Parser Goes Open Source -Parsey McParseface的诞生，一个神经语法树解析器.
    • Improving Inception and Image Classification in TensorFlow -非常有趣的CNN架构（例如，在减少参数数量方面，初始样式的卷积层很有希望且有效）.
    • WaveNet: A Generative Model for Raw Audio -逼真的说话器：完美的声音产生.
    • François Chollet's Twitter - Author of Keras - has interesting Twitter posts and innovative ideas.
    • Neuralink and the Brain’s Magical Future -关于大脑和脑机接口的未来的发人深省的文章.
    • Migrating to Git LFS for Developing Deep Learning Applications with Large Files -轻松管理您私人Git项目中的大文件.
    • The future of deep learning -弗朗索瓦·乔列（FrançoisChollet）关于深度学习的未来的思想.
    • Discover structure behind data with decision trees -增长决策树并将其可视化，推断数据背后的隐藏逻辑.
    • Hyperopt tutorial for Optimizing Neural Networks’ Hyperparameters -学习自动削减超参数空间，而不是手工削减.
    • Estimating an Optimal Learning Rate For a Deep Neural Network -巧妙的技巧，可以在任何一次完整的训练之前估算出最佳的学习速度.
    • The Annotated Transformer -有助于理解“注意就是您所需要的全部”（AIAYN）论文.
    • The Illustrated Transformer -也有助于理解“注意就是您所需要的”（AIAYN）论文.
    • Improving Language Understanding with Unsupervised Learning -从大型语料库的无监督预训练中跨许多NLP任务进行SOTA.
    • NLP's ImageNet moment has arrived -全部欢呼NLP的ImageNet时刻.
    • The Illustrated BERT, ELMo, and co. (How NLP Cracked Transfer Learning) -了解用于NLP ImageNet时刻的不同方法.
    • Uncle Bob's Principles Of OOD -不仅需要SOLID原则来进行简洁的代码，而且众所周知的REP，CCP，CRP，ADP，SDP和SAP原则对于开发必须捆绑在不同单独软件包中的大型软件也非常重要.
    • Why do 87% of data science projects never make it into production? -数据不可忽视，团队与数据科学家之间的沟通对于正确集成解决方案非常重要.
    • The real reason most ML projects fail -专注于明确的业务目标，除非拥有真正的干净代码，否则避免使用算法的枢纽，并且能够知道何时编写的代码“足够好”.

    Practical Resources¶

    Librairies and Implementations¶

    • Neuraxle, a framwework for machine learning pipelines -用于构建和部署机器学习项目的最佳框架，并且也与大多数框架兼容（例如：Scikit-Learn，TensorFlow，PyTorch，Keras等）.

    • Deep learning for lazybones -TensorFlow中的转移学习教程，用于从预先训练的CNN的高级嵌入中获得视觉，AlexNet 2012.

    • Deep stacked residual bidirectional LSTMs for HAR -对先前项目的改进.

    • Hyperopt for a Keras CNN on CIFAR-100 -在CIFAR-100数据集上自动（元）优化神经网络（及其体系结构）.
    • ML / DL repositories I starred -GitHub上充满了不错的代码示例和项目.
    • Smoothly Blend Image Patches -平滑的补丁合并 semantic segmentation with a U-Net.
    • Self Governing Neural Networks (SGNN): the Projection Layer -这样一来，您就可以在深度学习模型中使用单词，而无需训练或加载嵌入内容.
    • Neuraxle -Neuraxle是一个机器学习（ML）库，用于构建整洁的管道，提供正确的抽象以简化ML应用程序的研究，开发和部署.
    • Clean Machine Learning, a Coding Kata -通过练习来学习用于进行机器学习的好方法的良好设计模式.

    • Neural Networks and Deep Learning, ch.4 -视觉证明神经网络可以计算任何功能.

    • Gradient descent algorithms in a saddle point -可视化不同的优化器如何与鞍点交互.
    • Gradient descent algorithms in an almost flat landscape -可视化不同的优化器如何与几乎平坦的景观交互.
    • Gradient Descent -好的，我已经在上面列出了Andrew NG的Coursera类，但是该视频特别适合作为介绍并定义了梯度下降算法.
    • Gradient Descent: Intuition -上一个视频的内容如下：现在添加直觉.
    • Gradient Descent in Practice 2: Learning Rate -如何调整神经网络的学习率.
    • The Problem of Overfitting -关于过度拟合以及如何解决该问题的很好的解释.
    • Diagnosing Bias vs Variance -了解神经网络预测中的偏差和方差以及如何解决这些问题.
    • Self-Normalizing Neural Networks -令人难以置信的SELU激活功能的外观.
    • Learning to learn by gradient descent by gradient descent -RNN作为优化器：引入了L2L优化器（一种元神经网络）.

    Complex Numbers & Digital Signal Processing¶

    好的，信号处理可能与深度学习没有直接关系，但是研究在基于信号的神经体系结构开发中拥有更多的直觉是有趣的.

    • Window Functions -列出了一些已知窗口功能的Wikipedia页面-请注意， Hann-Poisson window 对于贪婪的爬坡算法（例如，梯度下降）而言，它特别有趣.
    • MathBox, Tools for Thought Graphical Algebra and Fourier Analysis -傅立叶分析的新面貌.
    • How to Fold a Julia Fractal -处理复数和波动方程的动画.
    • Animate Your Way to Glory, Math and Physics in Motion -物理引擎中的收敛方法，并应用于交互设计.
    • Animate Your Way to Glory - Part II, Math and Physics in Motion -用于旋转和带有四元数的旋转插值的精美动画，四元数是用于处理3D旋转的数学对象.
    • Filtering signal, plotting the STFT and the Laplace transform -有关信号处理的简单Python演示.

    Papers¶

    Recurrent Neural Networks¶

    • Deep Learning in Neural Networks: An Overview -You_Again的深度学习摘要/概述，主要是有关RNN的.
    • Bidirectional Recurrent Neural Networks -在时间轴上进行双向扫描的RNN进行更好的分类.
    • Learning Phrase Representations using RNN Encoder-Decoder for Statistical Machine Translation -将两个网络合二为一的seq2seq（序列到序列）编码器-解码器体系结构. RNN编码器-具有1000个隐藏单元的解码器. Adadelta优化器.
    • Sequence to Sequence Learning with Neural Networks -在WMT'14英语到法语数据集上，具有4个堆叠的LSTM单元，其隐藏大小为1000，带有反向输入语句并带有波束搜索.
    • Exploring the Limits of Language Modeling -在字符级CNN上使用单词级LSTM的不错的递归模型，使用了过多的GPU功能.
    • Neural Machine Translation and Sequence-to-sequence Models: A Tutorial -关于NMT的主题，有趣的概述是，我主要阅读有关RNN的第8部分，并作为重温.
    • Exploring the Depths of Recurrent Neural Networks with Stochastic Residual Learning -基本上，在呈现的情绪分析案例中，残余连接可以比堆叠的RNN更好.
    • Pixel Recurrent Neural Networks -非常适合像photoshop一样的“内容感知填充”来填充图像中缺少的色块.
    • Adaptive Computation Time for Recurrent Neural Networks -让RNN决定计算时间. 我很想知道它与神经图灵机的结合效果如何. 可以找到关于该主题的有趣的交互可视化效果 here.

    Convolutional Neural Networks¶

    • What is the Best Multi-Stage Architecture for Object Recognition? -非常适合使用“局部对比归一化”.
    • ImageNet Classification with Deep Convolutional Neural Networks -AlexNet，2012年ILSVRC，突破了ReLU激活功能.
    • Visualizing and Understanding Convolutional Networks -用于“ deconvnet层”.
    • Fast and Accurate Deep Network Learning by Exponential Linear Units -用于CIFAR视觉任务的ELU激活功能.
    • Very Deep Convolutional Networks for Large-Scale Image Recognition -有趣的想法是，在仅使用几个参数的情况下合并多个3x3 conv + ReLU即可合并成更大的过滤器大小. 对于“ ConvNet配置”，还有一个不错的表.
    • Going Deeper with Convolutions -GoogLeNet：“起始”层/模块的出现，其想法是将conv层并行化为许多不同大小的微型conv，并使用“相同”的填充，并在深度上进行连接.
    • Highway Networks -公路网：残余连接.
    • Batch Normalization: Accelerating Deep Network Training by Reducing Internal Covariate Shift -批量归一化（BN）：通过还对整个批次进行求和来归一化图层的输出，然后执行线性重新缩放和某个可训练量的平移.
    • U-Net: Convolutional Networks for Biomedical Image Segmentation -U-Net是编码器/解码器CNN，也具有跳过连接，适合在每个像素级别进行图像分割.
    • Deep Residual Learning for Image Recognition -具有批处理归一化层的非常深的残差层-又名“如何在层数过多的情况下过度拟合任何视觉数据集，并在给定足够数据的情况下使任何视觉模型都能正常工作”.
    • Inception-v4, Inception-ResNet and the Impact of Residual Connections on Learning -用于通过残余连接改进GoogLeNet.
    • WaveNet: a Generative Model for Raw Audio 史诗原始语音/音乐生成，具有基于散因果卷积的新架构，可捕获更多音频长度.
    • Learning a Probabilistic Latent Space of Object Shapes via 3D Generative-Adversarial Modeling -3D-GAN用于生成3D模型并从嵌入中获得有趣的3D家具算术（想像具有3D家具表示形式的word2vec单词算术）.
    • Accurate, Large Minibatch SGD: Training ImageNet in 1 Hour -CNN的快速分布式培训.
    • Densely Connected Convolutional Networks -2017年CVPR最佳论文奖，该新神经网络体系结构被称为DenseNet，对CIFAR-10，CIFAR-100和SVHN数据集的最新性能产生了改进.
    • The One Hundred Layers Tiramisu: Fully Convolutional DenseNets for Semantic Segmentation -融合了U-Net和DenseNet的思想，这种新的神经网络特别适用于图像分割中的大型数据集.
    • Prototypical Networks for Few-shot Learning -在损耗中使用距离度量，从一些示例中确定对象属于哪个类.

    Attention Mechanisms¶

    • Neural Machine Translation by Jointly Learning to Align and Translate -LSTM的注意机制！ 通常，数字和公式及其解释对我很有用. 我在那张纸上发表了演讲 here.
    • Neural Turing Machines -对于让神经网络学习算法具有良好的长期依赖性方面的出色表现而出类拔萃. 序列召回问题.
    • Show, Attend and Tell: Neural Image Caption Generation with Visual Attention -LSTM在CNN特征图上的关注机制确实令人惊讶.
    • Teaching Machines to Read and Comprehend -关于文本问题解答的一项非常有趣且富有创造力的工作，这是一项突破，与之相关.
    • Effective Approaches to Attention-based Neural Machine Translation -探索注意力机制的不同方法.
    • Matching Networks for One Shot Learning -通过使用注意力机制和查询将图像与其他图像进行分类的有趣的低数据单次学习方法.
    • Google’s Neural Machine Translation System: Bridging the Gap between Human and Machine Translation -2016年：对于编码器/解码器具有注意机制的堆叠残留LSTM，最适合NMT（神经机器翻译）.
    • Hybrid computing using a neural network with dynamic external memory -改进了基于NTM的可区分内存：现在是可区分神经计算机（DNC）.
    • Massive Exploration of Neural Machine Translation Architectures -在框架化的seq2seq问题表述中得出关于进行NMT的有效方法的界限的直觉. -[通过在梅尔谱图上调节WaveNet来合成天然TTS 预测]（https://arxiv.org/pdf/1712.05884.pdf）-A WaveNet 用作声码器的条件可以是Tacotron 2 LSTM神经网络生成的梅尔频谱图，但要注意从文本生成纯净的音频.
    • Attention Is All You Need （AIAYN）-引入具有位置编码的多头自注意神经网络以在没有任何RNN或CNN的情况下执行句子级NLP-这篇文章是必读 this explanation 和 this visualization 本文）.

    Other¶

    • ProjectionNet: Learning Efficient On-Device Deep Networks Using Neural Projections -在深度神经网络中，用单词投影替换单词嵌入，不需要预先提取字典，也不需要存储嵌入矩阵.
    • Self-Governing Neural Networks for On-Device Short Text Classification -本文是以上ProjectionNet的续集. SGNN在ProjectionNet上进行了详细说明，并且对优化进行了更深入的详细介绍（另请参见 attempt to reproduce the paper in code 看着 the talks' recording).
    • Matching Networks for One Shot Learning - Classify a new example from a list of other examples (without definitive categories) and with low-data per classification task, but lots of data for lots of similar classification tasks - it seems better than siamese networks. To sum up: with Matching Networks, you can optimize directly for a cosine similarity between examples (like a self-attention product would match) which is passed to the softmax directly. I guess that Matching Networks could probably be used as with negative-sampling softmax training in word2vec's CBOW or Skip-gram without having to do any context embedding lookups.
