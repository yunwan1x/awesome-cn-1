    Awesome Substrate ¶

    >真棒列表是社区策划的真棒列表.

    Substrate 是用于构建可升级，模块化和高效的框架 区块链 Substrate是一个开源库 Rust 编码 由维护 Parity Technologies.

    Official Resources¶

    • Developer Hub -开发人员门户.
    • Tutorials -指导练习，帮助您入门.
    • Knowledge Base -有关的深入文档 基材开发.
    • Recipes -解决常见问题的工作代码示例.
    • Reference Docs -Rustdoc API文档.
    • Seminar -协作学习 Crowdcast.
    • Technical Chat - 问问题 ＆ 与其他Substrate开发人员聊天.
    • Builders Program -有远见的白手套支持 团队.
    • Web3 Foundation Grants -为生态系统发展提供资金.
    • Users -与Substrate建立团队.
    • GitHub -基板源代码.
    • Developer Hub GitHub -开发人员中心存储库.
    • Polkadot GitHub -多种底物monorepo 奇偶校验赞助的链.
    • subport -支持资料库.
    • Parity Blog -了解底物生态系统中的更新.
    • Parity Events - 接下来的活动.
    • Twitter -跟着我们保持最新.
    • Polkassembly -在草间和波卡多上进行讨论和投票的地方 治理.

    Community Resources¶

    • DotLeap -社区内容和新闻通讯.
    • Stack Overflow -带标签的问题 底物
    • Reddit -官方subreddit.
    • Polkadot Stack -另一个 由我们的朋友维护的很棒的清单 Web3 Foundation.
    • Subsocial -Polkadot / Kusama的去中心化社交网络 生态系统. 建立在底物+ IPFS上.
    • DOTspot -底物生态系统项目共享其开放空间 最新消息.

    • Substrate Migrations -过去的运行时列表 带有注释和示例的存储迁移.

    Community Resources - Guides¶

    • Build Substrate in a few minutes at a low cost -指南 使用 cargo remote 和 Google Cloud 至 减少基材的建造时间； 由基板社区成员撰写 Matej Nemček.

    Frameworks¶

    • FRAME -惯用系统 构建底物运行时.
    • ink! -适用于底物链的智能合约语言.
    • Cumulus -简化的兼容Polkadot的平行链.
    • Bridges -一系列工具 跨链通信.
    • Frontier -基板的端到端以太坊仿真 链.
    • Polkadot-JS -丰富的前端开发框架.
    • SubstraTEE -使用的可信链外执行框架 Intel SGX 可信执行 环境.
    • FRAME Benchmarking - 用于运行时开发的FRAME系统的官方基准测试框架.

    Templates¶

    • Node -基于FRAME的全新 节点，准备进行黑客攻击.
    • Pallet -轻松创建 可插拔的区块链功能.
    • Parachain -新 基于Cumulus的Substrate节点，可以进行黑客攻击（有关更多信息，请参见Frameworks下的'Cumulus'）.
    • Front-End - 使用 Polkadot-JS API和 React 为基于基板的链构建前端.

    Pallets¶

    • Open Runtime Module Library (ORML) -社区 维护了Substrate运行时模块的集合.
    • Sunshine Bounty - 分散式 自治组织（DAO），用于管理赏金计划.
    • Sunshine Identity -受Keybase启发 身份管理.
    • Sunshine Faucet -分配 开发链的资源.
    • Commodities -定义和实现不可替代 令牌接口.

    Client Libraries¶

    • subxt -官方Rust客户.
    • Substrate API Client -Rust客户维护者 Supercomputing Systems AG.
    • Polkadot-JS API -的半官方JavaScript库 基于基质的链.
    • Python Substrate Interface -由维护 Polkascan Foundation.
    • Go Substrate RPC Client -由维护 Centrifuge.
    • Substrate C++ API -由维护 Usetech.
    • Substrate .Net API -由Usetech维护.
    • SubstrateNetApi -由基板社区成员Cedric维护 减速器（@darkfriend77 on GitHub).

    Ecosystem Tools¶

    • Polkadot-JS Apps UI -半官方的区块浏览器和前端 用于基于基质的链.
    • Polkadot-JS Extension -的浏览器扩展 与基于基质的链进行交互.
    • Sidecar -与REST服务同时运行 基板节点.
    • Archive -用于底物链的索引引擎.
    • TxWrapper -离线交易有用的库 创建.
    • Subkey -用于的命令行实用程序 使用加密密钥.
    • Polkascan -Polkascan维护的多链区块浏览器 基础.
    • VSCode Substrate -官方 Visual Studio Code的插件.
    • Parity Signer -将未使用的手机升级为空白 硬件钱包.
    • Fork Off Substrate -帮助引导程序的脚本 处于运行链状态的新链.
    • Substrate debug-kit -一系列工具 和用于调试基于基板的链的库，包括 offline-election, 这是用于预测提名的股权证明选举的工具.
    • sup -用于生成或升级 基板节点.
    • Substrate Dev Hub Utils -非正式 与官方Substrate Developer Hub资源一起使用的实用程序.
    • Subscan -基于基质的链的多网络资源管理器.
    • Substrate Docker Builders - 一套 Dockerfiles和GitHub Actions可自动构建和推送基于Substrate的链的Docker映像.
    • Halva -改善体验的工具链 在底材上进行显影.
    • offchain::ipfs -注入底物 IPFS.
    • Gantree Library -一套用于 通过快速旋转和拆卸来管理由基质驱动的平行链网络.
    • Redspot - 一种 Truffle类工具，用于FRAME的智能合约 合同托盘.
    • Proxy Hot Wallet Demo -一个示范 基于Substrate基元构建的安全，方便且灵活的热钱包架构.
    • Substrate Faucet Bot -基于Python的水龙头 发展宗旨.
    • Hydra -用于底物节点的GraphQL框架 hosted playground . 看看很棒 docs.
    • Substrate Graph -GraphQL索引器 基于基质的链.
    • Megaclite -Polkadot的零知识工具 生态系统.
    • Europa -用于底物运行时执行的沙箱 环境.
    • Jupiter -用于为 FRAME合同托盘.
    • Staking Rewards Collector -要解析的脚本 并输出给定​​Kusama或Polkadot地址的赌注奖励并交叉引用 with daily price data.
    • polkadot-launch -简单的CLI工具可启动 本地Polkadot测试网络.
    • substrate-js-utils -一组有用 使用Polkadot.js API的Substrate Java实用程序.
    • substrate-graph-benchmarks -图 FRAME托盘的基准输出.
    • substrate-balance-calculator - 分解 您的Substrate帐户的余额.
    • substrate-balance-graph - 创建一个 基板地址的令牌余额随时间变化的图表.
    • polkadot-js-bundle -独立的JS 包含Polkadot {JS}库的软件包.
    • substrate-staking -的基本前端 FRAME协会托盘.
    • Aleph.im -可扩展，去中心化的数据库，文件存储和计算 为底物链等提供服务.

    Alternative Implementations¶
