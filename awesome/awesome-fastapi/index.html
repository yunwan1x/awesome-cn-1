    Awesome fastapi

    Awesome FastAPI ¶

    >与FastAPI相关的精选事物的精选列表.

    FastAPI 是一个现代的，高性能的，包含电池的Python Web框架，非常适合构建RESTful API.

    Third-Party Extensions¶

    Admin¶

    • FastAPI Admin -功能管理面板，提供用于对数据执行CRUD操作的用户界面.

    Auth¶

    • FastAPI Auth -可插拔身份验证，支持带有JWT访问和刷新令牌的OAuth2密码流.
    • FastAPI Cloud Auth -FastAPI与云身份验证服务（AWS Cognito，Auth0，Firebase身份验证）之间的简单集成.
    • FastAPI Login -帐户管理和身份验证（基于 Flask-Login).
    • FastAPI JWT Auth -JWT身份验证（基于 Flask-JWT-Extended).
    • FastAPI Permissions -行级权限.
    • FastAPI Security -将身份验证和授权实现为FastAPI中的依赖项.
    • FastAPI Simple Security -开箱即用的API密钥安全性可通过路径操作进行管理.
    • FastAPI Users -帐户管理，身份验证，授权.

    Databases¶

    ORMs¶

    • FastAPI SQLAlchemy -FastAPI和 SQLAlchemy.
    • Tortoise ORM -受Django启发的易于使用的asyncio ORM（对象关系映射器）.
    • FastAPI Example -Tortoise-ORM FastAPI集成的示例.
    • Tutorial: Setting up Tortoise ORM with FastAPI
    • Aerich -乌龟ORM迁移工具.
    • GINO -在SQLAlchemy核心之上为Python asyncio构建的轻量级异步ORM.
    • FastAPI Example
    • ORM -异步ORM.
    • ormar -适用于Python的异步迷你ORM.
    • Piccolo -异步ORM和查询生成器，支持Postgres和SQLite，并带有电池（迁移，安全性等）.
    • FastAPI Examples -在Piccolo中使用FastAPI.
    • Piccolo Admin -使用Piccolo ORM的功能强大且现代的管理GUI.

    Query Builders¶

    • asyncpgsa -周围的包装纸 asyncpg 用于 SQLAlchemy Core.
    • Databases -可以在 SQLAlchemy Core 表达语言.

    ODMs¶

    • MongoEngine -用于从Python使用MongoDB的文档对象映射器（认为是ORM，但适用于文档数据库）.
    • Motor -MongoDB的异步Python驱动程序.
    • ODMantic -与AsyncIO MongoDB ODM集成 Pydantic.

    Other Tools¶

    • Pydantic-SQLAlchemy -将SQLAlchemy模型转换为 Pydantic 楷模.
    • FastAPI-CamelCase -利用FastAPI的CamelCase JSON支持 Pydantic.
    • CamelCase Models with FastAPI and Pydantic -扩展作者的随附博客文章.

    Developer Tools¶

    • FastAPI Code Generator -从OpenAPI文件创建FastAPI应用，从而启用架构驱动的开发.
    • FastAPI Client Generator -根据OpenAPI规范生成mypy和IDE友好的API客户端.
    • FastAPI Profiler -joerick / pyinstrument的FastAPI中间件，用于检查您的服务性能.
    • FastAPI Versioning -API版本控制.
    • Jupyter Notebook REST API -将Jupyter笔记本作为RESTful API端点运行.
    • Manage FastAPI -用于生成和管理FastAPI项目的CLI工具.
    • msgpack-asgi - 自动 MessagePack 内容协商.

    Email¶

    • FastAPI Mail -用于发送电子邮件和附件（个人和批量）的轻型邮件系统.

    Utils¶

    • FastAPI Cache -一个简单的轻量级缓存系统.
    • FastAPI Contrib -一套实用的实用程序：分页，身份验证中间件，权限，自定义异常处理程序，MongoDB支持和Opentracing中间件.
    • FastAPI CRUDRouter -FastAPI路由器，可以为您的模型自动创建并记录CRUD路由.
    • FastAPI Limiter -FastAPI的请求速率限制器.
    • FastAPI MQTT -MQTT协议的扩展.
    • FastAPI Pagination -FastAPI分页.
    • FastAPI Plugins -Redis和Scheduler插件.
    • FastAPI ServiceUtils -用于创建API服务的生成器.
    • FastAPI SocketIO -轻松集成FastAPI和SocketIO.
    • FastAPI Utilities -可重用的实用程序：基于类的视图，推理路由器，周期性任务，定时中间件，SQLAlchemy会话，OpenAPI规范简化.
    • FastAPI Websocket Pub/Sub -经典的发布/订阅模式可轻松在Web上和整个云中实时访问和扩展.
    • FastAPI Websocket RPC -通过Websockets进行的RPC（双向JSON RPC）可以轻松，可靠地进行生产.
    • Prerender Python Starlette -用于Prerender的Starlette中间件.
    • Prometheus FastAPI Instrumentator -适用于您的FastAPI应用程序的可配置模块化Prometheus Instrumentator.
    • SlowApi -速率限制器（基于 Flask-Limiter).
    • Starlette Context -允许您在项目中的任何位置存储和访问请求数据，对记录很有用.
    • Starlette Exporter -FastAPI和Starlette的另一种Prometheus集成.
    • Starlette OpenTracing -对Starlette和FastAPI的Opentracing支持.
    • Starlette Prometheus -用于FastAPI和Starlette的Prometheus集成.
