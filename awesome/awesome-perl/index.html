    Awesome Perl ¶

    精选的Perl5框架，库和软件的精选列表. 受启发 awesome-go.

    不是Perl6模块；-P

    我们还推荐这些列表.

    • Task::Kensho
    • Perlres - A list of resources about Perl
    • PerlMaven.com list of Perl Software

    Args¶

    *用于参数表达和验证的库.

    • Data::Validator -基于规则的类型约束系统验证器.
    • Params::Util -简单，紧凑和正确的参数检查功能.
    • Params::Validate -验证方法/功能参数.
    • Smart::Args

    Audio¶

    • Audio::CD -libcdaudio（cd + cddb）的接口
    • Audio::Wav -用于读取和写入Microsoft WAV文件的模块.

    Benchmarks¶

    用于基准测试的库

    • Benchmark
    • Dumbbench
    • Parallel::Benchmark -多进程基准

    Caches¶

    与缓存软件对话的库

    • CHI -统一的缓存处理界面，认为DBI用于缓存
    • CHI::Driver::DBI -CHI的DBI驱动程序
    • CHI::Driver::DBIC -CHI的DBIx :: Class驱动程序
    • CHI::Driver::Memcached -CHI的Memcached驱动程序
    • CHI::Driver::MongoDB -CHI的MongoDB驱动程序
    • CHI::Driver::Redis -CHI的Redis驱动程序
    • Catalyst::Plugin::Session::Store::CHI -使用CHI模块处理会话数据的存储后端
    • CGI::Application::Plugin::CHI - CGI-App plugin for CHI caching interface
    • Mojolicious::Plugin::CHI -与CHI缓存交互

    Class Builder¶

    支持编写类和元编程的库

    • Class::Accessor::Lite -简单的访问生成器.
    • Class::Accessor::Lite::Lazy -生成惰性访问器.
    • Homer -基于简单原型的对象系统.
    • Mo -微型对象. 钼少.
    • Moo - Class builder supporting meta programming.
    • Moose -唯一的，驼鹿.
    • Mouse -另一个类构建器，例如Moo / Moose.
    • Object::Pad -class Example {有$ x; 方法阅读器{return $ x}}，用于 Cor
    • Object::Tiny -简洁，快速且小巧的类生成器.
