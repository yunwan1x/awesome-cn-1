    Awesome translations

    Awesome Translations ¶

    >将文本的含义从一种语言转移到另一种语言.

    该列表包含工具，书籍，文章，博客，课程以及与翻译相关的所有内容（t9n - 翻译， l10n -本地化， g11n - 全球化， m17n -多语言化， i18n -国际化，是全球化的代名词）.

    Translations¶

    • English
    • Polish
    • Spanish
    • Italian

    Platforms¶

    I18n.

    • Crowdin -基于云的封闭源代码本地化技术和服务公司.
    • GitLocalize -强大的本地化平台，可与GitHub存储库同步，并让您不断翻译其内容.
    • LocaleApp -有效的应用本地化和翻译.
    • Transifex -专有的，基于Web的翻译平台，全球化管理系统（GMS）.
    • Weblate -最积极，最有能力的自由软件社区之一的平台.
    • Zanata -基于Web的翻译平台，供翻译人员，内容创建者和开发人员管理本地化项目.
    • Traduora -在团队中管理翻译工作流程的平台.
    • SimpleLocalize -软件项目的简单翻译管理.
    • Respresso -用于本地化，图像，颜色等的协作式在线资源管理器，可生成特定于平台的文件并将其直接传送到您的源代码中.

    Apps¶

    • RTranslator -第一个基于Google API的Android同步翻译器应用程序.
    • Crow Translate -适用于Linux和Windows的简单轻巧的翻译器，允许使用Google，Yandex和Bing翻译API来翻译和说文本.
    • Copy Translator -适用于Windows，Mac和Linux的应用程序，在剪贴板中复制后会自动翻译和调整文本.
    • Electronjs i18n - I18n manager for Windows.
    • OmegaT -适用于Windows，macOS和Linux的免费翻译记忆库应用程序.
    • Argos Translate -基于Python的完全基于OpenNMT的开源离线翻译应用程序. 它也可以用作Python库.
    • LibreTranslate -基于Argos Translate的100％自托管网络应用程序可翻译文本.

    Tools¶

    • deep-translator -用Phyton编写的灵活的免费和无限（取决于所使用的翻译器）库，可以使用多个翻译器以简单的方式在不同语言之间进行翻译，也可以在提示中直接使用它.
    • I18next -用JavaScript编写并用于JavaScript的国际化框架.
    • react-i18next -基于i18next的强大的React / React Native国际化框架.
    • MDN i18n -使您的浏览器扩展国际化的功能. 您可以使用这些API从与扩展一起打包的语言环境文件中获取本地化的字符串.
    • npm i18n -具有动态json存储的轻量级简单翻译模块.
    • Ruby I18n -Rails国际化（i18n）API.
    • Quasar I18n -Quasar框架中的应用程序国际化（i18n）.
    • Vue I18n -Vue.js的国际化插件.
    • i18n Ally -VSCode的扩展，全部关于i18n.
    • SwiftGoogleTranslate -一个框架，可在Swift中使用Google的云翻译API.
    • Translate -一套旨在帮助使本地化人员的生活更加富有成效和减少挫败感的软件和文档.
    • Pootle -具有翻译界面的在线翻译管理工具，专注于应用程序图形用户界面的本地化. 由LibreOffice使用.
    • Translator -基于Google，Yandex，Microsoft（Bing），百度，阿里巴巴，腾讯，网易（Youdao），搜狗，Deepl等翻译界面的翻译库.
    • Bergamot -浏览器中的客户端翻译.
    • Python-i18n -Python国际化（i18n）软件包.
    • Jsproxy for Crowdin -旨在帮助团队轻松地将网站本地化的技术.
    • attranslate -跨平台应用程序或网站的半自动翻译工具.
    • Polyglot -用于Jekyll博客的快速，轻松，开源的国际化插件.
    • Globalize -一个用于国际化和本地化的JavaScript库，该库利用了官方Unicode CLDR JSON数据.
    • SimpleLocalize CLI -用于在项目文件中查找翻译键的CLI.
    • Moment.js -使用JavaScript解析，验证，操作和显示日期和时间.

    Blogs¶

    • Speakt -全面的资源，可提供业内最佳专业人士有关翻译的一切方面的深入信息.
    • Lexico -提供英语和西班牙语词典的集合，以及有关许多单词和语言的好奇心（在“浏览”部分）. 它由牛津大学出版社制作.
    • The Translator's Teacup -为初学者提供了很好的提示.
    • Speaking of Translation -由翻译和本地化领域的两位杰出专家负责. 您可以听取有关自由​​职业者翻译工作的提示和意见的音频文件.
    • Between Translations -由德语翻译成英语的杰恩·福克斯（Jayne Fox）经营. 它讨论了翻译人员的持续专业发展（CPD），以及使用技术（尤其是CAT工具/ TEnT）来支持他们的工作.
    • eMpTy Pages -提供有关本地化管理，协作翻译平台，翻译管理系统，机器翻译以及本地化行业最大参与者的极其详细的描述和意见.

    Online Courses¶

    • UWashingtonX's I18n and l10n -华盛顿大学提供了3门国际化和数字产品国际化课程.
    • Basic English Grammar and punctuation -加利福尼亚大学为初学者提供的英语语法课程.
    • Intermediate English Grammar and punctuation -加利福尼亚大学为有经验的人提供的英语语法课程.
    • Advanced English Grammar and punctuation -由加州大学提供的专家英语语法课程.
    • Natural Language Processing -教授如何使用机器学习来理解和操纵人类语言. 它需要具备机器学习的工作知识，中级Python经验（包括DL框架）以及演算，线性代数和统计知识的熟练程度.
    • Machine Translation -解释了翻译算法的基本原理. 它由德国卡尔斯鲁厄技术学院提供.

    Books¶

    • Statistical Machine Translation -建立自然语言处理和统计的背景，然后发展基础知识直至当前的研究. 最终，读者可以构建自己的翻译系统. 作者是 Philipp Koehn.
    • Neural Machine Translation -对机器翻译的介绍从神经网络方法的基础开始，并达到了最新水平，同时提供了历史，语言和应用方面的知识. Python中的代码示例为实现提供了动手的蓝图. 作者仍然是Philipp Koehn.
    • Translation: An advanced resource book for students -从多种语言和文化角度介绍翻译的理论和实践. 作者是 Basil Hatim 和 Jeremy Munday.

    Other¶

    • Google Crowdsource -由Google开发的众包平台，旨在通过面向用户的不同算法培训（带有翻译和翻译验证部分）来改善一系列Google服务.
    • Localizejs -将网站和应用程序翻译成新语言并简化翻译工作流程.
    • LocalizationLab -由7000多个贡献者组成的全球社区，他们支持Internet自由工具的翻译和本地化.
    • Pontoon -各种Mozilla本地化项目.
