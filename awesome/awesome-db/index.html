    Database

    太棒了 - db ========= 精选的精彩数据库库，资源和闪亮的东西 Numetric Technologies

    Clojure¶

    • Datomic
    • clojure.jdbc
    • cravendb

    C / CPP¶

    • Aerospike - Aerospike数据库服务器 - 闪存优化，内存中，nosql数据库
    • Alchemy-Database - AlchemyDB现在是Aerospike
    • cstore_fdw - 使用PostgreSQL进行分析的快速柱状存储 website
    • LMDB - 具有完整ACID语义的非常快速的嵌入式键/值存储.
    • memcache - 免费和开源，高性能，分布式内存对象缓存系统
    • monetdb - website
    • mongo - Mongo数据库http://www.mongodb.org/
    • facebook/mysql-5.6 - Facebook的Oracle MySQL v5.6数据库分支
    • twitter/mysql - 在Twitter上维护和使用MySQL fork https://github.com/twitter/mysql/wiki
    • postgres
    • Redis - Redis是一个内存数据库，可以在磁盘上保留. 数据模型是键值，但支持许多不同类型的值：字符串，列表，集，排序集，哈希值 website
    • redis-nds - 这是Redis的一个版本，用于实现NDS（朴素磁盘存储）. 有关此功能的文档包含在README.nds中.
    • rethinkdb - 一个开源的分布式JSON文档数据库，具有令人愉快且功能强大的查询语言. http://www.rethinkdb.com
    • rocksdb - 用于快速存储的嵌入式键值存储http://rocksdb.org
    • ssdb - SSDB - 一个快速的NoSQL数据库，是Redis http://ssdb.io的替代品
    • SQLite - 一个完全嵌入的，功能齐全的关系数据库，只需几百kk就可以包含在您的项目中.
    • tokudb-engine- TokuDB是MySQL和MariaDB的高性能，写优化，压缩，事务存储引擎
    • TokuMX- TokuMX是MongoDB的高性能，并发，压缩，直接替换引擎
    • VoltDB - VoltDB是一种水平可扩展的内存中SQL RDBMS，专为具有极高读写吞吐量要求的应用程序而设计.

    dot-net¶

    • RavenDB - 支持linq的.NET文档数据库http://ayende.com/Blog/
    • OrigoDB - 用于NET / Mono的内存嵌入式数据库引擎

    Erlang¶

    • riak - Riak是Basho Technologies的分散数据库.
    • Pulsedb - Pulsedb是一个时间序列数据库服务器和库.

    Go-lang¶

    • bolt - Go的低级键/值数据库.
    • diskv - 本土磁盘支持的键值存储.
    • go-cache - 内存中的密钥：Go的值存储/缓存（类似于Memcached）库，适用于单机应用程序.
    • goleveldb - 执行 LevelDB Go中的键/值数据库.
    • groupcache - Groupcache是​​一个缓存和缓存填充库，在许多情况下用作memcached的替代品.
    • influxdb - 用于指标，事件和实时分析的可扩展数据存储
    • ledisdb - Ledisdb是一款基于LevelDB的高性能NoSQL，如Redis.
    • roshi - Roshi是一个用于时间戳事件的大规模CRDT集实现.
    • skydb.io - Sky是一个开源数据库，用于灵活，高性能的行为数据分析.
    • tiedot - 由Golang提供支持的NoSQL数据库.

    JAVA¶

    • cassandra - Cassandra是一个分区行商店. 行被组织成具有所需主键的表.
    • elasticsearch - 开源，分布式，RESTful搜索引擎 website
    • lmdbjni - LMDB for Java，它是一个非常快速的嵌入式键/值存储，具有完整的ACID语义.
    • mapdb - MapDB提供由磁盘存储或堆外内存支持的并发映射，集和队列. 它是一个快速且易于使用的嵌入式Java数据库引擎.http：//www.mapdb.org
    • neo4j - Neo4j 是世界领先的图形数据库.
    • orientdb - OrientDB是一个开源NoSQL DBMS，具有Document和Graph DBMS的功能.
    • voldemort - 亚马逊Dynamo的开源克隆. website
    • Titan - 分布式图形数据库http://titandb.io

    Javascript¶

    • couchdb - Apache CouchDB™是一个数据库，它使用JSON作为文档，使用JavaScript作为MapReduce索引，使用常规HTTP作为其API [网站]（http://couchdb.apache.org/）
    • mongodb - MongoDB是一个文档数据库，提供高性能，高可用性和易扩展性. 文档（对象）很好地映射到编程语言数据类型. 嵌入式文档和数组减少了对连接的需求. 动态模式使多态更容易. [网站]（https://www.mongodb.org/）

    Scala¶

    • BlinkDB - BlinkDB：​​超大数据的亚秒级近似查询[网站]（http://blinkdb.cs.berkeley.edu/）
