    免费软件

    Awesome Free Software ¶

    此列表中的软件以允许任何人出于任何目的使用，修改和共享该软件的条款进行分发. 最受欢迎的两个许可证是 MIT License 和 GNU GPL， 但是这里有 many other licenses 授予这些权限.

    Software¶

    Audio¶

    • Audacity - Sound recording and post-processing. (GNU GPLv2+)
    • Ardour -数字音频工作站（DAW）. （GNU GPLv2)
    • Libre.fm -免费流式传输，下载，混音和共享音乐. （GNU AGPLv3)
    • LMMS -完整的数字音频工作站. （GNU GPLv2+)
    • mStream -用于在多个设备上同步和流式传输音乐的软件套件. （GNU GPLv3)
    • Nuclear Music Player -流音乐播放器，可自动从免费资源中查找音乐. （GNU AGPLv3)

    Command Line Tools¶

    • asciinema -终端会话记录器. （GNU GPLv3)
    • Cygwin -Windows的类Unix环境，提供对Linux系统上常见的许多工具的访问. （GNU GPLv3)
    • Logdissect -CLI工具和Python API，用于分析日志文件和其他数据. （MIT)
    • PuTTY -通过SSH，Telnet，SCP和rlogin协议以及原始套接字连接进行连接的远程终端仿真器. （MIT)
    • ranger -最小，轻巧且美观的GNU / Linux文件浏览器. 它运行在终端窗口中，因此非常适合i3之类的窗口管理器. vi键绑定允许执行快速操作，包括重命名和删除. （GNU GPLv3)
    • GNU stow -GNU Stow是一个symlink服务器场管理器，它采用位于文件系统上单独目录中的不同软件和/或数据包，并使它们看起来安装在同一位置. （GNU GPLv3)

    Communication¶

    • Conversations -适用于Android的XMPP / Jabber客户端. （GNU GPLv3)
    • Gajim -桌面版XMPP / Jabber客户端. （GNU GPLv3)
    • Jitsi Meet -使用WebRTC进行安全的视频会议和桌面共享. （Apache License 2.0)
    • Mail-in-a-Box -易于部署的云计算机邮件服务器软件包. （CC0)
    • Matrix -分散，持久和可互操作的通信的开放标准. （Apache License 2.0)
    • Mattermost -开源，私有云替代Slack. （Licensing)
    • ProtonMail -安全的基于Web的电子邮件服务. （MIT)
    • Ring -通过分布式服务进行电话，电话会议和媒体共享. （GNU GPLv3)
    • Signal -私人消息传递和呼叫桌面，Android和iOS. （GNU GPLv3)
    • Tox -具有语音，视频和屏幕共享功能的加密和分布式消息传递平台. （GNU GPLv3)
    • Tutanota -免费加密电子邮件，并支持企业帐户. （GNU GPLv3)

    File Management¶

    • 7-Zip -用于创建和打开压缩文件的文件存档器. （GNU LGPLv2.1+)
    • Filestash -类似于Dropbox的网络客户端，用户可以在其中携带自己的后端（FTP，SFTP，Webdav，S3，Minio等）. （GNU AGPLv3)
    • FileZilla -通用FTP解决方案. （GNU GPLv2+)

    File Sharing¶

    • qBittorrent -易于使用的BitTorrent客户端. （GNU GPLv2+)

    Graphics¶

    • Blender -用于建模，仿真和动画的3D创建套件. （GNU GPLv2+)
    • ExifCleaner -GUI应用程序，通过简单的拖放操作即可从图像和视频中删除exif元数据. （MIT)
    • GIMP -图像处理软件. （GNU GPLv3)
    • Inkscape -适用于所有平台的专业矢量图形编辑器. （GNU GPL)

    • Pinta - Gtk# clone of Paint.NET. (MIT)
    • Piskel -在线精灵编辑器. （Apache License 2.0)

    IDEs¶
