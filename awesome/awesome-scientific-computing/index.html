    Awesome scientific computing

    Awesome Scientific Computing ¶

    >用于科学计算和数值分析的有用资源.

    科学计算和数值分析是旨在提供 借助以下方法解决来自科学各个领域的大规模问题的方法 电脑. 典型的问题是常微分方程和偏微分方程（ODE）， PDE），离散化和线性代数问题的解 them.

    Basic linear algebra¶

    • BLAS -用于执行基本向量和矩阵运算的标准构建块. （Fortran，公共领域， GitHub)
    • OpenBLAS -基于GotoBLAS2的优化的BLAS库. （C和汇编，BSD， GitHub)
    • BLIS -高性能的类似于BLAS的密集线性代数库. （C，BSD，GitHub）
    • LAPACK -求解线性方程组，线性最小二乘，特征值问题等的例程. （Fortran，BSD， GitHub)
    • Eigen -线性代数的C ++模板库. （C ++，MPL 2， GitLab)
    • Ginkgo -高性能manycore线性代数库，专注于稀疏系统. （C ++，BSD， GitHub)
    • blaze -用于密集和稀疏算术的高性能C ++数学库. （C ++，BSD，Bitbucket）

    Multi-purpose toolkits¶

    • PETSc -由PDE建模的科学应用的并行解决方案. （C，2句BSD， GitLab)
    • DUNE Numerics -使用基于网格的方法解决PDE的工具箱. (C++, GPL 2, GitLab)
    • SciPy -统计，优化，积分，线性代数等的Python模块 （Python，主要是BSD， GitHub)
    • NumPy -使用Python进行科学计算所需的基本软件包. （Python，BSD， GitHub)

    Finite Elements¶

    • FEniCS -用于解决Python和C ++中的PDE的计算平台. （C ++ / Python，LGPL 3， Bitbucket)
    • libMesh -使用非结构化离散化进行PDE数值模拟的框架. （C ++，LGPL 2.1， GitHub)
    • deal.II -支持创建有限元代码的软件库. （C ++，LGPL 2.1， GitHub)
    • Netgen/NGSolve -高性能多物理场有限元软件. （C ++，LGPL 2.1， GitHub)
    • Firedrake -使用有限元方法自动求解PDE的系统. （Python，LGPL 3， GitHub)
    • MOOSE -面向对象的多物理场仿真环境. （C ++，LGPL 2.1， GitHub)
    • MFEM -用于有限元方法的免费，轻量级，可扩展的C ++库. （C ++，LGPL 2.1， GitHub)
    • SfePy -Python中的简单有限元. （Python，BSD， GitHub)
    • FreeFEM -高级Multiphysics-multimesh有限元语言. （C ++，LGPL， GitHub)
    • libceed -有效的可扩展离散化代码. （C，2句BSD， GitHub)

    Meshing¶

    • Gmsh -具有预处理和后处理功能的三维有限元网格生成器. （C ++，GPL， GitLab)
    • pygmsh -Gmsh的Python界面. （Python，GPL 3，GitHub）
    • MeshPy -优质的三角形和四面体网格生成. （Python，麻省理工学院， GitHub)
    • meshio -各种网格格式的I / O，文件转换. （Python，麻省理工学院，GitHub）
    • CGAL -计算几何的算法. （C ++，混合的LGPL / GPL， GitHub)
    • pygalmesh -用于CGAL的3D网格化功能的Python界面. （Python，GPL 3，GitHub）
    • mshr -FEniCS的网格生成组件. （Python，GPL 3，Bitbucket）
    • MOAB -表示和评估网格数据. （C ++，主要是LGPL 3， Bitbucket)
    • TetGen -优质的四面体网格生成器和3D Delaunay三角剖分器. （C ++，AGPLv3）
    • Triangle -二维质量网格生成器和Delaunay三角剖分器. （C，非自由软件）
    • optimesh -三角网格平滑. （Python，GPL 3，GitHub）
    • distmesh -用于非结构化三角形和四面体网格的简单生成器. （MATLAB，GPL 3）
    • QuadriFlow -一种可扩展且鲁棒的四边形方法. （C ++，BSD， GitHub)
    • trimesh -加载和使用三角形网格，重点放在水密表面上. （Python，麻省理工学院， GitHub)
    • dmsh -受distmesh启发，用于非结构化三角形网格的简单生成器. （Python，GPL 3，GitHub）
    • pmp-library -多边形网格处理库. （C ++，MIT，雇主免责声明， GitHub)
    • Mmg -强大，开放源代码和多学科的软件，可用于重新定型. （C，LGPL 3， GitHub)
    • meshplex -单工网格的快速工具. （Python，GPL 3，GitHub）
    • TetWild -在野外强大的四面体啮合. （C ++，GPL 3， GitHub)
    • TriWild -带有曲线约束的鲁棒三角剖分. （C ++，MPL 2， GitHub)
    • fTetWild -在野外快速四面体啮合. （C ++，MPL 2， GitHub)
    • SeismicMesh -平行的2D / 3D三角形/四面体网格生成，带子去除. （Python和C ++，GPL 3，GitHub）

    Data formats¶

    • NetCDF -用于面向阵列的科学数据的软件库和数据格式. （C / C ++ / Fortran / Java / Python，[自定义开源 许可]（https://www.unidata.ucar.edu/software/netcdf/copyright.html）， GitHub)
    • HDF5 -用于存储和管理数据的数据模型，库和文件格式. （C / Fortran，BSD， GitHub)
    • XDMF -可扩展数据模型和格式，用于来自高性能计算代码的数据. （C ++， GitLab)
    • Zarr -用于存储分块，压缩的N维数组的格式. （Python，麻省理工学院， GitHub)

    Sparse linear solvers¶

    • SuperLU -大型，稀疏，非对称线性方程组的直接解. （C，主要是BSD， GitHub)
    • KryPy -用于线性代数系统求解的Krylov子空间方法. （Python，麻省理工学院，GitHub）
    • PyAMG -Python中的代数多重网格求解器. （Python，麻省理工学院， GitHub)
    • hypre -高性能预处理器和求解器库. （C，Apache 2.0 / MIT， GitHub)

    Visualization¶

    • ParaView -基于VTK的多平台数据分析和可视化应用程序. （C ++，BSD， GitLab)
    • VTK -处理图像并创建3D计算机图形. （C ++，BSD， GitLab)
    • Mayavi -使用Python进行3D科学数据可视化和绘图. （Python，BSD， GitHub)
    • Polyscope -用于3D几何处理的查看器和用户界面. （C ++，麻省理工学院， GitHub)
    • PyVista -通过简化的VTK界面进行3D绘图和网格分析. （Python，麻省理工学院， GitHub)
    • vedo -用于基于VTK对3D对象进行科学分析和可视化的库. （Python，麻省理工学院， GitHub)
    • yt -用于体积数据分析和可视化的工具包. （Python，BSD， GitHub)
    • F3D -具有科学可视化工具的跨平台，快速，简约的3D查看器. （C ++，BSD， GitLab)
    • TTK -拓扑数据分析和可视化. （C ++ / Python，BSD， GitHub).

    Other libraries and tools¶

    • FFTW -在一个或多个维度上的离散傅立叶变换，具有任意输入大小，实数和复数. （C，GPL2， GitHub)
    • Qhull -凸包，Delaunay三角剖分，Voronoi图，围绕某个点的半空间交点等. （C / C ++， custom open source license, GitHub)
    • GSL -随机数生成器，特殊函数和最小二乘拟合等. （C / C ++，GPL 3， Savannah)
    • OpenFOAM -免费的开源CFD（计算流体动力学）软件. （C ++，GPL 3， GitHub)
    • quadpy -Python中的数值积分（正交，容积）. （Python，GPL 3，GitHub）
    • FiPy -有限体积的PDE求解器. （Python，[自定义开源 许可]（https://www.nist.gov/open/copyright-fair-use-and-licensing-statements-srd-data-software-and-technical-series-publications）， GitHub)
    • accupy -适用于Python的准确的总和和点积. （Python，GPL 3，GitHub）
    • SLEPc -特征值问题计算的可扩展库. （C，2句BSD， GitLab)
    • Chebfun -具有约15位精度的函数计算. （MATLAB，BSD， GitHub)
    • pyMOR -使用Python减少模型订单. （Python，2句BSD， GitHub)
    • cvxpy -凸优化问题的建模语言. （Python，Apache 2.0， GitHub)
    • PyWavelets -Python中的小波变换. （Python，麻省理工学院， GitHub)
    • NFFT -非等距快速傅立叶变换. （C / MATLAB，GPL 2， GitHub)
    • preCICE -用于分区多物理场仿真（FSI，CHT等）的耦合库. （C ++，LGPL 3， GitHub)
    • orthopy -有效地计算正交多项式. （Python，GPL 3，GitHub）
    • pyGAM -Python中的通用加性模型. （Python，Apache 2.0， GitHub)
    • Dedalus -用频谱方法求解偏微分方程. （Python，GPL 3， GitHub)
    • PyGMO -大规模并行优化. （Python / C ++，MPL 2， GitHub)
