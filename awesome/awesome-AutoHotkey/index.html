    AutoHotkey

    Awesome AutoHotkey ¶

    精选的超赞清单 AutoHotkey 库，库发行版，脚本，工具和资源. 受其他启发 awesome lists . 请阅读 CONTRIBUTING.md 在贡献之前.

    过时或已停产，但仍可在 Historical.md

    发展状况：

    Libraries¶

    有用的AutoHotkey库列表. 库是具有一些可重用功能的代码，可以将这些功能与您自己的代码结合以创建新功能.

    Clipboard¶

    • WinClip -作者Deo-WinClip是一个剪贴板操作类，扩展了AutoHotkey的剪贴板功能，包括对RTF，HTML和图像的支持. 论坛主题： link.

    Console¶

    • AHKonsole -由G33kdude提供-用于控制台支持的基于类的AutoHotkey库. 该库使您可以创建一个代表要与之交互的控制台的对象，以及多个控制台缓冲区对象，以促进双重缓冲. 论坛主题： link.
    • LibCon -由joedf提供-控制台支持的AutoHotkey库. 该库使您可以编写控制台应用程序并与其他控制台实例进行交互. 基本上，此库可简化与控制台的编写和交互有关的所有工作. 论坛主题： link.

    Data format¶

    • AHK_ctable -通过Hoppfrosch-库以表格格式处理字符串-论坛主题： link.
    • AutoHotkey-JSON -由cocobelgica-AutoHotkey的JSON库. 论坛主题： link.
    • CSV -trueski / kdoske编写-使用CSV文件和Listview函数的库. 论坛主题： link.
    • List manipulation functions -通过Laszlo-函数库来操作逗号分隔的列表. 论坛主题： link.
    • ObjCSV -由JnLlnd-库，用于将CSV文件加载/保存到对象和Listview函数. 论坛主题： link.
    • ObjDump/ObjLoad -通过HotKeyIt-将对象串行化/反序列化到变量/内存.
    • SerDes -由cocobelgica-序列化/反序列化AutoHotkey对象结构. 论坛主题： link.
    • Table -由VxE-库以表格（TSV）格式和Listview函数操作字符串. 论坛主题： link.
    • XA -由trueski / hi5-向/从XML序列化/反序列化数组. 论坛主题： link.

    Data Structures and Algorithms¶

    • Facade -由Shambles-一组功能编程库. -论坛主题： link
    • HashTable -由Shambles-AutoHotkey的哈希表实现.
    • LibCrypt -由不同的作者-加密和编码功能的集合.
    • Type_Checking -通过Shambles-类型检查AutoHotkey-论坛主题： link

    Database¶

    • AHKDb -由AHKDb提供-用于制表符分隔数据的数据库库.
    • ahkDBA -由IsNull-一个OOP-SQL数据库访问框架. 论坛主题： link.
    • Class_SQLiteDB -仅由我-AHK SQLite API包装器类. 论坛主题： link.
    • Leya - MySQL API -通过kevgk-在自动热键中使用MySQL数据库，而无需将服务器凭据公开给客户端.

    Filesystem¶

    • FileGetProperties -by kon-检索扩展文件属性的功能.

    Graphics¶

    • GDIp -通过tic-功能齐全的库，可帮助与Microsoft的gdiplus.dll进行交互-论坛线程： link.
    • AHKv2-GDIP -更新了与AHK v1.1和AHK v2兼容的上述GDI +库-论坛主题： link.
    • GDIp_ImageSearch -通过tic-使用gdiplus.dll在屏幕上搜索图像实例的库. 请参阅该线程的末尾以获取MasterFocus的改进版本，或查看他的主题 GitHub repo here
    • Simple GDI class -由GeekDude-一个旨在简化使用低级GDI函数的类.
    • Particle System -通过tidbit-使用GDI +将粒子添加到GUI或屏幕上的简单类. 论坛主题： link.

    GUI¶

    Combobox¶

    • CbAutoComplete -通过Pulover-自动完成AHK组合框中的键入值. 论坛主题： link

    Custom Controls¶

    • Rebar -通过Pulover-AutoHotkey钢筋自定义控件的AHK类. 论坛主题： link
    • Toolbar -通过Pulover-AutoHotkey工具栏自定义控件的AHK类. 论坛主题： link

    Edit¶

    • Edit v2.0 -by jballi-轻巧且功能强大的默认Edit控件的库，用于显示和编辑文本. 论坛主题： link

    General¶

    • AutoXYWH -通过tmplinshi-调整GUI大小时自动移动和调整控件大小.
    • TaskDialog -仅由我-Win Vista +增强的MsgBox- link
    • OnWin -通过cocobelgica-窗口事件（WinWaitXXX异步）上的调用函数. 论坛主题： link
    • CGUI -由ChrisS85-AutoHotkey的面向对象的GUI库. 论坛主题： link
    • Class_ScrollGUI -仅由我-创建一个可滚动GUI作为AHK GUI窗口的父级. 论坛主题： link

    ListBox¶

    • LBEX -由 just me - a collection of utility functions for ListBoxes. Forum thread: link
    • TransparentListBox -仅由我-为AHK GUI提供透明的列表框控件. 论坛主题： link

    ListView¶

    • LV_Colors -仅由我-GUI ListView的单元格或行的单独背景和/或文本颜色. 论坛主题： link
    • LV_EX -仅由我-AHK GUI ListView控件的一些附加功能. 论坛主题： link
    • LV_InCellEdit -仅靠我自己-ListView控件的单元内编辑. 论坛主题： link
    • LV_Rows -通过Pulover-AHK ListView控件的附加功能. 论坛主题： link

    Menu¶

    • [[Lib]菜单]（https://autohotkey.com/boards/viewtopic.php?t=3068）-仅由我-与AHK菜单相关的某些功能. 论坛主题： link

    Web¶

    • Neutron -作者：G33kDude-使用AutoHotkey构建基于HTML的用户界面的工具集. 论坛主题： link

    Hotkeys¶

    • CHotkeyControl -由evilC-替代支持鼠标按钮等的AHK热键GuiControl（部分成熟）.
    • HParse -通过Avi-将有意义的快捷方式（Ctrl + X）转换为AutoHotkey语法（^ x）的功能.

    Joystick¶

    • CvJoyInterface -由evilC-使用AHK控制vJoy虚拟游戏杆.
    • JoystickWrapper - by evilC - Full event-based, 8 axis, 128 button, 4 POV joystick reading (C# DLL, Uses Lexikos' CLR).
    • XInput -通过Lexikos-使用XInput读取XBOX游戏手柄（仅用于独立读取L / R触发器的方式），控制隆隆声马达.

    Maths¶

    • calc() -数学表达式评估，包括方括号.
    • Eval -通过Pulover-评估字符串中的表达式. 论坛主题： link
    • Scientific Maths -Avi编写的-促进高精度数学的图书馆.
    • Time() -通过HotkeyIt-计算日期之间的天，小时，分钟，秒. 论坛主题： link

    Memory¶

    • classMemory -由RHCP（Kalamity）提供-具有模式扫描的AHK内存读取/写入类. 论坛主题： link

    Networking¶

    • AHKhttp -基本的HTTP服务器. 论坛 link
    • AHKsock -通过TheGood-基于函数的套接字库. 支持TCP. 论坛 link
    • Chrome.ahk -作者：G33kDude-使用本机AutoHotkey自动化Google Chrome-论坛 link
    • FTP - by jNizM - AutoHotkey wrapper for FTP Sessions (Class) - Forum link
    • Socket Class (überarbeitet) -由Bentschi-基于类的套接字库. 支持TCP和UDP.
    • Socket.ahk -由GeekDude-基于Bentschi的套接字库-论坛 link
    • WebSocket.ahk -由GeekDude-基于类的WebSocket库-论坛 link
    • WinSCP.ahk -Lipkau-Lib允许在AHK中使用WinSCP

    Plotting (graphs, bars, charts and etc)¶

    • Excel Charts -通过Xx7-用于在Excel中创建图形的库，将图形另存为图像并在GUI中显示.
    • XGraph -通过SKAN-功能库，以图形方式绘制实时数据.
    • SVGraph -CapnOdin撰写-SVGraph将图形和图表引入AutoHotkey. 论坛 link
    • gdiChartLib -由nnnik-autohotkey的gdip图表库. 论坛 link

    System¶

    • RunAsTask -通过SKAN-在没有UAC提示的情况下自动提升脚本.
    • Vista Audio Control Functions -由Lexikos提供-提供一些SoundSet / SoundGet子命令的替代选项，以及SoundSet / SoundGet不支持的一些其他功能. 论坛主题： Link

    Text manipulation¶

    • String Things -通过tidbit-独立的字符串操作功能.
    • TF -by hi5-用于操作文本文件的功能，例如 .txt，.ahk，.html，.css等和字符串（或变量）. 论坛主题： link.

    Scripts¶

    有用的AutoHotkey脚本列表. 脚本是旨在用作独立程序的代码，并不旨在与其他代码集成.

    Clipboard¶

    • CL3 -带有插件（搜索，预定义的插槽，ClipChain，FIFO，编辑器等）的剪贴板管理器（仅文本）. 论坛主题 link.
    • ClipBoardMonitor -监视剪贴板的变化，显示文字计数工具提示或图片的临时GUI.
    • Clipjump -是Windows的多剪贴板管理实用程序. 源代码： GitHub . 论坛主题： link 1, link 2.

    Filesystem¶

    • Belvedere -根据文件名，扩展名，大小，使用期限等，设置规则以对文件执行操作（移动，复制，删除等）. 更多信息 link.

    Graphics¶

    • Fun with GDIPlus -有趣的GDI +示例.

    GUI¶

    • Examples of Non-Standard GUIs (ActiveX, GDI, etc.) -使用非标准方法生成漂亮用户界面的GUI的示例.

    Maths¶

    • Monster -计算字符串中的数学表达式（计算器）.
    • Unit Converter -具有最常用的英制和科学单位以及从长度到密度到热导率的最常见数量的单位转换器. 还包括一个有关物理和数学常数的部分.

    Mouse¶

    • EitherMouse -多个鼠标，单独的设置，第二个鼠标上的自动交换鼠标按钮. 论坛主题： link.
    • MouseGestureL -通过鼠标手势控制应用程序. 手势和动作可以通过可定制的界面定义. 英文和日文文档-日语主页 link
    • Radial Menu -强大的热键，启动器，鼠标手势系统等（外观）-论坛主题： link

    Typing¶
