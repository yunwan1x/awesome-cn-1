    JMeter

    Awesome JMeter ¶

    精选的资源涵盖 Apache JMeter 以及相关的东西和闪亮的东西：插件，集成，测试技术，devops实践等.

    Apache JMeter 是开放源代码的纯Java应用程序，旨在加载测试功能行为和衡量性能.

    这个清单是从 an occasional answer 从Stack Exchange和与JMeter相关的个人链接收集中获得了进一步的启发 awesome 项目并通过这些进行了改进 amazing contributors.
