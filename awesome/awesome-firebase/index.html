    Awesome Firebase ¶

    最最新清单 Firebase 互联网必须提供的文档，讲座，工具，示例和文章.

    Translations: 🇬🇧 en · 🇷🇺 ru

    Firebase 是一个基于 Google Cloud Platform 提供服务和跨平台的SDK！

    传奇：博客文章·示例·文档·库·工具·讲座/视频

    Featured (new releases)¶

    • Remote Styles with Remote Config -存储在远程配置中的CSS的动态/有条件加载. （Launch post).
    • Firecoil -使用图片加载库Coil在您的Android应用中从GCS加载图片.
    • Svarog -使用JSON模式生成的安全规则帮助程序功能进行Cloud Firestore模式验证.
    • Firebase Extensions -Firebase Extensions为您的应用程序提供了扩展功能，而无需自己研究，编写或调试代码.
    • Firebase App Distribution -将应用的预发布版本分发给您值得信赖的测试人员.

    Official Firebase Docs & Quickstarts¶

    • Firebase Documentation -Firebase官方文档.
    • Firebase Status Dashboard -此页面提供有关Firebase一部分服务的状态信息.
    • Firebase Quickstarts -官方Firebase快速入门.
    • Google Codelabs | Firebase -Google Developers Codelabs提供了指导性的，动手的编码经验.

    Web¶

    • React Fire -带有钩子，上下文提供程序和组件的官方Firebase React库，可轻松与Firebase进行交互.
    • React Firebase Hooks -用于Firebase服务的React Hooks.
    • Firebase UI -FirebaseUI是用于Web的开源JavaScript库，它在Firebase SDK之上提供了简单的，可自定义的UI绑定，以消除样板代码并推广最佳实践.
    • Firebase UI for React -用于FirebaseUI Web的React Wrapper.
    • GeoFire for JavaScript -使用Firebase进行实时位置查询.
    • FirePad -由Firebase提供支持的协作文本编辑器.
    • Ember Fire -Firebase的官方Ember数据适配器.
    • Firebase Dart -Firebase的飞镖包装.
    • PolymerFire -用于Firebase的Polymer Web组件.
    • VueFire -Vue.js的Firebase绑定.
    • Angular Fire 2 - Official library for Firebase and Angular.
    • Re-base -中继启发式库，用于构建React.js + Firebase应用程序.
    • React Redux Firebase -Firebase的Redux绑定. 包括与React一起使用的高阶组件.
    • GatsbyJS Firebase Data Source -使用Gatsby将Firebase数据直接查询到静态生成的页面中.
    • Apollo Link Firebase -提供与RealtimeDB的本地GraphQL接口. 数据库在本地同步到设备，Apollo Link提供对本地数据库的查询.
    • BuckleScript Bindings for Firebase -用于Firebase的BuckleScript绑定，可在ReasonML项目中使用.
    • Angular Firebase PWA -由Firebase驱动的Angular PWA. 它可以作为学习此堆栈并推出更复杂功能的基础.
    • FireSQL -使用SQL语法查询Firestore. 发出获取所需数据所需的最少查询量.
    • Hosting Version History -自动删除旧版本的网站部署.
    • Firestorter -使用MobX（也用于react-native），以零的努力在React中使用Firestore.
    • Nextbase -适用于想要快速启动项目的开发人员的Next.js，Redux和Firebase样板.

    Mobile¶

    • Firebase Flutter Documentation -官方Firebase Flutter设置.
    • NativeScript plugin Firebase -Firebase的NativeScript插件.
    • FlutterFire -在跨平台中使用Firebase服务 Flutter 应用.
    • React Native Firebase -经过测试的功能丰富的模块化Firebase实施，用于React Native. 同时支持iOS和Android平台.
    • React Native Firebase Cloud Messaging - 用于Firebase Cloud Messaging和本地通知的React Native模块.
    • Expo Native Firebase -用于Firestore，通知，分析，存储，消息传递，数据库的本地Firebase Expo应用（iOS，Android）演示.
    • Flutter Calendar App - 新的Flutter应用程序实现了一个简单的移动日历应用程序，用于将基本事件存储到Firebase云数据库中.

    Android¶

    • GeoFire for Java -使用Firebase进行实时位置查询.
    • Firebase UI -为Firebase优化的UI组件.
    • FireXtensions -Firebase Android SDK的非官方Kotlin扩展.

    iOS¶

    • GeoFire for Objective-C -使用Firebase进行实时位置查询.
    • Firebase UI -用于Firebase的iOS UI绑定.
    • MLKit - ARCore -检测对象并在增强现实中使用3D标签标记它们的示例. 使用Firebase ML Kit，ARCore和Firebase RTDB.
    • MLKit - ARKit -使用Firebase ML Kit检测对象并在增强现实中使用3D标签标记对象的示例.

    Server-side (Cloud Functions, BigQuery etc)¶

    • Firebase Admin Documentation -官方Firebase Admin SDK服务器设置.
    • Functions Samples -集合了示例应用程序，这些示例应用程序使用Cloud Functions for Firebase展示了流行的用例.
    • Express Server on Cloud Functions -在Cloud Functions上托管Express服务器.
    • GraphQL Server on Cloud Functions -在Cloud Functions上使用GraphQL中间件托管Express服务器.
    • Compiled Code with Cloud Functions -使用Babel，TypeScript编译器或ParcelJS将Flow，TypeScript或ReasonML编译为正确的Node运行时.
    • BigQuery & Google Analytics -如何使用BigQuery在Google Analytics for Firebase中创建封闭的渠道.
    • Official Cloud Function #Firecasts -有关理解云功能如何工作的YouTube视频系列.
    • Firebase Hosting for Cloud Run Services -具有托管重写和云运行服务的动态内容.
    • Scheduled (Cron) Cloud Functions for Firebase -用于Firebase云功能的Firebase本地Cron触发器.
    • Integrify -使用预先罐装的Cloud Functions触发器在Firestore中加强参考和数据完整性.
    • Free Product Analytics with Firebase + BigQuery + Rakam -如何通过BigQuery Export和Rakam对Firebase事件数据进行行为和细分分析.

    CLI & Editor¶

    • VSCode Firebase Explorer -探索和管理您的Firebase项目.
    • Firebase Tools -Firebase命令行工具.
    • Firebase CI -简化的Firebase交互以实现持续集成.
    • VSFire -VSCode扩展，用于使用Firestore安全规则和索引突出显示语法和完成代码.
    • Firebase Firestore Snippets -包含VS Code编辑器中Firebase和Firestore的代码段.
    • Fuego -Firestore客户端CLI支持带有过滤和分页功能的文档添加/更新/查询.
    • Firestore Rules Generator -基于Google的协议缓冲区格式的Cloud Firestore的官方（但实验性）Firebase规则生成器.
    • Firepit -Firepit是Firebase CLI的独立，可移植版本，没有任何依赖（包括Node.js）.
    • Fireward -易于使用的Firestore规则语言，类似于Firebase Bolt.
