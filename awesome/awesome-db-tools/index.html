    Awesome db tools

    Awesome Database Tools ¶

    >社区驱动的数据库工具列表

    在这里，我们将收集有关很棒的有用和很棒的实验工具的信息，这些工具简化了DBA，DevOps，Developers和仅凡人的数据库的工作.

    随意添加有关您自己的db-tools或您喜欢的第三方db-tools的信息.

    IDE¶

    • AnySQL Maestro -用于数据库管理，控制和开发的高级多功能管理工具.
    • Aqua Data Studio -Aqua Data Studio是用于数据库开发人员，DBA和分析师的生产力软件.
    • Database .net -多个数据库管理工具，支持20多个数据库.
    • DataGrip -JetBrains的数据库和SQL跨平台IDE.
    • DBeaver -免费的通用数据库管理器和SQL客户端.
    • dbForge Studio for MySQL -用于MySQL和MariaDB数据库开发，管理和管理的通用IDE.
    • dbForge Studio for Oracle -强大的IDE，用于Oracle管理，管理和开发.
    • dbForge Studio for PostgreSQL -用于管理和开发数据库和对象的GUI工具.
    • dbForge Studio for SQL Server -强大的集成开发环境，用于SQL Server开发，管理，管理，数据分析和报告.
    • dbKoda -现代的（JavaScript / Electron框架），适用于MongoDB的开源IDE. 它具有支持MongoDB数据库上的开发，管理和性能调整的功能.
    • IBExpert -适用于Firebird和InterBase的综合GUI工具.
    • HeidiSQL -用Delphi编写的用于管理MySQL，MSSQL和PostgreSQL的轻量级客户端.
    • MySQL Workbench -MySQL Workbench是面向数据库架构师，开发人员和DBA的统一可视工具.
    • Navicat -一种数据库开发工具，可让您从单个应用程序同时连接到MySQL，MariaDB，SQL Server，Oracle，PostgreSQL和SQLite数据库.
    • Oracle SQL Developer -Oracle SQL Developer是一个免费的集成开发环境，可简化传统和云部署中Oracle数据库的开发和管理.
    • pgAdmin -PostgreSQL（世界上最先进的开源数据库）最受欢迎和功能最丰富的开源管理和开发平台.
    • pgAdmin3 -pgAdmin3的长期支持.
    • PL/SQL Developer -专门针对Oracle数据库存储程序单元开发的IDE.
    • PostgreSQL Maestro -PostgreSQL的功能强大的完整数据库管理，管理和开发工具.
    • Toad -Toad是面向开发人员，管理员和数据分析师的首要数据库解决方案. 使用单个数据库管理工具管理复杂的数据库更改.
    • Toad Edge -简化的MySQL和Postgres数据库开发工具.
    • TOra -TOra是针对Oracle，MySQL和PostgreSQL数据库的开源SQL IDE.
    • Valentina Studio -免费创建，管理，查询和浏览Valentina DB，MySQL，MariaDB，PostgreSQL和SQLite数据库.

    GUI Managers/Clients¶

    • Adminer -在单个PHP文件中进行数据库管理.
    • DbVisualizer -适用于开发人员，DBA和分析师的通用数据库工具.
    • HouseOps -用于您的企业ClickHouse Ops UI，可以运行查询，监视ClickHouse的运行状况并让其他人思考.
    • JackDB -直接SQL访问您所有数据，无论它们位于何处.
    • OmniDB -用于数据库管理的Web工具.
    • Pgweb -用于PostgreSQL的基于Web的数据库浏览器，用Go编写，可在macOS，Linux和Windows计算机上使用.
    • phpLiteAdmin -用PHP编写的基于Web的SQLite数据库管理工具，支持SQLite3和SQLite2.
    • phpMyAdmin -MySQL和MariaDB的Web界面.
    • psequel -PSequel提供了一个简洁的界面，可让您快速执行常见的PostgreSQL任务.
    • PopSQL -适用于您团队的现代化协作SQL编辑器.
    • Postico -Mac的现代PostgreSQL客户端.
    • Robo 3T -Robo 3T（以前称为Robomongo）是一个以外壳为中心的跨平台MongoDB管理工具.
    • Sequel Pro -Sequel Pro是一个快速，易于使用的Mac数据库管理应用程序，用于处理MySQL和MariaDB数据库.
    • SQL Operations Studio -一种数据管理工具，可从Windows，macOS和Linux使用SQL Server，Azure SQL DB和SQL DW.
    • SQLite Expert -图形界面支持所有SQLite功能.
    • sqlpad -基于Web的SQL编辑器在您自己的私有云中运行.
    • SQLPro -适用于macOS的简单，强大的Postgres管理器.
    • SQuirreL -用Java编写的图形化SQL客户端，它允许您查看JDBC兼容数据库的结构，浏览表中的数据，发出SQL命令等.
    • SQLTools -VSCode的数据库管理.
    • SQLyog -最完整，最易用的MySQL GUI.
    • Tabix -用于Clickhouse的SQL编辑器和开源简单业务智能.
    • TablePlus -用于关系数据库的现代，本地且友好的GUI工具：MySQL，PostgreSQL，SQLite等.
    • TeamPostgreSQL -PostgreSQL Web管理GUI-通过丰富，快速的AJAX Web界面，可在任何地方使用PostgreSQL数据库.

    CLI tools¶

    • ipython-sql -连接到数据库以在IPython或IPython Notebook中发出SQL命令.
    • iredis -具有自动完成功能和语法突出显示功能的Redis Cli.
    • pgcenter -PostgreSQL的顶级管理工具.
    • pg_activity -用于PostgreSQL服务器活动监视的顶级应用程序.
    • pg_top -PostgreSQL的“ top”.
    • pspg -Postgres传呼机.
    • SQLcl -Oracle SQL Developer命令行（SQLcl）是Oracle数据库的免费命令行界面.
    • usql -PostgreSQL，MySQL，Oracle数据库，SQLite3，Microsoft SQL Server的通用命令行界面， and many other databases 包括NoSQL和非关系数据库！

    dbcli¶

    • athenacli -AthenaCLI是用于AWS Athena服务的CLI工具，可以执行自动完成和语法突出显示.
    • litecli -具有自动完成功能和语法突出显示功能的SQLite数据库CLI.
    • mssql-cli - A command-line client for SQL Server with auto-completion and syntax highlighting.
    • mycli -具有自动完成功能和语法突出显示功能的MySQL终端客户端.
    • pgcli -具有自动完成功能和语法突出显示功能的Postgres CLI.
    • vcli -具有自动完成功能和语法突出显示功能的Vertica CLI.

    DB-schema navigation and visualization¶

    • dbdiagram.io -快速简单的工具，可帮助您绘制数据库关系图并使用简单的DSL语言快速进行处理.
    • ERAlchemy -实体关系图生成工具.
    • SchemaCrawler -免费的数据库架构发现和理解工具.
    • Schema Spy -将数据库生成为HTML文档，包括实体关系图.
    • tbls -使用Go编写的用于记录数据库文档的CI友好工具.

    Modelers¶

    • Navicat Data Modeler -功能强大且具有成本效益的数据库设计工具，可帮助您构建高质量的概念，逻辑和物理数据模型.
    • Oracle SQL Developer Data Modeler -Oracle SQL Developer数据建模器是一个免费的图形工具，可提高生产率并简化数据建模任务.
    • pgmodeler -专为PostgreSQL设计的数据建模工具.

    Migration tools¶

    • 2bass -利用幂等DDL脚本的概念的代码配置数据库工具.
    • flyway -数据库迁移工具.
    • gh-ost -MySQL的在线模式迁移.
    • liquibase -独立于数据库的库，用于跟踪，管理和应用数据库架构更改.
    • migra -与diff类似，但适用于PostgreSQL模式.
    • node-pg-migrate -专为postgres构建的Node.js数据库迁移管理. （但也可以用于其他符合SQL标准的DB，例如CockroachDB.）
    • Pyrseas -提供实用程序将PostgreSQL数据库模式描述为YAML.
    • SchemaHero -Kubernetes运算符，用于声明式数据库架构管理（数据库架构的gitop）.
    • Sqitch -敏感的数据库本地变更管理，用于无框架开发和可靠部署.
    • yuniql -另一个仅使用本机.NET Core 3.0+制作并有望更好的架构版本控制和迁移工具.

    Code generation tools¶

    • ddl-generator -从表数据推断SQL DDL（数据定义语言）.
    • scheme2ddl -用于将Oracle模式导出到ddl初始化脚本集的命令行util，具有过滤不良信息，在不同文件中分离DDL，输出漂亮格式的能力.

    Wrappers¶

    • DreamFactory -用于移动，Web和IoT应用程序的开源REST API后端.
    • Hasura GraphQL Engine -具有细粒度访问控制的Postgres上快速，即时的实时GraphQL API刷新数据库事件.
    • jl-sql -JSON和CSV流的SQL.
    • mysql_fdw -MySQL的PostgreSQL外部数据包装器.
    • Oracle REST Data Services -ORDS是一个中间层Java应用程序，它将HTTP（S）动词（GET，POST，PUT，DELETE等）映射到数据库事务，并返回使用JSON格式化的任何结果.
    • Prisma -Prisma将您的数据库转换为实时GraphQL API.
    • PostgREST -适用于任何Postgres数据库的REST API.
    • prest -一种从Go编写的任何数据库中提供RESTful API的方法.
    • restSQL -具有Java和HTTP API的SQL生成器，使用具有XML或JSON序列化的简单RESTful HTTP API.
    • resquel -轻松地将您的SQL数据库转换为REST API.
    • sandman2 -自动为您的旧数据库生成RESTful API服务.
    • sql-boot -用于SQL查询的高级REST和UI包装器.

    Backup tools¶

    • pgbackrest -可靠的PostgreSQL备份和还原.
    • BaRMan -PostgreSQL的备份和恢复管理器.

    Replication/Data operation¶

    • Datasette -用于探索和发布数据的工具.
    • dtle -适用于MySQL的分布式数据传输服务.
    • pgsync -在数据库之间同步Postgres数据.
    • pg_chameleon -用Python 3编写的MySQL到PostgreSQL副本系统.系统使用mysql-replication库从MySQL中提取行图像，这些图像以JSONB的形式存储在PostgreSQL中.
    • PGDeltaStream -使用Postgres逻辑解码功能，至少一次通过Websocket流式传输Postgres更改的Golang Web服务器.
    • repmgr -PostgreSQL最流行的复制管理器.

    Scripts¶

    • pgx_scripts -我们的PostgreSQL专家团队创建了一些有用的小脚本，用于数据库分析和管理.
    • pgsql-bloat-estimation -确保PostgreSQL索引和表中统计膨胀的查询.
    • pgWikiDont -SQL测试，检查您的数据库是否遵循以下规则 https://wiki.postgresql.org/wiki/Don't_Do_This .
    • pg-utils -有用的PostgreSQL实用程序.
    • Postgres cheat sheet -有用的SQL脚本和命令 .
    • postgres_dba -缺少适用于Postgres DBA和所有工程师的有用工具集.
    • postgres_queries_and_commands.sql -有用的PostgreSQL查询和命令.
    • TPT -这些sqlplus脚本用于Oracle数据库性能优化和故障排除.

    Monitoring/Statistics/Perfomance¶

    • ASH Viewer -提供Oracle和PostgreSQL DB中活动会话历史记录数据的图形视图.
    • Monyog -无代理且经济高效的MySQL监控工具.
    • mssql-monitoring -使用收集的InfluxDB和Grafana监视Linux上的SQL Server性能.
    • Navicat Monitor -安全，简单，无代理的远程服务器监视工具，其中包含强大的功能，可以使您的监视尽可能有效.
    • Percona Monitoring and Management -用于管理和监视MySQL和MongoDB性能的开源平台.
    • pganalyze collector -Pganalyze统计信息收集器，用于收集PostgreSQL指标和日志数据.
    • postgres-checkup -新一代诊断工具，可让用户对Postgres数据库的运行状况进行深入分析.
    • postgres_exporter -用于PostgreSQL服务器指标的Prometheus导出器.
    • pgDash -测量和跟踪PostgreSQL数据库的各个方面.
    • PgHero -Postgres的性能仪表板-运行状况检查，建议的索引等.
    • pgmetrics -从运行中的PostgreSQL服务器收集并显示信息和统计信息.
    • pgMustard -Postgres的用户界面说明了计划，以及提高性能的提示.
