    jQuery

    Awesome jQuery¶

    精选的超赞清单 jQuery 插件，资源和其他闪亮的东西. 受GitHub上awesome- *趋势的启发.

    jQuery 快速，小型且功能丰富 JavaScript 图书馆. 通过易于使用的API（可在多种浏览器中使用），它使HTML文档的遍历和操作，事件处理，动画和Ajax等事情变得更加简单. 兼具多功能性和可扩展性，jQuery改变了数百万人编写JavaScript的方式.

    Resources¶

    很棒的学习材料，可以了解jQuery和JavaScript或学习更高级的内容. 一些友好的建议：在深入研究jQuery之前，请先学习JavaScript.

    • API documentation -官方jQuery文档.
    • Code School -了解jQuery（JavaScript库）的基础知识，以修改网页上的元素并改善用户交互.
    • Codecademy jQuery track -学习jQuery的初学者轨道.
    • Learning jQuery -jQuery JavaScript库的提示，技巧和教程.
    • Style guides -贡献给jQuery项目时jQuery的代码样式指南.
    • Tuts+ -关于jQuery的有用且深入的文章.
    • jQuery guide for teams -编写一致的jQuery代码的技巧，易于团队使用和理解.

    Tutorials¶

    很棒的免费教程和课程来学习jQuery.

    • jQuery patterns -多种jQuery插件模式，可用于快速启动插件开发.
    • Learning jQuery Design Patterns -jQuery插件设计模式
    • Learn jQuery -jQuery学习中心.
    • Snoopcode -jQuery教程.
    • Extending jQuery -扩展jQuery的选择器功能.
    • Free jQuery Courses at Classpert -Classpert在线课程搜索中的免费jQuery课程列表

    • tipso -轻量级响应式jQuery工具提示插件.

    *用于处理，操纵或格式化时间和日期的插件.

    • h5Validate -jQuery的HTML5表单验证插件.
    • jQuery Form Validator -验证用户输入，同时保持HTML标记与JavaScript代码保持一致.
    • jquery-validation -对现有表单进行直接验证，同时进行各种自定义以适合您的应用程序非常容易.
    • jQuery.validationEngine -在浏览器中验证表单字段.
    • jQuery-form-validation -使用Laravel语法验证HTML表单.
