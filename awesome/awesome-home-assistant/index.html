    Home Assistant

    Awesome Home Assistant ¶


    https://awesome-ha.com

    家庭助理是一个开源的家庭自动化系统，可将本地控制和 隐私至上. 由全球修补匠和DIY社区提供支持 爱好者. 非常适合在Raspberry Pi或本地服务器上运行.

    如果您想在外观上留下印象， 你应该看看 Home Assistant online demo.

    很棒的家庭助理是精选的很棒的清单 Home Assistant 资源. 其他软件，教程，自定义集成，附加组件， 自定义Lovelace卡和插件，食谱，示例设置等.

    该列表分为几类. 这些类别中的链接没有 预先建立的订单； 该命令是供款. 如果你想贡献， 请阅读 guide.

    How to use¶

    很棒的家庭助理对于那些试图使每个人自动化的人来说是一个很棒的清单 他们家的一面. 使您的房屋自动化是一个漫长而艰辛的过程，并且从未完成 task that usually involves a lot of tinkering.

    您可以通过以下方式浏览列表：

    -只需按Command / Ctrl + F来搜索关键字 -或者，使用我们网站上的搜索：https://www.awesome-ha.com

    Installing¶

    家庭助理有几种安装/运行方法. 很多人有 不同的意见和个人喜好. 每种方法都有其自己 的优点和缺点. 重要的是要知道，没有错，或者就在这里， 每种技术都会安装 SAME Home Assistant.

    Home Assistant当前正在_推荐Home Assistant操作系统的安装方法.

    • Home Assistant OS -使用托管环境进行安装（推荐方法）.
    • Home Assistant Container -在Docker上安装.
    • Home Assistant Supervised -为专家安装半托管环境.
    • Home Assistant Core -使用Python虚拟环境进行手动安装.

    In case you need help¶

    _有多种方法可以与家庭助理社区联系. 无论您有任何疑问，需要帮助，想请求功能， 或只是说“嗨”.

    Official Channels¶

    • Home Assistant Discord -加入聊天，我们大多数人都在那里.
    • Home Assistant Community -论坛，也用于功能请求.
    • Home Assistant Subreddit -如果您喜欢Reddit，请订阅.
    • Home Assistant Facebook Group -面向发烧友的Facebook组.

    Other Channels¶

    • Dr. ZZs -Zzs博士的Facebook小组.
    • Home Assistant Community Add-ons Discord -获得有关家庭助理社区附加组件的支持.

    • Dutch Domotics Discord -具有家庭自动化爱好者的荷兰Discord服务器.

    Add-ons¶

    附加组件是可以与其他应用程序和服务一起运行 家庭助理. Home Assistant操作系统和受监督的安装类型， 提供主管，该主管能够运行和管理这些加载项.

    Official Add-ons¶

    _由家庭助理团队创建和维护.

    • DuckDNS -使用“加密”更新您的Duck DNS IP地址并生成SSL.
    • File editor -基于浏览器的配置文件编辑器.
    • Mosquitto - Fast and reliable MQTT broker.
    • Terminal & SSH -允许使用Web终端或SSH客户端远程登录.
    • Samba -使用Windows网络共享访问您的配置文件.
    • NGINX SSL proxy -具有SSL终止的反向代理.
    • deCONZ -使用Dresden Elektronik的ConBee或RaspBee硬件控制ZigBee网络.
    • TellStick -运行TellStick和TellStick Duo服务.
    • Ada -Ada是由Almond提供支持的语音助手，该助手是开放的并且可以保护隐私.
    • Almond -开放的，保留隐私的虚拟助手.
    • HomeMatic -基于OCCU的HomeMatic中央.
    • Let's Encrypt -从Let's Encrypt获得免费的SSL证书； 开放和自动的证书颁发机构（CA）.
    • MariaDB -开源关系数据库（MySQL的分支）.
    • OpenZWave -在QT OpenZWave守护程序中使用Z-Wave USB记忆棒.
