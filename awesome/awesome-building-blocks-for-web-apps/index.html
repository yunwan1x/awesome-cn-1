    Awesome building blocks for web apps

    Awesome Building Blocks for Web Apps ¶

    要集成到Web应用程序中的独立功能的累积列表：获得其他人构建的完整功能，而不必自己编写所有内容.

    如果您喜欢这些积木，请查看 componently.com，开发人员可以在其中搜索，比较和集成Web应用程序构建基块的市场.

    UI Elements¶

    • Tailwind UI -UI组件，由Tailwind CSS的创建者提供.
    • Webix -JavaScript UI库和框架，用于以不同方式显示数据.
    • Highcharts -网络互动图表.
    • Webatoms -Xamarin.Forms的JavaScript桥.
    • Creative Tim -React，Angular，Vue，Lavarel和HTML5的主题.
    • Syncfusion -数据可视化组件和PDF Viewer.
    • CoreUI -查看管理模板.
    • UIfort -适用于所有主要前端框架的UI设计系统.
    • Semantic UI - Open source UI component library for React, Meteor, Ember and Angular.
    • Design Revision's Shards Pro -用于Web或移动应用程序的设计块.
    • Themesberg -引导程序模板，主题和UI套件.
    • Treact -基于Tailwind CSS的React登陆页面模板.
    • Creative Designs Guru -使用Tailwind CSS构建的React，Nextjs和Eleventy主题.

    Exit Intent Popup¶

    • Wisepops -移动和桌面访问者的上下文弹出窗口，横幅和栏.
    • Listagram -用于收集电子邮件地址的交互式选择加入表格.
    • Poptin -可以在网站中任何位置集成的弹出窗口和表格.

    Auth¶

    • Auth0 -具有对Angular，React和VueJS的本机支持的身份验证管理.
    • Firebase -来自Google的PaaS，包括身份验证管理.
    • Okta -支持Angular，React和VueJS的身份验证和授权解决方案.
    • Onelogin -身份验证侧重于公司内部应用程序.
    • Microsoft Azure Active Directory -Azure中基于Microsoft帐户的身份验证管理.
    • AWS Cognito -AWS身份验证管理. 前端集成是其中的一部分 AWS Amplify.

    Booking¶

    • Calendly -单机预订解决方案.
    • Zoho Bookings -在线调度程序已集成到Zoho Suite中.
    • Acuity Scheduling -带有广泛API的约会安排.
    • 10to8 -预订工具，具有免费套餐中的许多功能.
    • SuperSaaS -预约安排，重点放在小组安排上.
    • SimplyBook.Me -具有自己的预订目录的预订工具.
    • OnceHub -通过ScheduleOnce进行计划，并通过InviteOnce进行会议管理.
    • YouCanBookMe -团队在线排程.
    • OnSched -针对市场和应用程序的调度API.
    • AppointmentPlus -附有优惠券的预约时间表.
    • Booxi -预订和在线目录组件.
    • Cogsworth Scheduler -具有强大API的企业在线计划程序.

    Notification¶

    • OneSignal -手机和网络通知.
    • Notific -包括PHP SDK的通知.
    • WonderPush -通知集中在速度上.
    • VWO Engage -通知是大型营销套件的一部分.
    • Signalize -添加带有付费分析功能的免费通知.

    Social¶

    • Stream -活动供稿和聊天API.
    • EmbedSocial's embedfeed -社交媒体聚合器.

    Chat Bot¶

    • Fresh chat -Freshworks支持套件中的Chatbot.
    • Tiledesh -基于AI的实时聊天.

    Chat¶

    • Daily.co -视频聊天.

    Monitoring¶

    • Sentry -包括发布管理在内的全栈错误监控.
    • Raygun Crash Reporting -作为Raygun套件的一部分进行错误监视.
    • Honeybadger -错误，正常运行时间和签到监控.

    • Rollbar -错误监视，具有重播导致错误的HTTP请求的功能.

    • Contentful -无头内容管理系统.

    Consent Management¶

    • Cookiefirst -Cookie同意管理员.
    • usercentrics -可以作为现有页面的一部分实施的法规遵从性管理器.
    • Sourcepoint -合规性管理作为套件的一部分，还包括广告拦截程序恢复.
    • PiwikPro -合规性管理作为分析工具的一部分.
    • Consent Management Provider -Cookie同意的德国管理工具.
    • UniConsent -符合GDPR，CCPA，COPPA要求的同意的管理平台.
    • TrustArc -CCPA，GDPR，LGPD和900多个其他全球法规的隐私管理.
    • Cookiebot -Cookie跟踪管理器.
    • iubenda -同意管理和条款和条件生成器.
    • Civic Cookie Control -较小的Cookie和GDPR合规产品.
    • CCM19 -德国Cookie同意管理员.
    • CookieInformation -通过仪表板专注于合规报告.
    • CookiePro -Cookie管理作为隐私管理解决方案的一部分.
    • React Cookie Consent -适用于React应用程序的小型可自定义cookie同意栏.
    • React Cookie Banner -React cookie横幅，可通过滚动自动关闭.

    Media¶

    • Mux -流托管和实时视频.
    • Spotify web playback SDK -客户端JavaScript库，可让您在Spotify Connect中创建新的播放器，并通过加密的媒体扩展名在浏览器中播放Spotify中的任何音轨.

    Form¶

    • Static Forms -将结果发送到您的电子邮件的表格.
    • FlyingForms -构建，版本控制和管理表单.

    Marketing¶

    • TrustPulse -在您的网站上显示最近的买家.

    Changelog¶

    • Beamer -推送通知和变更日志管理的结合.
    • Changelogfy -产品发行说明工具.
    • Changefeed -Changelog作为独立页面或可嵌入的小部件.
    • Canny's Changelog -将变更日志与产品路线图结合在一起.
    • ProductFlare's Changelog -产品团队任务管理系统周围的Changelog小部件.
    • Changes -Changelog构建套件.

    Telecommunication¶

    • Vonage -跨统一通信，联络中心和通信API的语音，消息，视频和数据功能.

    Social Media¶

    • AddThis -社交共享按钮，在AMP上可用.
    • ShareThis -社交分享按钮.
    • AddToAny -用于许多不同目标服务的共享按钮.

    Search¶

    • Algolia -搜索您的应用程序.
    • Search UI -ElastiSearch的UI集成.

    Feature flags¶

    • Optimizely's Rollouts -与A / B测试结合的功能标志.
    • LaunchDarkly -在CI服务器和票务系统中具有多个集成的功能管理.
    • Split -在CI服务器和票务系统中具有多种集成的另一项功能管理.
    • Rollout -功能管理侧重于分段推出功能.

    Reviews¶

    • JustReview -收集来自电子商务，社交媒体和评论服务提供商的评论，并将其嵌入网站中.
    • ReviewsOnMyWebsite -收集，监控和显示客户评论.
