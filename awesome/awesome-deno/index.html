    Awesome deno

    Awesome Deno ¶

    Deno是使用V8并内置于Rust的JavaScript和TypeScript的简单，现代且安全的运行时.

    该列表是最佳Deno模块和资源的集合.

    Docs¶

    Official Docs¶

    • Official Site
    • Deno API TypeDoc
    • Deno Manual
    • Deno Registry

    External Docs¶

    • V8 Docs for Deno

    Online Playgrounds¶

    • deno.town
    • Deno Playground
    • maman/deno-playground

    Modules¶

    NOTICE：Deno有一些官方模块，可以在以下位置找到 deno_std. 考虑提交给 the deno.land/x 资料库.

    • abc -更好的Deno框架来创建Web应用程序.
    • alosaur -Alosaur-具有许多装饰器的Deno Web框架.
    • aqua -用于Deno的最小和快速的Web框架.
    • autopilot -自动驾驶-使用Deno进行跨平台的Web自动化.
    • bytes_formater -格式化字节（Uint8Array，ArrayBufferView等）输出，在调试IO功能时很有用.
    • cac -用于构建命令行应用程序的简单而强大的框架.
    • camelcase -将破折号/点号/下划线/空格分隔的字符串转换为camelCase：foo-bar→fooBar.
    • colors -用于Deno的基本控制台颜色库.
    • cli-spinner -在执行长任务时，在终端中显示微调框.
    • csv -一个简单的CSV解析器.
    • dcc -Deno Cache Clean，在下次运行时重新加载dep.
    • dejs -用于Edeno的Ejs模板引擎.
    • denon -具有等待生成器的文件监视程序.
    • deno_case_style -用于不同大小写样式的字符串验证器和格式化程序. 例如：camelCase等.
    • deno-checksum -SHA1 / MD5算法.
    • deno-context -将期限，取消和其他要求范围的值传播给多个承诺. 行为就像Go的上下文.
    • deno_cron -cron Job调度程序，使您可以编写具有大量灵活性的可读cron语法
    • deno-deamon -使Deno程序在后台运行. -deno-dotenv
    • pietvanzoen/deno-dotenv -Dotenv行动为deno.
    • cardosomarcos/deno-dotenv -从.env加载Deno项目的环境变量.
    • deno-express -将Node Express移植到Deno.
    • deno-fnparse -一个非常简单的JavaScript解析器组合器.
    • deno-globrex -globrex的端口为deno，globex的端口为正则表达式.
    • deno-mysql -MySQL数据库驱动程序.
    • deno_mongo -MongoDB数据库驱动程序.
    • deno-opn -打开网站，文件，可执行文件之类的东西. 跨平台.
    • deno-plugin-prepare -用于管理Deno本机插件依赖项的库.
    • deno-pretty-assert -色彩鲜艳的assertE等于deno.
    • deno-prettystring -格式化，修剪和删除字符串中字符之间的多余空格.
    • deno_random_interval -帮助程序生成随机间隔.
    • deno-redis -Redis客户端的Deno的实验实现.
    • deno-slugify -deno的字符串节流器.
    • deno-smtp -SMTP的SMTP邮件发件人.
    • deno_tiny_templates -deno的模板渲染器.
    • deno_tokenizer -deno的简单标记器.
    • deno-using -带有deno语句的python样式.
    • deno-uuid -deno的UUID模块.
    • deno-ws -用于deno.ts的websocket服务器的实验性实现.
    • deno-websocket -一个简单的WebSocket库，例如node.js库的ws.
    • deno-xml-parser -从segmentio / xml-parser移植的Deno XML解析器.
    • denotrain -带有中间件支持的多合一Web框架，如Express或Fastify forNode.js.
    • dinatra -Sinatra像deno的轻量级Web应用程序框架.
    • djwt - Make JSON Web Tokens (JWT) on Deno based on JWT and JWS specifications.
    • dso -一个基于mysql的简单ORM库.
    • ensure -确保您运行的是Deno，Typescript或V8的最低版本.
    • evt -输入EventEmitter的安全替代品.
    • expect -编写笑话的助手，如deno中的期望测试.
    • flags -基于极简主义的Deno命令行参数解析器.
    • gardens -一个无处不在的JavaScript记录实用程序.
    • gentleRpc -用于Deno和浏览器的JSON-RPC 2.0 TypeScript库.
    • http -包括文件服务器的HTTP模块.
    • http-libs -HTTP模块和类型.
    • invert-kv -在Deno中反转键值对.
    • jwt -auth0 / jsonwebtoken的端口.
    • lazy -类似于linq的惰性求值迭代模块.
    • log -Deno的记录模块.
    • marked -Markdown到HTML转换器.
    • ms -轻松地将各种时间格式转换为毫秒.
    • normalize_diacritics -删除字符串中的重音符号/变音符号.
    • oak -用于Deno网络服务器的中间件框架.
    • online -检查您当前是否在Deno在线.
    • opine -从ExpressJS移植的快速，简约的Web框架.
    • path -Deno Path操作库.
    • pogo -Deno的服务器框架.
    • postgres -PostgreSQL数据库驱动程序.
    • qrcode -用于Deno的QR码图像生成器.
    • qs -具有嵌套支持的查询字符串解析器.
    • recursive-readdir -在Deno中递归读取目录.
    • router -高性能基本路由器可在任何地方使用.
    • rubico -[a]同步功能组成； 它只是工作.
    • sax-ts -从移植的SAX样式的XML解析器 sax-js.
    • servest -渐进式HTTP服务器/路由器.
    • sql-builder -sql查询生成器.
    • status -用于Deno的HTTP代码和状态实用程序.
    • textproto
    • type-fest -基本TypeScript类型的集合（sindresorhus端口/ type-fest）.
    • unexpected -可扩展的BDD断言工具包.
    • up -检查是否在Deno建立了网站.
    • wasm-gzip -为Deno加密和解密gzip.
    • watch -文件查看器.
    • websocket_server -WebSocket服务器库.
    • webview -Webview的Deno绑定，这是一个用于创建基于Web的桌面GUI的小型库.
    • wu-diff-js -差异库，使用wu（O（NP））算法计算两个切片之间的差异.
    • youtube-deno -YouTube数据API的Deno客户端库，用于与YouTube进行任何交互.

    Tools¶

    • clone -用于方便克隆的简单实用程序.
    • dedep -管理依赖版本.
    • denoget -Denoget安装可执行的Deno脚本.
    • denoify -对于希望支持Deno但不想编写和维护端口的NPM模块作者.
    • denoinit -Denoinit为Deno项目生成有用的文件.
    • denoliver -具有实时重新加载功能的简单，无依赖的文件服务器.
    • denomander -Deno命令行界面灵感来自commander.js.
    • denon -守护程序脚本运行程序，例如nodemon. 内置于Deno.
    • denopkg -在您的Deno项目中使用GitHub中代码的简便方法.
    • denoversion -Deno的SemVer和git版本管理.
    • denox -与packages.json脚本类似，但对于具有权限支持的Deno.
    • deno.mk -用于安装和运行Deno的跨平台Makefile. -maxmcd deno-docker 泊坞窗映像. -海德 deno-docker 几个docker映像.
    • deno-vscode -使用此扩展程序利用vscode中内置的typedef和intellisense.
    • dev_server -让TypeScript文件直接在script标签中使用.
    • dpm -Deno软件包管理器，为deno安装全局命令. 像denoget. -DVM
    • justjavac/dvm -Deno版本管理器：管理多个活动的Deno版本.
    • axetroy/dvm -没有运行时相关性的Deno版本管理器.
    • elm-live-reload -用Deno编写的Elm Live Reloader.
    • nessie -为PostgreSQL，MySQL和SQLite创建，迁移和回滚迁移.
    • packer-provisioner-deno -一个Packer插件，可轻松使用Deno脚本构建虚拟机映像.
    • pagic -使用Mark Deno构建的最简单的从markdown生成静态html页面的方法.
    • pika Deno plugin
    • starter -带有GitHub Actions CI的Deno模块入门模板
    • task-runner -像npm脚本一样编写任务.
    • typescript-deno-plugin -Deno语言服务插件，在编辑器内的TypeScript文件中提供智能感知.
    • udd -更新Deno依赖性：将导入语句更新为最新发布的版本.
    • velociraptor -Deno的npm风格脚本执行器.
    • vscode-deno -VS Code扩展，使用typescript-deno-plugin提供Deno支持.
