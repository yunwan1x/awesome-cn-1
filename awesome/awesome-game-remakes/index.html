    Awesome game remakes

    Awesome Game Remakes ¶

    积极维护的开源游戏翻拍.

    如果您也对克隆和类似游戏（0 AD等）感兴趣，请看一下 Open Source Game Clones.

    Action¶

    • Carnage3D -重新实现侠盗猎车手.
    • donut -《辛普森一家》的重新实现：在现代C ++和现代OpenGL中运行并运行.
    • MafiaUnity -开源黑手党游戏框架.
    • OpenJK -社区维护和改进Raven Software的Jedi学院和Jedi Outcast的努力.
    • OpenLara -经典古墓丽影开源引擎.
    • OpenMB -Taleworlds的Mount＆Blade系列的角色扮演游戏引擎.
    • OpenTomb -具有现代更新，功能和增添功能的原始古墓丽影1-5体验.
    • re3 -GTA III和GTA的反向源代码：Vice City.
    • remc2 -来自Assembler的Magic Carpet 2的逆向工程.
    • Rigel Engine -用C ++重新实现经典的DOS游戏Duke Nukem II.
    • San Andreas Unity -重新实现GTA：Unity中的San Andreas游戏引擎.
    • TOMB5 -古墓丽影：编年史反汇编成C源代码.

    Adventure¶

    • engge -Thimbleweed Park的实验性游戏引擎.
    • raw(gl) -重新实现另一个世界.
    • ResidualVM -重新实现游戏引擎，使您可以玩3D冒险游戏，例如Grim Fandango和Monkey Island中的Escape.
    • ScummVM -允许您运行某些经典的图形化点击式冒险游戏，例如Maniac Mansion和Monkey Island.

    Arcade¶

    • Barbarian -野蛮人：终极战士的开源克隆.
    • C-Dogs SDL -经典的头顶枪战游戏.
    • Mr.Boom -RetroArch / Libretro的8名Bomberman克隆人.
    • OpenClonk -多人动作游戏，您可以控制小而灵活的类人动物.
    • Open Kick-Off -改写邪教足球比赛开赛2的尝试.
    • OpenTyrian -Tyrian的开源端口，从Turbo Pascal到C.
    • Taisei -东方系列的粉丝制作的开放源代码克隆.
    • The Great Escape in C -便携式C中的经典ZX Spectrum游戏《大逃亡》.
    • Zelda Classic -一个游戏和编辑工具，可让您创建自定义的《塞尔达传说》任务.

    FPS¶

    • Allegiance -效忠是RTS和FPS太空战斗游戏.
    • BStone -布雷克·斯通（Blake Stone）：《黄金异形》和布雷克·斯通（Blake Stone：Planet Strike）的原始港口.
    • CatacombGL -Catacomb 3D和Catacomb Adventure系列的Windows源端口.
    • Chocolate Doom -源端口是极简主义的，在历史上是准确的，包括异端和赫克森的端口以及Strife.
    • Doom Legacy -最受欢迎的DOOM源端口之一，可通过TCP / IP网络，OpenGL渲染器等功能对其进行增强.
    • DoomRetro -Windows PC的经典精致DOOM源端口.
    • Doomsday Engine -id Software的DOOM I / II和Raven Software的Heretic和Hexen的源端口.
    • DukeGDX -Duke Nukem 3D端口.
    • ECWolf -基于Wolf4SDL的Wolfenstein 3D端口.
    • ezQuake -结合了所有现代QuakeWorld客户的功能，ezQuake使QuakeWorld的启动和播放变得更加容易.
    • Freedoom -基于经典DOOM引擎的免费软件FPS.
    • GZDoom -基于DOOM引擎的对Modder友好的OpenGL和Vulkan源端口.
    • Hexen II: Hammer of Thyrion -Hexen II游戏的跨平台端口.
    • ioquake3 -可用于玩Quake 3或制作自己的游戏的免费软件FPS引擎.
    • NBlood -经典FPS鲜血之港.
    • Odamex -在线多人DOOM端口，着重于原始游戏玩法，同时提供了广泛的增强功能.
    • OpenSpades -体素FPS黑桃王牌的兼容客户端.
    • QuakeSpasm -现代化的跨平台Quake游戏引擎.
    • RBDOOM-3-BFG -Doom 3 BFG Edition具有柔和的阴影，清理的源代码，Linux和64位支持.
    • RedneckGDX -乡下人狂暴的港口.
    • The Eternity Engine -高级DOOM源端口.
    • The Force Engine -适用于现代系统的Jedi Engine以及使用该引擎的游戏-Dark Forces and Outlaws.
    • Tyrquake -特里克（Tyrquake）的利伯特罗（Libretro）港口.
    • Yamagi Quake II -id Software Quake II的增强客户端，专注于脱机和合作游戏.
    • vkQuake -使用Vulkan而不是OpenGL进行渲染的Quake端口.

    Online¶

    • 2006rebotted -2006 Runescape仿真服务器.
    • FreeSO -重新实现《模拟人生在线》，主要针对移动设备.
    • Meridian 59 -第一个3D MMORPG，于1996年发布，于2012年开源.Meridian59的原始代码库.
    • Opengate -重制Jumpgate.
    • RuneLite -开源旧式RuneScape客户端.
    • Ryzom Core -Ryzom Core包含用于制作商业MMORPG Ryzom的全部代码（客户端，服务器，工具）.
    • Space Station 14 -空间站14是重制空间站13的尝试.

    RPG¶

    • Arx Libertatis -第一人称角色扮演游戏Arx Fatalis的跨平台端口.
    • Barony -Barony的开源版本.
    • BtBuilder -Bard's Tale Construction Set的开源实现.
    • Daggerfall Unity -在Unity引擎中创建Daggerfall.
    • Devilution -暗黑破坏神原始源代码的重构形式.
    • DevilutionX -为现代操作系统构建暗黑破坏神.
    • DGEngine -暗黑破坏神游戏引擎的实现.
    • Exult -为现代操作系统重新创建Ultima 7的项目.
    • Falltergeist -用C ++和SDL编写的Crossplatform Fallout 2游戏引擎.
    • Freeablo -Diablo引擎的开源实现.
    • FreeFT -受辐射策略启发的实时等距动作游戏引擎.
    • GemRB -无限引擎的便携式开源重新实现（由Baldur's Gate，Icewind Dale等使用）.
    • OpenDiablo2 -暗黑破坏神2的开源重新实现.
    • OpenGothic -重新实现哥特式2：乌鸦之夜.
    • OpenMW -重新实现了晨风游戏引擎.
    • OpenTESArena -Bethesda Softworks对《上古卷轴：竞技场》的现代引擎重新实现.
    • OpenVIII - Final Fantasy VIII engine implementation in C#.
    • REGoth -哥特式和哥特式II背后的引擎zEngine的开源重新实现.
    • Riiablo -使用Java和LibGDX重新制作了《暗黑破坏神II》.
    • sfall -经典游戏《辐射2（Fallout 2）》的一组引擎修改.
    • Shockolate -同样出色的系统冲击力，新的绝佳品味.
    • UAlbion -重拍1995年的Albion.
    • WitchavenGDX -Witchaven Java端口.
    • xoreos -BioWare的Aurora引擎及其派生工具的实现（Neverwinter Nights，The Witcher等使用）.
    • zorkClone -重新实现由Infocom开发的原始Zork游戏.

    Platformer¶

    • Alive Reversing -重新实现奇幻世界：安倍晋三的出埃及记.
    • Commander Genius -指挥官基恩游戏的现代译员.
    • Jazz² Resurrection -重新实现Jazz Jackrabbit 2.
    • Lionheart Remake -Lionheart Amiga游戏的Java重制.
    • NXEngine-evo -改写杰作跳跳跑平台游戏《 Doukutsu Monogatari》（也称为《洞穴物语》）.
    • Omnispeak -在再见星系中重新执行基恩指挥官的任务.
    • Prince-Monogame -波斯王子2的Monogame实现.
    • SDLPoP -基于DOS版本的拆卸，波斯王子港.
    • The Secret Chronicles of Dr. M -改编《秘密马里奥编年史》.
    • VVVVV -VVVVVV的源代码.

    Racing¶

    • DreeRally -死亡拉力赛引擎的重新实现.
    • OpenNFS -尝试重新创建经典的极品飞车游戏（1-6）.
    • Vangers -Racing / RPG混合动力由其创建者以C ++开源.

    Simulator¶

    • DXX-Rebirth -适用于Windows，macOS，Linux的Descent和Descent 2引擎的源端口.
    • FS2Open -FreeSpace2源代码项目.
    • Librelancer -重新实现了2003年的太空游戏Freelancer.

    Strategy¶

    • Antares -重制战神战神，这是一种战术太空战斗游戏.
    • CivOne -实施西德·迈尔（Sid Meier）的文明.
    • CorsixTH -主题医院的克隆，支持现代操作系统，高分辨率等.
    • Cortex Command Community Project Source -由社区推动的努力，以继续开发Cortex Command.
    • FreeCol -基于回合制策略游戏，基于旧版游戏《殖民》.
    • Jagged Alliance Stracciatella -使《铁血联盟2》可在多种平台上使用.
    • Julius -重新实现凯撒三世.
    • KaM Remake -从零开始用Delphi编写的骑士和商人的RTS游戏重制.
    • KeeperRL - Roguelike dungeon builder inspired by Dwarf Fortress.
    • openage -帝国时代II引擎克隆.
    • OpenApocalypse -重建X-COM：启示录引擎.
    • openblack -重新实现黑白.
    • OpenDUNE -流行游戏Dune II的娱乐.
    • Open Fodder -Cannon Fodder的开源实现.
    • OpenHoW -重新制作Gremlin的《战猪》.
    • OpenHV -硬真空的开源实现.
    • OpenLoco -重新实现克里斯·索耶（Chris Sawyer）的运动.
    • Open Panzer -HTML5 Panzer General 2.
    • OpenRA -使用.NET / mono和OpenGL实现Red Alert引擎. 在Windows，Linux和macOS上运行.
    • OpenRCT2 -过山车大亨的娱乐2.
    • OpenSAGE - C# reimplementation of SAGE, the game engine used by a bunch of EA Pacific games, including Command & Conquer: Generals and Battle for Middle Earth.
    • OpenSC2K -由Maxis重新制作SimCity 2000.
    • OpenTTD -OpenTTD是一款基于Chris Sawyer编写的流行游戏《运输大亨豪华版》的运输模拟游戏.
    • OpenXcom -不明飞行物的克隆：未知的敌人和X-COM：深渊的恐怖.
    • OutpostHD -重制Sierra On-Line的前哨基地.
    • Race into Space -Interplay的Buzz Aldrin的《太空竞赛》的版本.
    • Return To The Roots -用C ++编写的Settlers II重制版.
    • Seven Kingdoms: Ancient Adversaries -GPL发布的《七个王国：古代敌人》.
    • Sourcehold -由Firefly Studios实施要塞.
    • VCMI -魔法门英雄III引擎.
    • Warcraft Remake -《魔兽争霸：兽人与人类》的Java重制.
    • Wargus -《魔兽争霸2》模组，可让您使用Stratagus引擎玩《魔兽争霸2》.
    • Warzone 2100 -免费和开源的实时策略游戏.
    • Zed Online -1996年经典游戏Z的翻拍.
