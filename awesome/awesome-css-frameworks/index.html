    Awesome css frameworks

    Awesome CSS Frameworks ¶

    很棒的CSS框架列表.

    随意地 contribute.

    Base / Reset / Normalize¶

    • normalize.css -支持HTML5的CSS重置替代品.
      Repo | #CSS

    • sanitize.css -最佳实践CSS基础.
      Repo | #CSS

    • minireset.css -微小的现代CSS重置.
      Repo | #Sass

    • inuitcss -可扩展，可扩展，基于Sass的OOCSS框架，用于大型且持久的UI项目.
      Repo | #SCSS

    • ress -现代CSS重置.
      Repo | #CSS

    • modern-normalize - Normalize for Chrome, Firefox, and Safari only (no support for Edge or Internet Explorer).
      Repo | #CSS

    Class-less¶

    使用语义HTML且不依赖类的框架.

    • Water.css -只需添加CSS样式集，即可使简单的网站变得更好.
      Repo | #CSS

    • sakura -最小的无类CSS框架/主题.
      Demo, Repo | #SCSS

    • Marx -无类CSS重置.
      Docs, Repo | #SCSS #Stylus

    • Tacit -假人的CSS框架，不带类.
      Repo | #SCSS

    • awsm.css -CSS图书馆，用于语义HTML标记，没有类，属性等.
      Demo, Repo | #SCSS

    Very Lightweight¶

    小于〜5KB的框架.

    • Pure -可在每个Web项目中使用的小型响应式CSS模块集.
      Demo, Docs, Repo | #CSS

    • Milligram -极简CSS框架.
      Docs, Repo | #Sass

    • Picnic CSS -轻巧美观的图书馆.
      Demo, Docs, Repo | #SCSS

    • Wing -专为极简主义者设计的漂亮CSS框架.
      Docs, Repo | #Stylus

    • Chota -很小的CSS框架.
      Repo | #CSS

    General Purpose¶

    • Bootstrap -最受欢迎的HTML，CSS和JavaScript框架，用于在网络上开发响应式，移动优先项目.
      Demo, Docs, Themes, Repo | #SCSS

    • Semantic UI -用户界面是网络的语言.
      注意：该项目似乎处于非活动状态，但是有一个活动的fork称为 Fomantic UI.
      Docs, Repo | #LESS

    • Bulma -基于Flexbox的现代CSS框架.
      Demo, Docs, Repo | #Sass

    • Foundation -世界上最先进的响应式前端框架.
      Demo, Docs, Repo | #SCSS

    • UIkit -轻巧的模块化前端框架，用于开发快速而强大的Web界面.
      Demo, Docs, Repo | #LESS #SCSS

    • Spectre.css -轻巧，响应迅速的现代CSS框架.
      Docs, Repo | #SCSS

    • Tachyons -无需编写CSS，即可快速构建和设计新的UI.
      Demo, Docs, Repo | #CSS

    • Primer -支持GitHub前端设计的CSS框架.
      Docs, Repo | #SCSS

    • Carbon Components -IBM Carbon Design System背后的组件库.
      Docs, Repo | #SCSS

    • Blaze UI -开源模块化工具包，为可扩展和可维护的基础快速构建网站提供了良好的结构.
      Demo, Docs, Repo | #SCSS

    • PatternFly -企业Web应用程序的UI框架.
      Demo, Docs, Repo | #SCSS #LESS

    • Base -坚固可靠的响应式HTML / CSS框架.
      Docs, Repo | #SCSS #LESS

    • turretcss -用于可访问和语义网站的响应式前端框架.
      Demo, Docs, Repo | #CSS

    • Cirrus -具有精美控件和简单结构的响应式综合CSS框架.
      Docs, Repo | #CSS

    • HiQ -具有响应式排版和输入样式的简单CSS基础.
      Docs, Repo | #PostCSS

    Material Design¶

    • Material Components Web -用于网络的模块化和可自定义的Material Design UI组件.
      Demo, Docs, Repo | #SCSS

    • Materialize -基于材料设计的CSS框架.
      Demo, Docs, Repo | #SCSS

    • MUI -遵循Google的Material Design指南的轻量级CSS框架.
      Docs, Repo | #SCSS

    • Propeller -基于Google的Material Design标准和Bootstrap的前端响应框架.
      Docs, Repo | #SCSS

    Specialized¶

    • Tailwind CSS -实用程序优先的CSS框架，用于快速的UI开发.
      Docs, Repo | #CSS

    • NES.css -NES样式的CSS框架.
      Docs, Repo | #SCSS

    • Gutenberg -可以正确打印网络的现代框架.
      Demo, Repo | #SCSS

    • Tufte CSS -像Edward Tufte的讲义一样为您的网页设置样式.
      Repo | #CSS

    • Bojler -用于开发响应式和轻量级电子邮件模板的电子邮件框架.
      Docs, Repo | #SCSS

    Toolkits¶

    工具箱更接近于库而不是框架. 它们没有提供实际的样式，而是提供了可在CSS预处理器中使用的实用程序（例如mixins）.

    • Bourbon -轻量级Sass工具集.
      Docs, Repo, Scaffold | #SCSS
