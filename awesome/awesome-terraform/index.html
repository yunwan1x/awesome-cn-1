    Terraform

    Awesome Terraform ¶

    > 精选的资源列表 HashiCorp's Terraform.

    > 你 contributions 受欢迎的！

    Terraform 使您能够安全且可预测地创建、更改和改进生产基础设施. 它是一种开源工具，可将 API 编码为声明性配置文件，可在团队成员之间共享、将其视为代码、进行编辑、审查和版本控制.
