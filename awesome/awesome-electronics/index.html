    Electronics

    Awesome Electronics ¶

    >精选的电子工程师和业余爱好者的绝佳资源清单

    电子工程（EE）是理解，设计和构建电子电路的实践. 它通常与电气工程的区别在于，它主要处理低功率DC电子电路而不是高功率AC系统，但是电子与电气工程之间存在很多重叠之处.

    实验和构建电子电路也是一种流行的业余爱好，许多专业资源通常同样适用于业余爱好者，反之亦然.

    该列表适用于网站，服务，软件，工具等：您认为在电子工程领域中令人敬畏的一切. 如果您要添加任何内容，请按照中的说明进行操作. https://github.com/kitspace/awesome-electronics/blob/master/contributing.md.

    Learning¶

    Technical Tutorials¶

    • "skill" tag on learn.sparkfun.com -有关各种与EE相关技能的技术教程.
    • Soldering is Easy -涉及焊接基础知识的漫画书已被翻译成多种语言.
    • Uses of Different Soldering Iron Tips -涵盖所有这些不同烙铁头的优点.
    • How to design a motherboard for your electronics project -原理图和PCB设计入门教程

    Theory and Courses¶

    • Lessons In Electric Circuits -免费提供高质量的教科书和工作表，重点放在理论，模拟和苏格拉底式的方法上.
    • Ultimate Electronics: Practical Circuit Design and Analysis -CircuitLab免费的在线书籍，其中包含交互式原理图和仿真（正在开发中）.
    • Khan Academy - Electrical Engineering -非营利性学习平台，提供有关电气工程和相关主题的完整课程.
    • NEETS (Navy Electricity and Electronics Training Series) -美国海军非居民培训课程资料.

    University Course Archives¶

    • Berkeley EECS -全面的EE和CS课程网站档案.
    • Dr. Jacob Baker -课程和教程，内华达大学拉斯维加斯分校教授.
    • Dr. Abraham, Dr. McDermot， 和 Dr. Valvano -课程资料，UT Austin教授

    Simulators¶

    Analog and Mixed Signal Circuit Simulators¶

    • LTspice -凌力尔特（Linear Technologies）提供的行业标准免费SPICE电路仿真器. 另请参阅非官方 LTwiki 和 Group.
    • ngspice -开源SPICE电路模拟器.
    • Circuit JS/Falstad -具有电子流可视化功能的免费，开放源代码在线模拟器（由Paul Falstad重写了原始Java applet）.
    • EveryCircuit -免费试用在线，可视，交互式电路模拟器，以简化电路.
    • Qucs -开源，跨平台，非基于SPICE的电路仿真器，具有S参数和谐波平衡功能.
    • QucsStudio -Qucs的免费，开源，仅Windows分支，具有类似的界面，新的引擎和更多功能.
    • Open Circuit Design Software -开源，完整的EDA套件芯片设计套件，专注于跟上商业工具的发展.
    • TINA-TI - 独家 DesignSoft-TINA 与德州仪器（TI）模型捆绑在一起的德州仪器（TI）版本.
    • CppSim -利用C ++语言实现非常快速的仿真时间的免费，开放源电路仿真器.
    • Scilab with Xcos - Free, open source numerical computing alternative to MATLAB. Xcos provides Electrical System modeling capability similar to Simulink.
    • iCircuit -易于使用的电子电路仿真器，其先进的仿真引擎可处理模拟和数字电路，并具有实时始终在线分析功能.
    • Micro-Cap -具有多种交互式仿真类型的专业级混合信号仿真器.

    Verilog HDL Simulators¶

    • Verilator -免费的开源Verilog编译器. 测试平台使用C ++或SystemC. 速度非常快，但仅限于基于周期的2状态仿真和可合成的代码.
    • Icarus Verilog -免费的开源verilog解释器. 测试平台采用行为验证. 仿真是4状态的，基于事件.

    Gerber Viewers¶

    Online¶

    • Tracespace Viewer -Gerber查看器，可让您检查各个图层以及板的预览.
    • Gerblook -由Gerbv提供支持的在线Gerber查看器.
    • Mayhew Labs 3dpcb -3D Gerber查看器.
    • CircuitPeople -没有Gerbers多余的2D图层查看器，无需过多的处理.
    • Stackrate Viewer -易于使用的在线gerber查看器以及跟踪悬停和测量工具.

    Installable¶

    • Gerbv -适用于Linux和BSD的出色Gerber查看器.
    • KiCad GerbView -KiCad gerber查看器.

    Free EDA Packages¶

    • KiCad -带有推和推路由器，差分对等的开源EDA软件包.
    • Eagle -由于它的免费版本（受电路板尺寸限制），它是最受欢迎的EDA软件包之一.
    • DesignSpark PCB -由RS Components赞助的Gratis EDA软件包，无限制.
    • Altium CircuitMaker -转到专业软件制造商处的免费软件包.
    • gEDA -另一个开源软件包，仅对喜欢脚本和makefile，Linux和BSD的人有用.
    • DipTrace -具有免费版本的质量原理图捕获和PCB设计软​​件（受引脚和信号层限制）.
    • LibrePCB -适用于所有人，跨平台和GNU GPLv3的新的，功能强大且直观的EDA工具.
    • Horizon EDA -专注于快捷操作的免费开放源EDA工具.

    CAD Specific¶

    KiCad¶

    • Xesscorp's list of KiCad 3rd party tools
    • Contextual Electronics' Shine on You Crazy KiCad -入门视频教程，可帮助您尽快进入人造板.
    • Contextual Electronics' Getting to Blinky Tutorial -更全面的初学者到中级视频教程.
    • KiCad.info Forums -用户讨论和帮助论坛.
    • Keyboard PCB Guide -全面的书面教程，可带您创建键盘PCB.
    • Cheatsheet （还 in landscape）-简短的PDF，可用于大多数常用操作的菜单和键盘快捷键.
    • Footprint Collection -收集所有可在线获得的KiCad足迹以及一些用于管理它们的脚本.
    • InteractiveHtmlBom -用于手动拾取和放置的html BOM生成工具.
    • kicad-exports -自动为任何KiCAD项目生成文件（示意图，gerbers，BoM，图，3D模型）. 在本地运行，或在每个git push上运行，如Github Actions.

    Eagle¶

    • List of ULPs everyone should know
    • Adafruit Eagle Library
    • SparkFun Electronics Eagle Libraries

    PCB Syntax Highlighting¶

    • Language PCB -各种PCB格式的语法高亮显示.

    PCB Batching Services¶

    • PCBShopper -比较服务，可用于很多不同的PCB批处理和组装服务.
    • OSH Park -低成本PCB批处理服务，带有带有签名紫色丝印的高品质板.
    • Aisler -在欧洲（德国）制造和运输的价格合理的优质电路板.
    • Dirty PCBs -低成本的PCB批处理服务，以其“脏”质量而自豪.
    • JLCPCB -低成本的PCB批处理服务以及内部的低成本SMT服务.

    Part Search Engines¶

    • Octopart -可能是最知名的零件搜索引擎.
    • Findchips -从供应框架中搜索零件.
    • Parts.io -Supply Frame的另一个搜索引擎致力于发现新零件.

    Project Sharing Platforms¶

    • Kitspace -项目共享网站，可帮助您购买零件和重建项目. 开源，真正由您开发.
    • Hackaday.io -社交网站，用于分享热门博客中的项目.
    • Hackster.io -另一个用于共享项目的社交网站. 按平台，主题和产品井井有条.
    • InventHub -基于Git的项目托管和协作平台，用于硬件开发.
    • CADLAB -另一个基于Git的项目托管和协作平台，用于硬件开发.

    Inventory Management and Purchasing¶

    • 1-click BOM -浏览器扩展程序，可自动执行购买和零件搜索.
    • PartsBox -通过良好的用户界面和Octopart集成来管理零件库存的Web服务.
    • PartKeepr -开源Web服务，用于通过参数搜索和自动数据表导入来管理零件库存.
    • Part-DB -另一个开源Web服务，用于使用许可系统和良好的条形码生成器来管理零件库存.

    Miscellaneous Web Services¶

    • Eyrie -用于在线查看Eagle和KiCad设计.
    • SnapEDA -具有免费符号和封装的零件库. （与Eagle，KiCad，Altium，OrCad，Allegro等兼容）

    Development Board Retailers¶

    • Sparkfun -开源电子开发板以及其他设备和材料的零售商和设计师，并附有出色的教程.
    • Adafruit -另一家零售商和设计师，提供出色的选择和教程.
    • Tindie -供电子制造商销售小批量自己设计的市场.

    Blogs¶

    • Hackaday -可能是最受欢迎的博客，涵盖了整个作家团队，涉及电子和硬件黑客.
    • bunniestudios.com -Andrew'Bunnie'Huang涵盖了硬件黑客，开放式硬件，制造等等.
    • Bald Engineer -James Lewis撰写的有关电子产品和嵌入式软件的项目日志，教程和文章.
    • Rheingold Heavy -Dan Hienzsch撰写的有关电子产品和嵌入式软件的更多项目日志，教程和文章.
    • Hackster.io -另一个涵盖电子产品的博客.
    • Dangerous Prototypes -有关开源硬件项目和有趣的应用笔记的博客.
    • N-O-D-E -有关DIY电子产品，硬件和技术的博客.

    Forums¶

    Discussion¶

    • EEVBlog forum -可能是讨论电子工程主题的最大，最活跃的论坛.
    • /r/electronics 和 /r/ECE 是EE主题中最活跃的两个子Reddit.

    Help¶

    • /r/askelectronics -Subreddit致力于提供有关电子主题的帮助.
    • Electronics Stack Exchange -在流行的Stack Overflow服务上运行的电子产品的问答网站.
    • EEVBlog beginners forum -初学者提问的好地方，EEVblog上的其他子论坛应该适合于更高级的话题.

    Podcasts¶

    • The Amp Hour -与Chris Gammel和Dave Jones（EEVBlog）经常进行电子交流，并经常与来宾进行交流
    • Embedded.fm -Christopher和Elecia White经常与来宾一起讨论嵌入式系统开发等等.
    • The Spark Gap Podcast -每集都有一个特定的EE主题，有时还会有嘉宾.
    • MacroFab Engineering Podcast -每周播客，来自MacroFab的Parker和Stephen讨论EE主题和行业新闻.
    • The Engineering Commons Podcast -涵盖从机械到电气的一般工程主题.

    Videos¶

    • EEVblog -戴夫·琼斯（Dave Jones）进行拆解，教程等工作的最早，最成功的YouTube频道之一.
    • BigClive - YouTube channel 有关拆卸（包括危险产品），电路逆向工程和教程的信息.
    • ElectroBOOM -YouTube频道，对EE主题进行揭穿并解释了很多喜剧内容.
    • Micah Scott -逆向工程的视频日志，并以创新的方式重新利用消费类电子硬件.
